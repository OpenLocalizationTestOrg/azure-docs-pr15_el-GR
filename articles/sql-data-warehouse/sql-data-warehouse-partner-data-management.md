# <a name="sql-data-warehouse-data-management-partners"></a>Συνεργάτες διαχείρισης δεδομένων αποθήκη δεδομένων του SQL

Για να δημιουργήσετε τη λύση σας αποθήκη δεδομένων σε ολοκληρωμένες, επιλέξτε από μια μεγάλη ποικιλία κορυφαίες εργαλεία. Σε αυτό το άρθρο επισημαίνει εταιρείες συνεργάτες της Microsoft με εργαλεία διαχείρισης δεδομένων και λύσεις υποστήριξης αποθήκη δεδομένων του SQL Azure.
