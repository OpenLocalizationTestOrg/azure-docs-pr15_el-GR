# <a name="sql-data-warehouse-data-integration-partners"></a>Συνεργάτες ενοποίησης δεδομένων αποθήκη δεδομένων του SQL

Για να δημιουργήσετε τη λύση σας αποθήκη δεδομένων σε ολοκληρωμένες, επιλέξτε από μια μεγάλη ποικιλία κορυφαίες εργαλεία. Σε αυτό το άρθρο επισημαίνει εταιρείες συνεργάτες της Microsoft με λύσεις ενοποίησης δεδομένων υποστήριξης αποθήκη δεδομένων του SQL Azure.
