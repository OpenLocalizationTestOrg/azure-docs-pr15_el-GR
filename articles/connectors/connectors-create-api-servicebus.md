# <a name="get-started-with-the-azure-service-bus-connector"></a>Γρήγορα αποτελέσματα με τη γραμμή σύνδεσης Bus υπηρεσίας Azure

Σύνδεση με Azure Bus υπηρεσία για την αποστολή και λήψη μηνυμάτων. Μπορείτε να εκτελέσετε ενέργειες όπως η αποστολή σε ουρά, αποστολή στο θέμα, λαμβάνετε από ουρά και να λάβετε από τη συνδρομή.

Για να χρησιμοποιήσετε [οποιαδήποτε γραμμή σύνδεσης](./apis-list.md), πρέπει πρώτα να δημιουργήσετε μια εφαρμογή λογικής. Μπορείτε να ξεκινήσετε, [δημιουργώντας μια εφαρμογή της λογικής τώρα](../app-service-logic/app-service-logic-create-a-logic-app.md).

## <a name="connect-to-service-bus"></a>Σύνδεση σε υπηρεσία Bus

Πριν από την εφαρμογή της λογικής να αποκτήσετε πρόσβαση σε οποιαδήποτε υπηρεσία, πρέπει πρώτα να δημιουργήσετε μια σύνδεση στην υπηρεσία. [Σύνδεση](./connectors-overview.md) παρέχει σύνδεση ανάμεσα σε μια εφαρμογή λογικής και μια άλλη υπηρεσία.  

>[AZURE.INCLUDE [Steps to create a connection to Azure Service Bus](../../includes/connectors-create-api-servicebus.md)]

## <a name="use-a-service-bus-trigger"></a>Χρησιμοποιήστε ένα έναυσμα Bus υπηρεσίας

Ένα έναυσμα είναι ένα συμβάν που μπορούν να χρησιμοποιηθούν για να ξεκινήσετε τη ροή εργασίας που ορίζονται από το σε μια εφαρμογή για λογική. [Μάθετε περισσότερα σχετικά με τα εναύσματα](../app-service-logic/app-service-logic-what-are-logic-apps.md#logic-app-concepts).  

>[AZURE.INCLUDE [Steps to create a Service Bus trigger](../../includes/connectors-create-api-servicebus-trigger.md)]  

## <a name="use-a-service-bus-action"></a>Χρησιμοποιήστε μια ενέργεια Bus υπηρεσίας

Μια ενέργεια είναι μια ενέργεια που εκτελείται από τη ροή εργασίας που ορίζονται από το σε μια εφαρμογή για λογική. [Μάθετε περισσότερα σχετικά με τις ενέργειες](../app-service-logic/app-service-logic-what-are-logic-apps.md#logic-app-concepts).

## <a name="technical-details"></a>Τεχνικές λεπτομέρειες

Εδώ θα βρείτε τις λεπτομέρειες σχετικά με τα εναύσματα, ενέργειες και απαντήσεων που υποστηρίζει αυτή τη σύνδεση.

### <a name="service-bus-triggers"></a>Υπηρεσία Bus εναυσμάτων

Υπηρεσία Bus περιλαμβάνει τα παρακάτω εναύσματα:  

|Έναυσμα | Περιγραφή|
|--- | ---|
|[Όταν λαμβάνετε ένα μήνυμα σε μια ουρά](connectors-create-api-servicebus.md#when-a-message-is-received-in-a-queue)|Αυτή η λειτουργία εκκινεί μια ροή όταν λαμβάνετε ένα μήνυμα σε μια ουρά.|
|[Όταν λαμβάνετε ένα μήνυμα σε μια συνδρομή θέμα](connectors-create-api-servicebus.md#when-a-message-is-received-in-a-topic-subscription)|Αυτή η λειτουργία εκκινεί μια ροή όταν λαμβάνετε ένα μήνυμα σε μια συνδρομή το θέμα.|


### <a name="service-bus-actions"></a>Ενέργειες Bus υπηρεσίας

Υπηρεσία Bus περιλαμβάνει τις ακόλουθες ενέργειες:


|Ενέργεια|Περιγραφή|
|--- | ---|
|[Αποστολή μηνύματος](connectors-create-api-servicebus.md#send-message)|Αυτή η λειτουργία στέλνει ένα μήνυμα σε ουρά ή το θέμα.|
### <a name="action-and-trigger-details"></a>Η ενέργεια και έναυσμα λεπτομέρειες

Εδώ θα βρείτε τις λεπτομέρειες για τις ενέργειες και εναύσματα για αυτή η γραμμή σύνδεσης, μαζί με τις απαντήσεις τους.



#### <a name="send-message"></a>Αποστολή μηνύματος

|Όνομα ιδιότητας| Εμφανιζόμενο όνομα|Περιγραφή|
| ---|---|---|
|ContentData *|Περιεχόμενο|Το περιεχόμενο του μηνύματος.|
|ContentType|Τύπος περιεχομένου|Τύπος περιεχομένου του περιεχομένου του μηνύματος.|
|Ιδιότητες|Ιδιότητες|Ζεύγη κλειδιού-τιμής για κάθε όπου μεσολαβούν ιδιότητα.|
|το όνομα οντότητας *|Όνομα ουρά/θέματος|Το όνομα της ουράς ή το θέμα.|

Αυτές οι για προχωρημένους παραμέτρους είναι επίσης διαθέσιμες:

|Όνομα ιδιότητας| Εμφανιζόμενο όνομα|Περιγραφή|
| ---|---|---|
|Αναγνωριστικού μηνύματος|Αναγνωριστικό μηνύματος|Μια τιμή που ορίζονται από το χρήστη που μπορεί να χρήση Bus υπηρεσίας για τον προσδιορισμό διπλότυπα μηνύματα, εάν είναι ενεργοποιημένη.|
|Για να|Για να|Διεύθυνση για να στείλετε.|
|Τη|Απάντηση σε|Η διεύθυνση της ουράς για να απαντήσετε.|
|ReplyToSessionId|Απάντηση σε αναγνωριστικό περιόδου λειτουργίας|Αναγνωριστικό από την περίοδο λειτουργίας για να απαντήσετε.|
|Ετικέτα|Ετικέτα|Εφαρμογή ετικέτας.|
|ScheduledEnqueueTimeUtc|ScheduledEnqueueTimeUtc|Ημερομηνία και ώρα, UTC, όταν το μήνυμα θα προστεθεί στην ουρά.|
|Αναγνωριστικό περιόδου λειτουργίας|Αναγνωριστικό περιόδου λειτουργίας|Αναγνωριστικό της περιόδου λειτουργίας.|
|CorrelationId|Αναγνωριστικό συσχέτισης|Αναγνωριστικό του το συσχέτισης.|
|Το TimeToLive|Χρόνος διατήρησης|Η διάρκεια, σε διαβαθμίσεις, ότι ένα μήνυμα είναι έγκυρη. Η διάρκεια ξεκινά από όταν το μήνυμα αποστέλλεται Bus υπηρεσίας.|



Μια * υποδεικνύει ότι απαιτείται μια ιδιότητα.

### <a name="http-responses"></a>HTTP απαντήσεων

Προηγούμενη ενεργειών και εναύσματα μπορεί να επιστρέψει ένα ή περισσότερα από τους ακόλουθους κωδικούς κατάστασης HTTP:

|Όνομα|Περιγραφή|
|---|---|
|200|Ok|
|202|Αποδοχή|
|400|Ακατάλληλη αίτηση|
|401|Μη εξουσιοδοτημένο|
|403|Δεν επιτρέπεται|
|404|Δεν βρέθηκε|
|500|Εσωτερικό σφάλμα διακομιστή. Άγνωστο σφάλμα.|
|προεπιλεγμένη|Απέτυχε η λειτουργία.|
