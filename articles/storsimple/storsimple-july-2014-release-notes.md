# <a name="storsimple-8000-series-release-version-release-notes---july-2014"></a>Σημειώσεις έκδοσης StorSimple 8000 σειρά αφήστε έκδοση - Ιούλιος 2014 

## <a name="overview"></a>Επισκόπηση

Σημειώσεις έκδοσης παρακολούθηση προσδιορίστε το άνοιγμα κρίσιμα ζητήματα για τη σειρά 8000 StorSimple έκδοση γενικής διαθεσιμότητας (GA) Ιουλίου 2014 του Microsoft Azure StorSimple. Σε αυτήν την έκδοση που αντιστοιχεί στην έκδοση λογισμικού 6.3.9600.17215.  

Εκτός εάν ορίζεται διαφορετικά, αυτές τις σημειώσεις έκδοσης ισχύουν για όλα τα μοντέλα της συσκευής StorSimple. Σημειώσεις έκδοσης ενημερώνονται συνεχώς; καθώς εντοπιστούν κρίσιμα ζητήματα που απαιτεί μια λύση, προστίθενται. Πριν να αναπτύξετε τη λύση Microsoft Azure StorSimple σας, λάβετε υπόψη τις ακόλουθες πληροφορίες.
