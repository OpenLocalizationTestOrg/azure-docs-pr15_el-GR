# <a name="storsimple-system-limits"></a>Όρια του συστήματος StorSimple

## <a name="overview"></a>Επισκόπηση

StorSimple παρέχει μεταβλητού μεγέθους και ευέλικτη χώρου αποθήκευσης για το κέντρο δεδομένων. Ωστόσο, υπάρχουν κάποια όρια που που θα πρέπει να λάβετε υπόψη όταν σχεδιασμός, ανάπτυξη και λειτουργία τη λύση σας StorSimple. Ο παρακάτω πίνακας περιγράφει αυτά τα όρια και παρέχει ορισμένες συστάσεις, έτσι ώστε να μπορείτε να αξιοποιήσετε στο έπακρο τη λύση σας StorSimple.

| Αναγνωριστικό όριο | Όριο | Σχόλια |
|----------------- | ------|--------- |
| Μέγιστος αριθμός διαπιστευτήρια λογαριασμού χώρου αποθήκευσης | 64 | |
| Μέγιστος αριθμός τόμου κοντέινερ | 64 | |
| Μέγιστος αριθμός όγκους | 255 | |
| Μέγιστος αριθμός τοπικά καρφιτσωμένη όγκους | 32 | |
| Μέγιστος αριθμός των χρονοδιαγραμμάτων ανά πρότυπο εύρους ζώνης | 168 | Ένα χρονοδιάγραμμα για κάθε ώρα, κάθε ημέρα της εβδομάδας (24 * 7). |
| Μέγιστο μέγεθος ενός ομότιμου τόμου σε φυσική συσκευές | 64 TB για 8100 και 8600 | 8100 και 8600 είναι φυσική συσκευές. |
| Μέγιστο μέγεθος ενός ομότιμου τόμου σε εικονικές συσκευές στο Azure | 30 TB για 8010 <br></br> 64 TB για 8020 | 8010 και 8020 είναι εικονικές συσκευές στο Azure που χρησιμοποιούν τυπική χώρου αποθήκευσης και αποθήκευσης Premium αντίστοιχα. |
| Μέγιστο μέγεθος ενός τοπικά καρφιτσωμένη τόμου σε φυσική συσκευές | 8,5 TB για 8100 <br></br> 22,5 TB για 8600 | 8100 και 8600 είναι φυσική συσκευές. |
| Μέγιστος αριθμός των συνδέσεων iSCSI | 512 | |
| Μέγιστος αριθμός των συνδέσεων iSCSI από εκκινητές | 512 | |

| Μέγιστος αριθμός των στιγμιότυπων, τα οποία μπορούν να βρίσκονται σε οποιαδήποτε συσκευή | 10.000 | |
| Μέγιστος αριθμός όγκους που είναι δυνατή η επεξεργασία παράλληλα για αντιγράφου ασφαλείας, επαναφορά ή κλωνοποίηση | 16 |<ul><li>Εάν υπάρχουν περισσότερες από 16 όγκους, τους επεξεργάζονται διαδοχικά κατά την επεξεργασία υποδοχές γίνονται διαθέσιμες.</li><li>Επαναφέρει ομότιμου τόμου ή αντίγραφα ασφαλείας για μια κλωνοποιημένη δεν μπορεί να προκύψουν μέχρι να ολοκληρωθεί η λειτουργία. Ωστόσο, για ένα τοπικό όγκος, δημιουργία αντιγράφων ασφαλείας επιτρέπονται μετά την ένταση ήχου είναι σε σύνδεση.</li></ul>|
| Επαναφορά και κλωνοποίηση ανάκτηση χρόνο για ομότιμου όγκους | < 2 λεπτά | <ul><li>Η ένταση διατίθεται εντός 2 λεπτών της λειτουργίας επαναφοράς ή Κλωνοποίηση, ανεξάρτητα από το μέγεθος του τόμου.</li><li>Η απόδοση ένταση αρχικά μπορεί να είναι μικρότερη από το κανονικό όπως περισσότερα δεδομένα και μετα-δεδομένων εξακολουθεί να βρίσκεται στο cloud. Μετα-δεδομένων ενημερώνεται αυτόματα στη συσκευή στο παρασκήνιο κατά τη χρέωση 5 λεπτά ανά TB δεδομένων τόμου που έχει εκχωρηθεί. Το επιτόκιο αυτό μπορεί να επηρεάζεται από το εύρος ζώνης Internet στο cloud.</li><li>Η επαναφορά ή η λειτουργία κλωνοποίηση έχει ολοκληρωθεί, όταν είναι όλα τα μετα-δεδομένα στη συσκευή.</li><li>Λειτουργίες δημιουργίας αντιγράφων ασφαλείας δεν μπορεί να εκτελεστεί μέχρι την επαναφορά ή κλωνοποίηση λειτουργία έχει ολοκληρωθεί πλήρως.|
| Επαναφορά ανάκτηση χρόνο για τοπικά καρφιτσωμένη όγκους | < 2 λεπτά | <ul><li>Η ένταση διατίθεται εντός 2 λεπτών της λειτουργίας επαναφοράς, ανεξάρτητα από το μέγεθος του τόμου.</li><li>Η απόδοση ένταση αρχικά μπορεί να είναι μικρότερη από το κανονικό όπως περισσότερα δεδομένα και μετα-δεδομένων εξακολουθεί να βρίσκεται στο cloud. Απόδοση μπορεί να αυξήσουν ως ροών δεδομένων από το cloud στη συσκευή StorSimple.</li><li>Ο συνολικός χρόνος για να κάνετε λήψη μετα-δεδομένων εξαρτάται από το μέγεθος του τόμου που έχει εκχωρηθεί. Μετα-δεδομένων ενημερώνεται αυτόματα στη συσκευή στο παρασκήνιο κατά τη χρέωση 5 λεπτά ανά TB δεδομένων τόμου που έχει εκχωρηθεί. Το επιτόκιο αυτό μπορεί να επηρεάζεται από το εύρος ζώνης Internet στο cloud.</li><li>Σε αντίθεση με ομότιμου όγκους, για τοπικά καρφιτσωμένη όγκους, η ένταση είναι επίσης λήψη των δεδομένων τοπικά στη συσκευή. Η διαδικασία επαναφοράς έχει ολοκληρωθεί όταν όλα τα δεδομένα ένταση έφερε στη συσκευή.</li><li>Οι λειτουργίες επαναφοράς μπορεί να είναι μεγάλο. Ο συνολικός χρόνος για την ολοκλήρωση της επαναφοράς θα εξαρτώνται από το μέγεθος του την προμήθεια του φακέλου τοπικού τόμου, το εύρος ζώνης Internet και τα υπάρχοντα δεδομένα στη συσκευή. Λειτουργίες δημιουργίας αντιγράφων ασφαλείας του τοπικά καρφιτσωμένη τόμου επιτρέπονται ενώ η λειτουργία επαναφοράς βρίσκεται σε εξέλιξη.|
|Ταχύτητα επεξεργασίας για το cloud στιγμιότυπα| 15 λεπτά/TB | <ul><li>Ελάχιστο χρόνο για να κάνετε στο cloud στιγμιότυπο έτοιμη για αποστολή, ανά TB δεδομένων τόμου που έχει εκχωρηθεί στο αντίγραφο ασφαλείας. </li><li> Συνολικό cloud στιγμιότυπο ώρα υπολογίζεται με την προσθήκη αυτήν τη στιγμή στο χρόνο αποστολής στιγμιότυπου, ο οποίος επηρεάζεται από το εύρος ζώνης Internet στο cloud.
| Μέγιστο προγράμματος-πελάτη μετάδοσης ανάγνωσης/εγγραφής (όταν δημοσιευτεί από τη σειρά SSD) * | 920/720 MB/s με μία μόνο διασύνδεση δικτύου 10 GbE | Έως 2 x με MPIO και δύο διασυνδέσεις δικτύου. |
| Μέγιστο προγράμματος-πελάτη μετάδοσης ανάγνωσης/εγγραφής (όταν δημοσιευτεί από τη σειρά του σκληρού ΔΊΣΚΟΥ) * | 120/250 MB/s |
| Μέγιστο προγράμματος-πελάτη ανάγνωσης/εγγραφής μετάδοσης (όταν δημοσιευτεί από τη σειρά cloud)* για ενημέρωση 3 και νεότερες εκδόσεις** | 40/60 MB/s για επιπέδων όγκους<br><br>60/80 MB/s για επιπέδων όγκους με αρχειοθέτησης την επιλογή κατά τη δημιουργία της έντασης ήχου | Ανάγνωση μετάδοσης εξαρτάται από προγράμματα-πελάτες δημιουργία και τη διατήρηση αρκετό βάθος ουράς εισόδου/εξόδου. <br><br>Ταχύτητα επίτευξη εξαρτάται από την ταχύτητα του υποκείμενου λογαριασμού χώρου αποθήκευσης που χρησιμοποιείται. | 

& #42; Μέγιστη ταχύτητα μετάδοσης ανά τύπο εισόδου/εξόδου ήταν υπολογίζεται με 100 τοις εκατό ανάγνωσης και εγγραφής 100 τοις εκατό σενάρια. Πραγματική μετάδοσης μπορεί να είναι χαμηλότερη και εξαρτώνται από εισόδου/εξόδου Δημιουργήστε συνδυασμό και δικτύου συνθήκες.

& #42- & #42; Αριθμοί επιδόσεων πριν από την ενημέρωση 3 μπορεί να είναι χαμηλότερη.
