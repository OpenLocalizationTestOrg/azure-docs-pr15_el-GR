# <a name="monitor-a-storage-account-in-the-azure-portal"></a>Παρακολούθηση ενός λογαριασμού χώρου αποθήκευσης στην πύλη του Azure

## <a name="overview"></a>Επισκόπηση

Μπορείτε να παρακολουθήσετε το λογαριασμό χώρου αποθήκευσης από την [Πύλη Azure](https://portal.azure.com). Όταν ρυθμίζετε τις παραμέτρους του λογαριασμού χώρου αποθήκευσης για την παρακολούθηση μέσω της πύλης, αποθήκευσης Azure χρησιμοποιεί [Χώρο αποθήκευσης ανάλυσης](http://msdn.microsoft.com/library/azure/hh343270.aspx) για την παρακολούθηση μετρικά για το λογαριασμό σας και καταγραφή αίτηση δεδομένων.

> [AZURE.NOTE]Πρόσθετο κόστος συσχετίζονται με εξέταση παρακολούθησης δεδομένων στην [Πύλη του Azure](https://portal.azure.com). Για περισσότερες πληροφορίες, ανατρέξτε στο θέμα <a href="http://msdn.microsoft.com/library/azure/hh360997.aspx">ανάλυση χώρου αποθήκευσης και τις χρεώσεις</a>. <br />

> Azure χώρο αποθήκευσης αρχείων προς το παρόν υποστηρίζει μετρικά αποθήκευσης ανάλυση, αλλά ακόμα δεν υποστηρίζει καταγραφή. Μπορείτε να ενεργοποιήσετε τις μετρήσεις για χώρο αποθήκευσης αρχείων Azure μέσω του [Azure πύλη](https://portal.azure.com).

> Οι λογαριασμοί χώρου αποθήκευσης με έναν τύπο αναπαραγωγής της ζώνης πλεονάζοντα χώρο αποθήκευσης (ZRS) δεν διαθέτουν το μετρικά ή τη δυνατότητα καταγραφής ενεργοποιήσει αυτήν τη στιγμή. 

> Για ένα αναλυτικό οδηγό σχετικά με τη χρήση ανάλυσης χώρου αποθήκευσης και άλλα εργαλεία για τον προσδιορισμό, διάγνωση και αντιμετώπιση προβλημάτων που σχετίζονται με το χώρο αποθήκευσης Azure, ανατρέξτε στο θέμα [Εποπτεία, διάγνωση, και αντιμετώπιση προβλημάτων του Microsoft Azure αποθήκευσης](storage-monitoring-diagnosing-troubleshooting.md).


## <a name="how-to-configure-monitoring-for-a-storage-account"></a>Πώς μπορείτε να: ρύθμιση παραμέτρων παρακολούθησης για ένα λογαριασμό του χώρου αποθήκευσης

1. Στην [Πύλη του Azure](https://portal.azure.com), κάντε κλικ στην επιλογή **Αποθήκευση**και, στη συνέχεια, κάντε κλικ στο όνομα του λογαριασμού χώρου αποθήκευσης για να ανοίξετε τον πίνακα εργαλείων.

2. Κάντε κλικ στην επιλογή **Ρύθμιση παραμέτρων**και κάντε κύλιση προς τα κάτω για τις ρυθμίσεις **παρακολούθησης** για το blob, πίνακα και τις υπηρεσίες ουρά.

    ![MonitoringOptions](./media/storage-monitor-storage-account/Storage_MonitoringOptions.png)

3. **Παρακολούθηση**, ορίστε το επίπεδο της παρακολούθησης και την πολιτική διατήρησης δεδομένων για κάθε υπηρεσία:

    -  Για να ορίσετε το επίπεδο παρακολούθησης, επιλέξτε ένα από τα εξής:

      **Ελάχιστο** - μετρικά συλλέγει όπως εισόδου/εξόδου, διαθεσιμότητα, λανθάνοντα χρόνο και την επιτυχία ποσοστά που έχουν συναθροιστεί για το blob, πίνακα και τις υπηρεσίες ουρά.

      **Λεπτομερής** - εκτός από το ελάχιστο μετρήσεις, συλλέγει το ίδιο σύνολο μετρήσεις για κάθε λειτουργία χώρου αποθήκευσης σε το API των υπηρεσιών Azure αποθήκευσης. Λεπτομερής μετρικά Ενεργοποίηση πιο κοντά ανάλυσης θεμάτων που προκύπτουν κατά τη διάρκεια λειτουργίες εφαρμογών.

      **Απενεργοποίηση** - απενεργοποιεί την παρακολούθηση. Υπάρχουσες παρακολούθηση δεδομένων είναι σταθερές έως το τέλος της περιόδου διατήρησης.

- Για να ορίσετε την πολιτική διατήρησης δεδομένων, **Διατήρηση (σε ημέρες)**, πληκτρολογήστε τον αριθμό των ημερών των δεδομένων για να κρατήσει από 1 έως 365 ημέρες. Εάν δεν θέλετε να ορίσετε μια πολιτική διατήρησης, πληκτρολογήστε μηδέν. Εάν δεν υπάρχει πολιτική διατήρησης, είναι ώστε να μπορείτε να διαγράψετε τα δεδομένα παρακολούθησης. Συνιστάται να τη ρύθμιση πολιτικής διατήρησης που βασίζονται σε χρονικό διάστημα που θέλετε να διατηρούνται αποθήκευσης ανάλυσης δεδομένων για το λογαριασμό σας, έτσι ώστε να μπορούν να διαγραφούν δεδομένων analytics παλαιά και που δεν χρησιμοποιείται από το σύστημα χωρίς κόστος.

4. Όταν ολοκληρώσετε τη ρύθμιση παραμέτρων παρακολούθησης, κάντε κλικ στην επιλογή **Αποθήκευση**.

Θα πρέπει να ξεκινάτε βλέπουν παρακολούθηση δεδομένων σε πίνακα εργαλείων και σελίδα **οθόνη** μετά περίπου μία ώρα.

Μέχρι να ρυθμίζετε τις παραμέτρους παρακολούθησης για ένα λογαριασμό του χώρου αποθήκευσης, χωρίς παρακολούθησης δεδομένα που συλλέγονται και τα γραφήματα μετρικά στη σελίδα **οθόνη** και πίνακα εργαλείων είναι κενά.

Μετά τη ρύθμιση των επιπέδων παρακολούθησης και των πολιτικών διατήρησης, μπορείτε να επιλέξετε ποιο από τα διαθέσιμα μετρικά για την παρακολούθηση της στην [Πύλη του Azure](https://portal.azure.com), και ποια μετρικά για να απεικονίσετε σε γραφήματα μετρήσεις. Εμφανίζεται ένα προεπιλεγμένο σύνολο μετρικά σε κάθε επίπεδο παρακολούθησης. Μπορείτε να χρησιμοποιήσετε **Μετρικά Προσθήκη** για να προσθέσετε ή να καταργήσετε μετρικά από τη λίστα μετρήσεις.

Μετρικά αποθηκεύονται στο λογαριασμό χώρου αποθήκευσης σε τέσσερα πίνακες που ονομάζονται $MetricsTransactionsBlob, $MetricsTransactionsTable, $MetricsTransactionsQueue και $MetricsCapacityBlob. Για περισσότερες πληροφορίες, ανατρέξτε στο θέμα [Σχετικά με μετρικά αποθήκευσης ανάλυσης](http://msdn.microsoft.com/library/azure/hh343258.aspx).


## <a name="how-to-customize-the-dashboard-for-monitoring"></a>Πώς μπορείτε να: Προσαρμογή του πίνακα εργαλείων για παρακολούθηση

Στον πίνακα εργαλείων, μπορείτε να επιλέξετε έως και έξι μετρήσεις για να απεικονίσετε στο γράφημα μετρικά από εννέα διαθέσιμες μετρήσεις. Για κάθε υπηρεσία (blob πίνακα και ουρά), τα μετρικά διαθεσιμότητα, ποσοστό επιτυχίας και το σύνολο των αιτήσεων είναι διαθέσιμες. Τα διαθέσιμα στον πίνακα εργαλείων μετρικά είναι τα ίδια για την παρακολούθηση ελάχιστους ή λεπτομερές.

1. Στην [Πύλη του Azure](https://portal.azure.com), κάντε κλικ στην επιλογή **Αποθήκευση**και, στη συνέχεια, κάντε κλικ στο όνομα του λογαριασμού χώρου αποθήκευσης για να ανοίξετε τον πίνακα εργαλείων.

2. Για να αλλάξετε τα μετρικά που σχεδιάζονται στο γράφημα, εκτελέστε μία από τις εξής ενέργειες:

    - Για να προσθέσετε μια νέα μέτρηση στο γράφημα, κάντε κλικ στην επιλογή το έγχρωμο πλαίσιο ελέγχου δίπλα μετρικό κεφαλίδα του πίνακα κάτω από το γράφημα.

    - Για να αποκρύψετε ένα μετρικό που θα σχεδιαστούν στο γράφημα, καταργήστε την επιλογή από το έγχρωμο πλαίσιο ελέγχου δίπλα στην κεφαλίδα μετρικό.

        ![Monitoring_nmore](./media/storage-monitor-storage-account/storage_Monitoring_nmore.png)

3. Από προεπιλογή, το γράφημα εμφανίζει τάσεις, εμφανίζει μόνο την τρέχουσα τιμή κάθε μέτρησης ( **σχετική** επιλογή στο επάνω μέρος του γραφήματος). Για να εμφανίσετε έναν άξονα Y, ώστε να μπορείτε να δείτε απόλυτες τιμές, επιλέξτε **απόλυτη**.

4. Για να αλλάξετε το εύρος χρόνου το γράφημα εμφανίζει μετρήσεις, επιλέξτε 6 ώρες, 24 ώρες ή 7 ημέρες στο επάνω μέρος του γραφήματος.


## <a name="how-to-customize-the-monitor-page"></a>Πώς μπορείτε να: Προσαρμογή της σελίδας οθόνη

Στη σελίδα " **οθόνη** ", μπορείτε να προβάλετε το πλήρες σύνολο μετρικά για το λογαριασμό χώρου αποθήκευσης.

- Εάν ο λογαριασμός χώρου αποθήκευσης έχει ελάχιστους παρακολούθηση έχει ρυθμιστεί, μετρικά όπως εισόδου/εξόδου, διαθεσιμότητα, λανθάνων χρόνος και ποσοστά επιτυχίας έχουν συναθροιστεί από το blob, πίνακα και τις υπηρεσίες ουρά.

- Εάν ο λογαριασμός σας χώρο αποθήκευσης έχει λεπτομερής παρακολούθηση έχει ρυθμιστεί, τα μετρικά είναι διαθέσιμες σε καλύτερο ανάλυση των εργασιών μεμονωμένα αποθήκευσης εκτός από τις συγκεντρώσεις επίπεδο εξυπηρέτησης.

Χρησιμοποιήστε τις ακόλουθες διαδικασίες για να επιλέξετε ποια μετρικά αποθήκευσης για να προβάλετε τα γραφήματα μετρικά και πίνακα που εμφανίζονται στη σελίδα " **οθόνη** ". Αυτές οι ρυθμίσεις δεν επηρεάζουν το συλλογής συνάθροισης και χώρου αποθήκευσης της παρακολούθησης δεδομένων στο λογαριασμό χώρου αποθήκευσης.

## <a name="how-to-add-metrics-to-the-metrics-table"></a>Πώς μπορείτε να: Προσθέστε μετρικά στον πίνακα μετρικά


1. Στην [Πύλη του Azure](https://portal.azure.com), κάντε κλικ στην επιλογή **Αποθήκευση**και, στη συνέχεια, κάντε κλικ στο όνομα του λογαριασμού χώρου αποθήκευσης για να ανοίξετε τον πίνακα εργαλείων.

2. Κάντε κλικ στην επιλογή **Παρακολούθηση**.

    Ανοίγει η σελίδα **οθόνη** . Από προεπιλογή, ο πίνακας μετρικά εμφανίζει ένα υποσύνολο των μετρικών που είναι διαθέσιμες για παρακολούθηση. Η εικόνα εμφανίζει την οθόνη προεπιλογή για ένα λογαριασμό του χώρου αποθήκευσης με λεπτομερή παρακολούθηση έχει ρυθμιστεί για όλες τις τρεις υπηρεσίες. Χρησιμοποιήστε **Μετρικά Προσθήκη** για να επιλέξετε τα μετρικά που θέλετε να παρακολουθήσετε από όλα τα διαθέσιμα μετρικά.

    ![Monitoring_VerboseDisplay](./media/storage-monitor-storage-account/Storage_Monitoring_VerboseDisplay.png)

    > [AZURE.NOTE] Εξετάστε το ενδεχόμενο κόστους όταν επιλέγετε τα μετρικά. Υπάρχουν συναλλαγής και εξόδου κόστος που είναι συσχετισμένο με την ανανέωση παρακολούθησης που εμφανίζεται. Για περισσότερες πληροφορίες, ανατρέξτε στο θέμα [ανάλυση χώρου αποθήκευσης και τις χρεώσεις](http://msdn.microsoft.com/library/azure/hh360997.aspx).

3. Κάντε κλικ στην επιλογή **Προσθήκη μετρήσεις**.

    Τα μετρικά συγκεντρωτικών αποτελεσμάτων που είναι διαθέσιμες στο ελάχιστους παρακολούθηση είναι στο επάνω μέρος της λίστας. Εάν είναι επιλεγμένο το πλαίσιο ελέγχου, τη μέτρηση εμφανίζεται στη λίστα μετρήσεις.

    ![AddMetricsInitialDisplay](./media/storage-monitor-storage-account/Storage_AddMetrics_InitialDisplay.png)

4. Τοποθετήστε το δείκτη επάνω από τη δεξιά πλευρά του παραθύρου διαλόγου για να εμφανίσετε μια γραμμή κύλισης που μπορείτε να σύρετε για να πραγματοποιήσετε κύλιση πρόσθετων μετρήσεων στην προβολή.

    ![AddMetricsScrollbar](./media/storage-monitor-storage-account/Storage_AddMetrics_Scrollbar.png)


5. Κάντε κλικ στο βέλος προς τα κάτω κατά ένα μετρικό για να αναπτύξετε μια λίστα με τις λειτουργίες τη μέτρηση είναι ρυθμισμένο για να συμπεριλάβετε. Επιλέξτε κάθε εργασία που θέλετε να δείτε στον πίνακα μετρικά στην [Πύλη του Azure](https://portal.azure.com).

    Στην παρακάτω εικόνα, έχει αναπτυχθεί τη μέτρηση ΠΟΣΟΣΤΟΎ ΣΦΆΛΜΑΤΟΣ ΕΞΟΥΣΙΟΔΌΤΗΣΗΣ.

    ![ExpandCollapse](./media/storage-monitor-storage-account/Storage_AddMetrics_ExpandCollapse.png)


6. Αφού επιλέξετε μετρήσεις για όλες τις υπηρεσίες, κάντε κλικ στο κουμπί OK (σημάδι ελέγχου) για να ενημερώσετε τις ρυθμίσεις παραμέτρων παρακολούθησης. Το επιλεγμένο μετρικά προστίθενται στον πίνακα μετρήσεις.

7. Για να διαγράψετε ένα μετρικό από τον πίνακα, κάντε κλικ στην επιλογή τη μέτρηση για να το επιλέξετε και, στη συνέχεια, κάντε κλικ στην επιλογή **Διαγραφή μετρικό σύστημα**.

    ![DeleteMetric](./media/storage-monitor-storage-account/Storage_DeleteMetric.png)

## <a name="how-to-customize-the-metrics-chart-on-the-monitor-page"></a>Πώς μπορείτε να: Προσαρμογή του γραφήματος μετρικά στη σελίδα "Παρακολούθηση"

1. Στη σελίδα " **οθόνη** " για το λογαριασμό χώρου αποθήκευσης, στον πίνακα μετρικά, επιλέξτε έως 6 μετρικά για να απεικονίσετε στο γράφημα μετρήσεις. Για να επιλέξετε ένα μετρικό, επιλέξτε το πλαίσιο ελέγχου από την αριστερή πλευρά. Για να καταργήσετε ένα μετρικό από το γράφημα, καταργήστε την επιλογή από το πλαίσιο ελέγχου.

2. Για να μεταβείτε στο γράφημα μεταξύ σχετικών τιμών (τελική τιμή εμφανίζεται μόνο) και απόλυτες τιμές (εμφανίζεται άξονα Y), επιλέξτε **σχετική** ή **απόλυτη** στο επάνω μέρος του γραφήματος.

3.  Για να αλλάξετε την ώρα περιοχής τα μετρικά γραφήματος που εμφανίζεται, επιλέξτε **6 ώρες**, **24 ώρες**ή **7 ημέρες** στο επάνω μέρος του γραφήματος.



## <a name="how-to-configure-logging"></a>Πώς μπορείτε να: ρύθμιση παραμέτρων καταγραφής

Για κάθε μία από τις υπηρεσίες χώρου αποθήκευσης είναι διαθέσιμες με το λογαριασμό χώρου αποθήκευσης (blob πίνακα και ουρά), μπορείτε να αποθηκεύσετε αρχεία καταγραφής διαγνωστικών για αιτήσεις ανάγνωση, εγγραφή αιτήσεις ή/και διαγραφή προσκλήσεων και να ορίσετε την πολιτική διατήρησης δεδομένων για κάθε μία από τις υπηρεσίες.

1. Στην [Πύλη του Azure](https://portal.azure.com), κάντε κλικ στην επιλογή **Αποθήκευση**και, στη συνέχεια, κάντε κλικ στο όνομα του λογαριασμού χώρου αποθήκευσης για να ανοίξετε τον πίνακα εργαλείων.

2. Κάντε κλικ στην επιλογή **Ρύθμιση παραμέτρων**και χρησιμοποιήστε το κάτω βέλος στο πληκτρολόγιο για να κάνετε κύλιση προς τα κάτω, **Καταγραφή**.

    ![Storagelogging](./media/storage-monitor-storage-account/Storage_LoggingOptions.png)


3. Για κάθε υπηρεσία (blob πίνακα και ουρά), ρυθμίστε τα εξής:

    - Οι τύποι αίτηση για να συνδεθείτε: προσκλήσεις ανάγνωση, εγγραφή αιτήσεις και διαγραφή προσκλήσεων.

    - Ο αριθμός των ημερών για να διατηρήσετε τα συνδεδεμένα δεδομένα. Πληκτρολογήστε μηδέν είναι εάν δεν θέλετε να ορίσετε μια πολιτική διατήρησης. Εάν δεν ορίσετε μια πολιτική διατήρησης, είναι ώστε να μπορείτε να διαγράψετε τα αρχεία καταγραφής.

4. Κάντε κλικ στην επιλογή **Αποθήκευση**.

Τα αρχεία καταγραφής διαγνωστικών αποθηκεύονται με το όνομα $logs στο λογαριασμό σας στο χώρο αποθήκευσης αντικειμένων blob κοντέινερ. Για πληροφορίες σχετικά με την πρόσβαση στο κοντέινερ $logs, ανατρέξτε στο θέμα [Σχετικά με το χώρο αποθήκευσης ανάλυση καταγραφή](http://msdn.microsoft.com/library/azure/hh343262.aspx).
