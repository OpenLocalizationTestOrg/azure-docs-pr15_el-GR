# <a name="controlling-azure-web-app-traffic-with-azure-traffic-manager"></a>Έλεγχος κυκλοφορία Azure web app με τη Διαχείριση κίνηση Azure

> [AZURE.NOTE] Σε αυτό το άρθρο παρέχει συνοπτικές πληροφορίες για το Microsoft Azure κίνηση Manager όσον αφορά την Azure εφαρμογής υπηρεσίας Web Apps. Μπορείτε να βρείτε περισσότερες πληροφορίες σχετικά με το Azure κίνηση Διαχείριση ίδια με την επίσκεψή σας οι συνδέσεις στο τέλος αυτού του άρθρου.

## <a name="introduction"></a>Εισαγωγή
Μπορείτε να χρησιμοποιήσετε Azure κίνηση Manager για να ελέγξετε τον τρόπο αιτήσεις από προγράμματα-πελάτες web διανέμονται στις εφαρμογές web apps στο Azure εφαρμογής υπηρεσίας. Όταν τα τελικά σημεία εφαρμογής web προστίθενται ένα προφίλ διαχείρισης κίνηση Azure, Azure κίνηση Manager διατηρεί παρακολούθηση της κατάστασης από τις εφαρμογές web (εκτελείται, διακοπεί ή διαγραφεί), έτσι ώστε το να αποφασίσετε ποια από αυτά τα τελικά σημεία πρέπει να λάβετε την κυκλοφορία.

## <a name="load-balancing-methods"></a>Φόρτωση εξισορρόπησης μεθόδους
Διαχείριση Azure κίνηση χρησιμοποιεί τρεις μέθοδοι εξισορρόπησης φόρτου διαφορετικό. Αυτές περιγράφονται στην παρακάτω λίστα όπως πελάτη που αφορούν Azure web apps.

* **Ανακατεύθυνση**: Εάν έχετε αντίγραφα εφαρμογής web σε διαφορετικές περιοχές, μπορείτε να χρησιμοποιήσετε αυτήν τη μέθοδο για να ρυθμίσετε ένα web app για να υπηρεσίας όλη την κυκλοφορία προγράμματος-πελάτη web και ρύθμιση παραμέτρων άλλη εφαρμογή web σε διαφορετική περιοχή για την εξυπηρέτηση η κυκλοφορία σε περίπτωση που το πρώτο web app δεν είναι διαθέσιμη.

* **ROUND Robin**: Εάν έχετε αντίγραφα εφαρμογής web σε διαφορετικές περιοχές, μπορείτε να χρησιμοποιήσετε αυτήν τη μέθοδο για τη διανομή κίνηση εξίσου σε όλες τις εφαρμογές web σε διαφορετικές περιοχές.

* **Απόδοση**: Η απόδοση μέθοδο κατανέμει την κυκλοφορία με βάση την πιο μικρή ώρα αποστολής και επιστροφής σε υπολογιστές-πελάτες. Η μέθοδος επιδόσεων μπορεί να χρησιμοποιηθεί για τις εφαρμογές web μέσα στην ίδια περιοχή ή σε διαφορετικές περιοχές.

##<a name="web-apps-and-traffic-manager-profiles"></a>Εφαρμογές Web και την κυκλοφορία Διαχείριση προφίλ
Για να ρυθμίσετε τις παραμέτρους του στοιχείου ελέγχου web app κίνησης, δημιουργήστε ένα προφίλ στο Azure κίνηση Manager που χρησιμοποιεί ένα από τα τρία φόρτωση εξισορρόπησης μεθόδους που περιγράφονται προηγουμένως και, στη συνέχεια, προσθέστε τα τελικά σημεία (σε αυτήν την περίπτωση, οι εφαρμογές web) για την οποία θέλετε να ελέγξετε την κυκλοφορία για το προφίλ. Την κατάσταση εφαρμογής web (εκτελείται, διακοπεί ή διαγραφεί) τακτικά μεταδίδονται στο προφίλ έτσι ώστε να Azure κίνηση Manager να κατευθύνουν την κυκλοφορία.

Όταν χρησιμοποιείτε Azure κίνηση Manager με Azure, λάβετε υπόψη τα εξής σημεία:

* Για web app αναπτύξεις μόνο μέσα στην ίδια περιοχή, Web Apps παρέχει ήδη ανακατεύθυνσης και λειτουργικότητα round robin ανεξάρτητα από κατάσταση εφαρμογής web.

* Για αναπτύξεις στην ίδια περιοχή που χρησιμοποιούν Web Apps σε συνδυασμό με την άλλη υπηρεσία Azure cloud, μπορείτε να συνδυάσετε και των δύο τύπων για να ενεργοποιήσετε την υβριδική σενάρια τα τελικά σημεία.

* Μπορείτε να καθορίσετε μόνο ένα τελικό σημείο εφαρμογής web ανά περιοχή σε ένα προφίλ. Όταν επιλέγετε μια εφαρμογή web ως ένα τελικό σημείο για μία περιοχή, τις υπόλοιπες εφαρμογές web σε αυτήν την περιοχή θα είναι διαθέσιμες για την επιλογή για αυτό το προφίλ.

* Τα τελικά σημεία εφαρμογής web που καθορίζετε σε ένα προφίλ διαχείρισης κίνηση Azure θα εμφανίζονται κάτω από την ενότητα **Ονομάτων τομέων** στη σελίδα Ρύθμιση παραμέτρων για την εφαρμογή web στο προφίλ, αλλά δεν θα είναι δυνατή η ρύθμιση παραμέτρων εκεί.

* Αφού προσθέσετε μια εφαρμογή web σε προφίλ, τη **Διεύθυνση URL της τοποθεσίας** στον πίνακα εργαλείων του σελίδα της πύλης του web app θα εμφανίσει τη διεύθυνση URL του προσαρμοσμένου τομέα του web app, εάν έχετε ρυθμίσει ένα. Διαφορετικά, θα εμφανίσει τη διεύθυνση URL προφίλ Manager κίνηση (για παράδειγμα, `contoso.trafficmgr.com`). Τόσο το όνομα τομέα απευθείας από την εφαρμογή web και τη διεύθυνση URL κίνηση Manager θα είναι ορατό στη σελίδα Ρύθμιση παραμέτρων του web app στην ενότητα **Domain Names** .

* Τα ονόματα προσαρμοσμένου τομέα σας θα λειτουργούν όπως αναμένεται, αλλά εκτός από την προσθήκη τους στις εφαρμογές web, πρέπει επίσης να ρυθμίσετε χάρτη σας DNS στην οποία θα οδηγεί η διεύθυνση Manager κίνηση URL. Για πληροφορίες σχετικά με τη ρύθμιση του προσαρμοσμένου τομέα για μια εφαρμογή Azure web, ανατρέξτε στο θέμα [Ρύθμιση ένα προσαρμοσμένο όνομα τομέα για μια τοποθεσία web Azure](web-sites-custom-domain-name.md).

* Μπορείτε να προσθέσετε μόνο εφαρμογές web που χρησιμοποιούνται στην τυπική λειτουργία σε προφίλ Azure κίνηση Manager.
