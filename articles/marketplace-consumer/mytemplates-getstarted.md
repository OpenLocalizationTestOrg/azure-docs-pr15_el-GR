# <a name="get-started-with-private-templates-on-the-azure-portal"></a>Γρήγορα αποτελέσματα με τα ιδιωτικά πρότυπα στην πύλη του Azure

Ένα πρότυπο [Από διαχειριστή πόρων Azure](../resource-group-authoring-templates.md) είναι μια δηλωτικό προτύπου που χρησιμοποιείται για να ορίσετε την ανάπτυξη. Μπορείτε να ορίσετε τους πόρους για να αναπτύξετε μια λύση, και να καθορίσετε τις παραμέτρους και μεταβλητές που δίνουν τη δυνατότητα να εισαγάγετε τιμές για διαφορετικά περιβάλλοντα. Το πρότυπο αποτελείται από JSON και παραστάσεις που μπορείτε να χρησιμοποιήσετε για να δημιουργήσετε τιμές για την ανάπτυξη.

Μπορείτε να χρησιμοποιήσετε τη νέα δυνατότητα **πρότυπα** στην [Πύλη του Azure](https://portal.azure.com) μαζί με την υπηρεσία παροχής του πόρου **Microsoft.Gallery** ως επέκταση από το [Azure Marketplace](https://azure.microsoft.com/marketplace/) για να επιτρέψετε στους χρήστες να δημιουργήσετε, να διαχειριστείτε και ανάπτυξη ιδιωτικό προτύπων από μια προσωπική βιβλιοθήκη.

Αυτό το έγγραφο σας καθοδηγεί στις Προσθήκη, τη διαχείριση και κοινή χρήση ενός ιδιωτικού **προτύπου** με την πύλη Azure.

## <a name="guidance"></a>Καθοδήγηση

Τις παρακάτω προτάσεις θα σας βοηθήσουν να αξιοποιήσετε πλήρως **πρότυπα** κατά την εργασία με τις λύσεις:

- **Το πρότυπο** είναι ένας πόρος encapsulating που περιέχει ένα πρότυπο διαχείρισης πόρων και πρόσθετων μετα-δεδομένων. Πολύ ομοίως συμπεριφέρεται σε ένα στοιχείο στην αγορά. Η βασική διαφορά είναι ότι είναι ενός ιδιωτικού στοιχείου σε αντίθεση με τα δημόσια στοιχεία Marketplace.
- Η βιβλιοθήκη **προτύπων** λειτουργεί καλά για χρήστες που χρειάζονται για να προσαρμόσετε τους αναπτύξεις.
- **Πρότυπα** λειτουργεί καλά για χρήστες που χρειάζονται ένα απλό αποθετήριο εντός Azure.
- Ξεκινήστε με ένα υπάρχον πρότυπο διαχείρισης πόρων. Βρείτε πρότυπα στο [github](https://github.com/Azure/azure-quickstart-templates) ή να [εξαγάγετε το πρότυπο](../resource-manager-export-template.md) από μια υπάρχουσα ομάδα πόρων.
- **Πρότυπα** συνδέονται με το χρήστη που δημοσιεύει τους. Το όνομα του publisher είναι ορατό σε οποιονδήποτε έχει πρόσβαση για ανάγνωση σε αυτήν.
- **Πρότυπα** είναι οι πόροι για τη διαχείριση πόρων και δεν είναι δυνατή η μετονομασία δημοσιεύονται μία φορά.

## <a name="add-a-template-resource"></a>Προσθήκη πόρου προτύπου

Υπάρχουν δύο τρόποι για να δημιουργήσετε έναν πόρο **προτύπου** στην πύλη του Azure.

### <a name="method-1--create-a-new-template-resource-from-a-running-resource-group"></a>Μέθοδος 1: Δημιουργήστε ένα νέο πόρο προτύπου από μια εκτελείται ομάδα πόρων

1. Μεταβείτε σε μια υπάρχουσα ομάδα πόρων στην πύλη του Azure. Επιλέξτε **Εξαγωγή προτύπου** στις **Ρυθμίσεις**.
2. Μετά την εξαγωγή του προτύπου για τη διαχείριση πόρων, χρησιμοποιήστε το κουμπί **Αποθήκευση προτύπου** για να την αποθηκεύσετε στο αποθετήριο **πρότυπα** . Εύρεση πλήρεις λεπτομέρειες για εξαγωγή πρότυπο [εδώ](../resource-manager-export-template.md).
<br /><br />
![Εξαγωγή ομάδα πόρων](media/rg-export-portal1.PNG)  <br />

3. Επιλέξτε το κουμπί εντολής **Αποθήκευση στο πρότυπο** .
<br /><br />

4. Εισαγάγετε τις παρακάτω πληροφορίες:

    - Όνομα-όνομα του αντικειμένου προτύπου (ΣΗΜΕΊΩΣΗ: αυτό είναι ένα όνομα για το Azure διαχείριση πόρων με βάση. Όλοι οι κανόνες περιορισμοί ισχύουν και δεν μπορεί να αλλάξει αφού δημιουργηθεί).
    - Περιγραφή-γρήγορη σύνοψη σχετικά με το πρότυπο.

    ![Αποθήκευση προτύπου](media/save-template-portal1.PNG)  <br />

5. Κάντε κλικ στην επιλογή **Αποθήκευση**.

    > [AZURE.NOTE] Η εξαγωγή blade προτύπου εμφανίζει ειδοποιήσεις όταν το εξαγόμενο πρότυπο διαχείρισης πόρων έχει σφάλματα, αλλά θα εξακολουθείτε να μπορείτε να αποθηκεύσετε αυτό το πρότυπο διαχείρισης πόρων για τα πρότυπα. Βεβαιωθείτε ότι έλεγχος και διόρθωση διαχείριση πόρων πρότυπο προβλήματα πριν από την επανάληψη ανάπτυξης το εξαγόμενο πρότυπο διαχείρισης πόρων.

### <a name="b-method-2--add-a-new-template-resource-from-browse"></a>B. Μέθοδος 2: Προσθήκη νέου πόρου πρότυπο από την αναζήτηση

Μπορείτε επίσης να προσθέσετε ένα νέο **πρότυπο** από εκτός σελίδας χρησιμοποιώντας το + Προσθήκη κουμπιού εντολής στο **Αναζήτηση > πρότυπα**. Θα πρέπει να δώσετε ένα όνομα, περιγραφή και το πρότυπο διαχείρισης πόρων JSON.

![Προσθήκη προτύπου](media/add-template-portal1.PNG)  <br />

> [AZURE.NOTE] Microsoft.Gallery είναι ένας μισθωτής βάσει υπηρεσίας παροχής Azure πόρων. Ο πόρος πρότυπο είναι συνδεδεμένη με το χρήστη που το δημιούργησε. Αυτό δεν είναι συνδεδεμένη με οποιαδήποτε συγκεκριμένες συνδρομή. Πρέπει να επιλεγεί μόνο όταν ένα πρότυπο για την ανάπτυξη μιας συνδρομής.

Αυτό περιλαμβάνει **πρότυπα** που έχετε δημιουργήσει, καθώς και αυτές που έχουν τεθεί σε κοινή χρήση μαζί σας με διάφορα επίπεδα δικαιωμάτων. Περισσότερες λεπτομέρειες στην παρακάτω ενότητα [Έλεγχος πρόσβασης](#access-control-for-a-tenant-resource-provider) .

![Προβολή προτύπου](media/view-template-portal1.PNG)  <br />

Μπορείτε να προβάλετε τις λεπτομέρειες ενός **προτύπου** , κάνοντας κλικ σε ένα στοιχείο στη λίστα.

![Προβολή προτύπου](media/view-template-portal2c.png)  <br />

## <a name="edit-a-template-resource"></a>Επεξεργασία ενός πόρου προτύπου

Μπορείτε να ξεκινήσετε τη ροή επεξεργασία για ένα **πρότυπο** , κάνοντας δεξί κλικ στο στοιχείο στη λίστα αναζήτηση ή επιλέγοντας το κουμπί εντολής "Επεξεργασία".

![Επεξεργασία προτύπου](media/edit-template-portal1a.PNG)  <br />

Μπορείτε να επεξεργαστείτε την περιγραφή ή κείμενο προτύπου για τη διαχείριση πόρων. Δεν μπορείτε να επεξεργαστείτε το όνομα, επειδή πρόκειται για ένα όνομα πόρου από διαχειριστή πόρων. Όταν επεξεργάζεστε το πρότυπο διαχείρισης πόρων JSON θα σας θα επικύρωση για να βεβαιωθείτε ότι είναι έγκυρη JSON. Επιλέξτε **OK** και, στη συνέχεια, **Αποθήκευση** για να αποθηκεύσετε το πρότυπό σας ενημερωμένο.

![Επεξεργασία προτύπου](media/edit-template-portal2a.PNG)  <br />

Αφού το **πρότυπο** αποθηκεύεται θα δείτε μια ειδοποίηση επιβεβαίωσης.

![Επεξεργασία προτύπου](media/edit-template-portal3b.png)  <br />

## <a name="deploy-a-template-resource"></a>Ανάπτυξη προτύπου πόρου

Μπορείτε να αναπτύξετε οποιοδήποτε **πρότυπο** που έχετε δικαιώματα **ανάγνωσης** στο. Η ροή ανάπτυξης εκκινεί το τυπικό blade ανάπτυξη προτύπου Azure. Συμπληρώστε τις τιμές για τις παραμέτρους του προτύπου για τη διαχείριση πόρων για να προχωρήσετε στην ανάπτυξη.

![Ανάπτυξη προτύπου](media/deploy-template-portal1b.png)  <br />

## <a name="share-a-template-resource"></a>Κοινή χρήση πόρων προτύπου

Ένας πόρος **πρότυπο** μπορεί να είναι κοινόχρηστη με τους συνεργάτες σας. Κοινή χρήση συμπεριφέρεται παρόμοια με [ανάθεσης ρόλων για κάθε πόρο σε Azure](../active-directory/role-based-access-control-configure.md). Ο κάτοχος του **προτύπου** παρέχει δικαιώματα σε άλλους χρήστες που μπορούν να αλληλεπιδράσουν με έναν πόρο πρότυπο. Το άτομο ή ομάδα από άτομα που μοιράζονται το **πρότυπο** με θα μπορούν να δουν το πρότυπο διαχείρισης πόρων και τις ιδιότητές της συλλογής.

### <a name="access-control-for-the-microsoftgallery-resources"></a>Έλεγχος πρόσβασης για τους πόρους Microsoft.Gallery

Ρόλος | Δικαιώματα
---|----
Κάτοχος | Επιτρέπει την πλήρη έλεγχο στον πόρο πρότυπο όπως κοινή χρήση
Πρόγραμμα ανάγνωσης | Επιτρέπει την ανάγνωση και Execute(Deploy) στον πόρο προτύπου
Συμβολής | Επιτρέπει την επεξεργασία και διαγραφή δικαιωμάτων στον πόρο πρότυπο. Χρήστη δεν είναι δυνατό να κάνετε κοινή χρήση του προτύπου με άλλους

Επιλέξτε **κοινή χρήση** στο στοιχείο Αναζήτηση κάνοντας δεξί κλικ στην επιλογή ή την προβολή blade ενός συγκεκριμένου στοιχείου. Έτσι ανοίγει μια εμπειρία κοινή χρήση.

![Κοινή χρήση προτύπου](media/share-template-portal1a.png)  <br />

 Τώρα, μπορείτε να επιλέξετε ένα ρόλο και ένα χρήστη ή μια ομάδα για την παροχή πρόσβασης σε ένα συγκεκριμένο **πρότυπο**. Οι διαθέσιμες ρόλοι είναι κάτοχο, ανάγνωσης και συμβολής. Περισσότερες λεπτομέρειες στην παραπάνω ενότητα [Έλεγχος πρόσβασης](#access-control-for-a-tenant-resource-provider) .

![Κοινή χρήση προτύπου](media/share-template-portal2b.png)  <br />

![Κοινή χρήση προτύπου](media/share-template-portal3b.png)  <br />

Κάντε κλικ στο κουμπί **Ok**και **Επιλέξτε** . Τώρα, μπορείτε να δείτε τους χρήστες ή τις ομάδες που προστίθενται στον πόρο.

![Κοινή χρήση προτύπου](media/share-template-portal4b.png)  <br />

> [AZURE.NOTE] Ένα πρότυπο μπορεί να είναι κοινόχρηστη μόνο με χρήστες και ομάδες στον ίδιο μισθωτή Azure Active Directory. Εάν κάνετε κοινή χρήση ενός προτύπου με μια διεύθυνση ηλεκτρονικού ταχυδρομείου που δεν περιλαμβάνεται στο μισθωτή σας, μια πρόσκληση θα σταλεί ερώτησης προς το χρήστη για να συμμετάσχετε στο μισθωτή ως επισκέπτης.
