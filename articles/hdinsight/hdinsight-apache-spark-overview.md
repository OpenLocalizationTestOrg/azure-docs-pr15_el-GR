# <a name="overview-apache-spark-on-hdinsight-linux"></a>Επισκόπηση: Apache τους σε HDInsight Linux
 
<a href="http://spark.apache.org/" target="_blank">Apache τους</a> είναι μια παράλληλα ανοιχτού κώδικα επεξεργασίας framework που υποστηρίζει στη μνήμη επεξεργασίας για να ενισχύσει την απόδοση των εφαρμογών ανάλυσης μεγάλο δεδομένων. Μηχανισμός επεξεργασίας τους είναι σχεδιασμένες για ταχύτητα, ευκολία στη χρήση και εξελιγμένο analytics. Δυνατότητες στη μνήμη κατά τον υπολογισμό του τους να είναι μια καλή επιλογή για επαναληπτικού αλγόριθμους σε μηχανικής εκμάθησης και graph υπολογισμούς. Τους επίσης είναι συμβατά με το χώρο αποθήκευσης αντικειμένων Blob του Azure (WASB), ώστε τα υπάρχοντα δεδομένα σας είναι αποθηκευμένα σε Azure εύκολα μπορεί να γίνει επεξεργασία μέσω τους.

Όταν δημιουργείτε ένα σύμπλεγμα τους σε HDInsight, μπορείτε να δημιουργήσετε Azure υπολογισμού πόρων με τους εγκατασταθεί και ρυθμιστεί. Χρειάζονται μόνο περίπου δέκα λεπτά για να δημιουργήσετε ένα σύμπλεγμα τους στο HDInsight. Τα δεδομένα προς επεξεργασία αποθηκεύονται στο χώρο αποθήκευσης αντικειμένων Blob του Azure. Ανατρέξτε στο θέμα [Χρήση αποθήκευσης αντικειμένων Blob του Azure με HDInsight][hdinsight-storage].

## <a name="why-use-spark-on-azure-hdinsight"></a>Γιατί να χρησιμοποιήσετε τους σε Azure HDInsight; 

Azure HDInsight προσφέρει μια πλήρως διαχειριζόμενη υπηρεσία τους. Πλεονεκτήματα της χρήσης τους σε HDInsight είναι οι εξής:

| Η δυνατότητα                             | Περιγραφή       |
|-------------------------------------|-------------------|
| Διευκόλυνση της δημιουργίας συμπλεγμάτων            | Μπορείτε να δημιουργήσετε ένα νέο σύμπλεγμα τους σε HDInsight σε λεπτά χρησιμοποιώντας την πύλη διαχείρισης του Azure, Azure PowerShell ή το .NET SDK HDInsight. Ανατρέξτε στο θέμα [Γρήγορα αποτελέσματα με το σύμπλεγμα τους στο HDInsight](hdinsight-apache-spark-jupyter-spark-sql.md) |
| Ευκολία στη χρήση                     | Τους στο HDInsight συμπλεγμάτων περιλαμβάνει σημειωματάρια Jupyter προ-έχει ρυθμιστεί. Μπορείτε να χρησιμοποιήσετε αυτές τις αλληλεπιδραστικές επεξεργασίας δεδομένων και απεικόνιση. Οι διευθύνσεις URL για την είναι https://CLUSTERNAME.azurehdinsight.net/jupyter. Αντικαταστήστε __CLUSTERNAME__ με το όνομα του συμπλέγματος HDInsight τους.|
| ΥΠΌΛΟΙΠΟ APIs                       | Τους στο HDInsight περιλαμβάνει [Λίβιος](https://github.com/cloudera/hue/tree/master/apps/spark/java#welcome-to-livy-the-rest-spark-server), ένα REST API, με βάση τους διακομιστή εργασιών για να υποβάλετε από απόσταση και να παρακολουθείτε τις εργασίες που εκτελούνται. |
| Υποστήριξη για το χώρο αποθήκευσης λίμνης δεδομένων Azure | Τους σε HDInsight μπορεί να ρυθμιστεί για να χρησιμοποιήσετε το χώρο αποθήκευσης λίμνης δεδομένων Azure ως μια επιπλέον χώρο αποθήκευσης. Για περισσότερες πληροφορίες στο χώρο αποθήκευσης λίμνης δεδομένων, ανατρέξτε στο θέμα [Επισκόπηση των Azure λίμνης χώρου αποθήκευσης δεδομένων](../data-lake-store/data-lake-store-overview.md).
| Ενοποίηση με τις υπηρεσίες του Azure | Τους σε HDInsight συνοδεύεται από μια γραμμή σύνδεσης με διανομείς συμβάν Azure. Οι πελάτες να δημιουργήσετε ροή εφαρμογές που χρησιμοποιούν το συμβάν διανομείς, εκτός από την [Kafka](http://kafka.apache.org/), που είναι ήδη διαθέσιμη ως μέρος τους. |
| Υποστήριξη για το διακομιστή R  | Μπορείτε να ρυθμίσετε ένα διακομιστή R σε σύμπλεγμα HDInsight τους για να εκτελέσετε υπολογισμούς κατανέμεται R με τις ταχύτητες υποσχόμενη με ένα σύμπλεγμα τους. Για περισσότερες πληροφορίες, ανατρέξτε στο θέμα [Γρήγορα αποτελέσματα με το διακομιστή R στο HDInsight](hdinsight-hadoop-r-server-get-started.md).   |
| Ενοποίηση με το IntelliJ ΙΔΈΑ  | Μπορείτε να χρησιμοποιήσετε την προσθήκη HDInsight για IntelliJ για να δημιουργήσετε και να υποβάλετε αιτήσεις σε ένα σύμπλεγμα HDInsight τους. Για περισσότερες πληροφορίες ανατρέξτε στο θέμα [Χρήση HDInsight εργαλεία προσθήκης για IntelliJ ΙΔΈΑ για να δημιουργήσετε εφαρμογές για το σύμπλεγμα Linux τους HDInsight τους](hdinsight-apache-spark-intellij-tool-plugin.md). |
| Ταυτόχρονες ερωτημάτων              | Τους στο HDInsight υποστηρίζει ταυτόχρονες ερωτήματα. Αυτή η δυνατότητα επιτρέπει πολλών ερωτημάτων από ένα χρήστη ή πολλών ερωτημάτων από διάφορες χρήστες και εφαρμογές για να κάνετε κοινή χρήση τους ίδιους πόρους συμπλέγματος. |
| Προσωρινή αποθήκευση σε SSD                 | Μπορείτε να επιλέξετε να cache δεδομένων στο μνήμης ή στο SSD που έχουν επισυναφθεί σε κόμβους συμπλέγματος. Προσωρινή αποθήκευση στη μνήμη παρέχει τις καλύτερες επιδόσεις ερωτήματος, αλλά θα μπορούσε να είναι ακριβή; προσωρινή αποθήκευση σε SSD παρέχει μια εξαιρετική επιλογή για τη βελτίωση της απόδοσης του ερωτήματος χωρίς να χρειάζεται να δημιουργήσετε ένα σύμπλεγμα μεγέθους που απαιτείται για την προσαρμογή του συνόλου δεδομένων ολόκληρο στη μνήμη.|
| Ενοποίηση με εργαλεία Επιχειρηματικής ευφυΐας       | Τους για HDInsight παρέχει συνδέσεις για εργαλεία Επιχειρηματικής ευφυΐας όπως [Power BI](http://www.powerbi.com/) και [Tableau](http://www.tableau.com/products/desktop) για ανάλυση δεδομένων.|
| Προ-φορτωμένου Anaconda βιβλιοθήκες        | Τους συμπλεγμάτων σε HDInsight παρέχονται με βιβλιοθήκες Anaconda προ-εγκατεστημένο. [Anaconda](http://docs.continuum.io/anaconda/) παρέχει κοντά 200 βιβλιοθηκών για μηχανικής εκμάθησης, ανάλυση δεδομένων, απεικόνισης, κ.λπ.|
| Κλιμάκωση                     | Παρόλο που μπορείτε να καθορίσετε τον αριθμό των κόμβους στο σύμπλεγμά σας κατά τη δημιουργία, που μπορεί να θέλετε να μεγέθυνση ή σμίκρυνση του συμπλέγματος ώστε να ταιριάζει με φόρτο εργασίας. Όλα συμπλεγμάτων HDInsight σάς επιτρέπουν να αλλάξετε τον αριθμό των κόμβους του συμπλέγματος. Επίσης, να καταργηθεί συμπλεγμάτων τους χωρίς απώλεια δεδομένων δεδομένου ότι όλα τα δεδομένα που είναι αποθηκευμένη στο χώρο αποθήκευσης Blob του Azure. |
| Υποστήριξη 24/7                    | Τους σε HDInsight συνοδεύεται από εταιρικό επίπεδο υποστήριξης 24/7 και μια SLA του 99,9% του χρόνου.|



## <a name="what-are-the-use-cases-for-spark-on-hdinsight"></a>Τι είναι το περιπτώσεις χρήσης για τους σε HDInsight;

Apache τους στο HDInsight επιτρέπει την παρακάτω βασικά σενάρια.

### <a name="interactive-data-analysis-and-bi"></a>Ανάλυση αλληλεπιδραστικών δεδομένων και BI

[Δείτε ένα πρόγραμμα εκμάθησης](hdinsight-apache-spark-use-bi-tools.md)

Apache τους στο HDInsight αποθηκεύει δεδομένα σε αντικείμενα BLOB Azure. Ειδικούς επιχειρήσεων και κλειδιού αποφάσεων να ανάλυση και δημιουργία αναφορών πάνω από αυτά τα δεδομένα και χρησιμοποιήστε το Microsoft Power BI για δημιουργία αλληλεπιδραστικών αναφορών από την ανάλυση δεδομένων. Οι αναλυτές μπορούν να Έναρξη από μη δομημένα/ημιδιαφανές δομημένα δεδομένα στο χώρο αποθήκευσης Azure, ορίζουν ένα σχήμα για τα δεδομένα με σημειωματάρια και, στη συνέχεια, να δημιουργήσετε μοντέλα δεδομένων χρησιμοποιώντας το Microsoft Power BI. Τους στο HDInsight υποστηρίζει επίσης έναν αριθμό εργαλεία άλλων κατασκευαστών BI όπως Tableau, Qlikview και Lumira SAP καθιστώντας μια ιδανική πλατφόρμα για τους αναλυτές δεδομένων, ειδικούς επιχειρήσεων και κλειδιού αποφάσεων.

### <a name="iterative-machine-learning"></a>Επαναληπτικού μηχανικής εκμάθησης

[Δείτε ένα πρόγραμμα εκμάθησης: πρόβλεψης δημιουργία θερμοκρασίες uisng HVAC δεδομένων](hdinsight-apache-spark-ipython-notebook-machine-learning.md)

[Δείτε ένα πρόγραμμα εκμάθησης: πρόβλεψης τροφίμων στα αποτελέσματα ελέγχου](hdinsight-apache-spark-machine-learning-mllib-ipython.md)

Τους Apache συνοδεύεται από [MLlib](http://spark.apache.org/mllib/), μια μηχανικής εκμάθησης βιβλιοθήκη ενσωματωμένη επάνω τους. Εκτός από αυτό, τους σε HDInsight περιλαμβάνει επίσης Anaconda, μια κατανομή Python με μια ποικιλία πακέτων για μηχανικής εκμάθησης. Συνδυάσετε αυτό με μια ενσωματωμένη υποστήριξη για τα σημειωματάρια Jupyter και θα έχετε ένα περιβάλλον επάνω γραμμής για τη δημιουργία εφαρμογών μηχανικής εκμάθησης.  

### <a name="streaming-and-real-time-data-analysis"></a>Ανάλυση δεδομένων ροής και σε πραγματικό χρόνο

[Δείτε ένα πρόγραμμα εκμάθησης](hdinsight-apache-spark-eventhub-streaming.md)

Ανάλυση δεδομένων σε πραγματικό χρόνο χρησιμοποιείται για σενάρια διαβάθμιση από μείωση του χρόνου σε πληροφορίες για δεδομένα με την επεξεργασία δεδομένων, όπως καταλήξει, για να δημιουργείτε μια τιμή true ροής λύση. Τους στο HDInsight προσφέρει μια εμπλουτισμένη υποστήριξη για τη δημιουργία λύσεων σε πραγματικό χρόνο ανάλυση. Ενώ τους έχει ήδη γραμμές σύνδεσης για να ingest δεδομένα από πολλές προελεύσεις, όπως Kafka, αγωγός νερού, στο Twitter, ZeroMQ ή TCP sockets, τους στο HDInsight προσθέτει κορυφαία υποστήριξη για ingesting δεδομένων από το Azure συμβάν διανομείς. Συμβάν διανομείς είναι η υπηρεσία ουράς χρησιμοποιούνται ευρύτερα στην Azure. Αντιμετωπίζετε ένα υποστήριξη εκτός του πλαισίου για διανομείς συμβάν κάνει αυξήσετε στο HDInsight μια ιδανική πλατφόρμα για τη δημιουργία διοχέτευσης ανάλυση πραγματικό χρόνο.
