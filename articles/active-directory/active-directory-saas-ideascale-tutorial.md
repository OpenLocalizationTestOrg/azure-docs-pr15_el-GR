#<a name="tutorial-azure-active-directory-integration-with-ideascale"></a>Πρόγραμμα εκμάθησης: Ενοποίηση καταλόγου Azure Active Directory με IdeaScale
  
Είναι ο στόχος αυτού του προγράμματος εκμάθησης για να εμφανίσετε την ενοποίηση του Azure και IdeaScale.  
Το σενάριο που περιγράφονται σε αυτό το πρόγραμμα εκμάθησης προϋποθέτει ότι έχετε ήδη τα ακόλουθα στοιχεία:

-   Μια έγκυρη συνδρομή Azure
-   Μια IdeaScale καθολικής σύνδεσης με δυνατότητα συνδρομή
  
Μετά την ολοκλήρωση αυτού του προγράμματος εκμάθησης, θα μπορείτε να καθολικής σύνδεσης στην εφαρμογή με την [Εισαγωγή στο πλαίσιο πρόσβασης](active-directory-saas-access-panel-introduction.md)Azure AD χρηστών που έχουν εκχωρηθεί σε IdeaScale.
  
Το σενάριο που περιγράφονται σε αυτό το πρόγραμμα εκμάθησης αποτελείται από τα εξής δομικά στοιχεία:

1.  Ενεργοποίηση την ενοποίηση εφαρμογής για IdeaScale
2.  Ρύθμιση παραμέτρων Καθολικής σύνδεσης
3.  Ρύθμιση παραμέτρων προμήθεια του χρήστη
4.  Αντιστοίχιση χρηστών

![Σενάριο] (./media/active-directory-saas-ideascale-tutorial/IC790838.png "Σενάριο")
##<a name="enabling-the-application-integration-for-ideascale"></a>Ενεργοποίηση την ενοποίηση εφαρμογής για IdeaScale
  
Είναι ο στόχος αυτής της ενότητας για να εφαρμόσετε διάρθρωση πώς μπορείτε να ενεργοποιήσετε την ενοποίηση εφαρμογής για το IdeaScale.

###<a name="to-enable-the-application-integration-for-ideascale-perform-the-following-steps"></a>Για να ενεργοποιήσετε την ενοποίηση εφαρμογής για IdeaScale, ακολουθήστε τα παρακάτω βήματα:

1.  Στην κλασική πύλη Azure, στο αριστερό παράθυρο περιήγησης, κάντε κλικ στην επιλογή **Υπηρεσία καταλόγου Active Directory**.

    ![Υπηρεσία καταλόγου Active Directory] (./media/active-directory-saas-ideascale-tutorial/IC700993.png "Υπηρεσία καταλόγου Active Directory")

2.  Από τη λίστα **καταλόγου** , επιλέξτε τον κατάλογο για την οποία θέλετε να ενεργοποιήσετε την ενοποίηση καταλόγου.

3.  Για να ανοίξετε τις εφαρμογές προβολή, στην προβολή του καταλόγου, κάντε κλικ στην επιλογή **εφαρμογές** στο μενού επάνω.

    ![Εφαρμογές] (./media/active-directory-saas-ideascale-tutorial/IC700994.png "Εφαρμογές")

4.  Κάντε κλικ στην επιλογή **Προσθήκη** στο κάτω μέρος της σελίδας.

    ![Προσθήκη εφαρμογής] (./media/active-directory-saas-ideascale-tutorial/IC749321.png "Προσθήκη εφαρμογής")

5.  Στο παράθυρο διαλόγου **Τι θέλετε να κάνετε** , κάντε κλικ στην επιλογή **Προσθήκη εφαρμογής από τη συλλογή**.

    ![Προσθήκη εφαρμογής από gallerry] (./media/active-directory-saas-ideascale-tutorial/IC749322.png "Προσθήκη εφαρμογής από gallerry")

6.  Στο **πλαίσιο αναζήτησης**, πληκτρολογήστε **IdeaScale**.

    ![Συλλογή εφαρμογών] (./media/active-directory-saas-ideascale-tutorial/IC790841.png "Συλλογή εφαρμογών")

7.  Στο παράθυρο αποτελεσμάτων, επιλέξτε **IdeaScale**και, στη συνέχεια, κάντε κλικ στην επιλογή **Ολοκλήρωση** , για να προσθέσετε την εφαρμογή.

    ![IdeaScale] (./media/active-directory-saas-ideascale-tutorial/IC790842.png "IdeaScale")
##<a name="configuring-single-sign-on"></a>Ρύθμιση παραμέτρων Καθολικής σύνδεσης
  
Στόχος αυτής της ενότητας είναι να περιγράψει τον τρόπο για να επιτρέψετε στους χρήστες να ελέγχουν την ταυτότητα IdeaScale με το λογαριασμό στο Azure AD χρησιμοποιώντας Ομοσπονδία με βάση το πρωτόκολλο SAML.  
Ρύθμιση παραμέτρων Καθολικής σύνδεσης για IdeaScale απαιτεί να ανακτήσετε μια τιμή αποτύπωση από ένα πιστοποιητικό.  
Εάν δεν είστε εξοικειωμένοι με αυτήν τη διαδικασία, ανατρέξτε στο θέμα [πώς μπορείτε να ανακτήσετε ένα πιστοποιητικό αποτύπωση τιμή](http://youtu.be/YKQF266SAxI).

###<a name="to-configure-single-sign-on-perform-the-following-steps"></a>Για να ρυθμίσετε τις παραμέτρους της καθολικής σύνδεσης, ακολουθήστε τα παρακάτω βήματα:

1.  Στην κλασική πύλη Azure, στη σελίδα ενοποίησης εφαρμογής **IdeaScale** , κάντε κλικ στην επιλογή **Ρύθμιση παραμέτρων Καθολικής σύνδεσης** για να ανοίξετε το παράθυρο διαλόγου **Ρύθμιση παραμέτρων Καθολικής σύνδεσης** .

    ![Ρύθμιση παραμέτρων Καθολικής σύνδεσης] (./media/active-directory-saas-ideascale-tutorial/IC790843.png "Ρύθμιση παραμέτρων Καθολικής σύνδεσης")

2.  Στη σελίδα **Πώς θέλετε οι χρήστες να πραγματοποιούν είσοδο IdeaScale** , επιλέξτε **Microsoft Azure AD καθολικής σύνδεσης**και, στη συνέχεια, κάντε κλικ στο κουμπί **Επόμενο**.

    ![Ρύθμιση παραμέτρων Καθολικής σύνδεσης] (./media/active-directory-saas-ideascale-tutorial/IC790844.png "Ρύθμιση παραμέτρων Καθολικής σύνδεσης")

3.  Στη σελίδα **Ρύθμιση παραμέτρων URL εφαρμογής** , στο πλαίσιο κειμένου **IdeaScale εισόδου στη διεύθυνση URL** , πληκτρολογήστε τη διεύθυνση URL που χρησιμοποιούνται από τους χρήστες σας για να συνδεθείτε εφαρμογή σας IdeaScale (π.χ.: "*https://company.IdeaScale.com*"), και, στη συνέχεια, κάντε κλικ στο κουμπί **Επόμενο**.

    ![Ρύθμιση παραμέτρων εφαρμογής διεύθυνσης URL] (./media/active-directory-saas-ideascale-tutorial/IC790845.png "Ρύθμιση παραμέτρων εφαρμογής διεύθυνσης URL")

4.  Στη σελίδα **Ρύθμιση παραμέτρων Καθολικής σύνδεσης στο IdeaScale** , για να κάνετε λήψη του μετα-δεδομένων, κάντε κλικ στην επιλογή **Λήψη μετα-δεδομένων**και, στη συνέχεια, αποθηκεύστε το αρχείο μετα-δεδομένων τοπικά στον υπολογιστή σας.

    ![Ρύθμιση παραμέτρων Καθολικής σύνδεσης] (./media/active-directory-saas-ideascale-tutorial/IC790846.png "Ρύθμιση παραμέτρων Καθολικής σύνδεσης")

5.  Σε ένα παράθυρο προγράμματος περιήγησης web διαφορετικά, συνδεθείτε στην τοποθεσία της εταιρείας σας IdeaScale ως διαχειριστής.

6.  Μεταβείτε στις **Ρυθμίσεις Κοινότητας**.

    ![Ρυθμίσεις Κοινότητας] (./media/active-directory-saas-ideascale-tutorial/IC790847.png "Ρυθμίσεις Κοινότητας")

7.  Μεταβείτε στις επιλογές **ασφαλείας \> μονό πραγματοποιώ ρυθμίσεις**.

    ![Ρυθμίσεις μόνο πραγματοποιώ] (./media/active-directory-saas-ideascale-tutorial/IC790848.png "Ρυθμίσεις μόνο πραγματοποιώ")

8.  Ως **Πραγματοποιώ ενός τύπου**, επιλέξτε **SAML 2.0**.

    ![Μόνο πραγματοποιώ τύπου] (./media/active-directory-saas-ideascale-tutorial/IC790849.png "Μόνο πραγματοποιώ τύπου")

9.  Στο παράθυρο διαλόγου **Ρυθμίσεις καθολικής πραγματοποιώ** , ακολουθήστε τα παρακάτω βήματα:

    ![Ρυθμίσεις μόνο πραγματοποιώ] (./media/active-directory-saas-ideascale-tutorial/IC790850.png "Ρυθμίσεις μόνο πραγματοποιώ")

    1.  Στην κλασική πύλη Azure, στη σελίδα παράθυρο διαλόγου **Ρύθμιση παραμέτρων Καθολικής σύνδεσης στο IdeaScale** , αντιγράψτε την τιμή **Αναγνωριστικό οντότητας** και στη συνέχεια επικολλήστε το στο πλαίσιο κειμένου **Το Αναγνωριστικό οντότητας IdP SAML** .
    2.  Αντιγράψτε το περιεχόμενο του αρχείου λήψης μετα-δεδομένων και, στη συνέχεια επικολλήστε το στο πλαίσιο κειμένου **SAML μετα-δεδομένων IdP** .
    3.  Στην κλασική πύλη Azure, στη σελίδα παράθυρο διαλόγου **Ρύθμιση παραμέτρων Καθολικής σύνδεσης στο IdeaScale** , αντιγράψτε την τιμή **Διεύθυνση URL απομακρυσμένης αποσυνδεθείτε** και κατόπιν επικολλήστε το στο πλαίσιο κειμένου **Διεύθυνση URL επιτυχίας αποσυνδεθείτε** .
    4.  Κάντε κλικ στην επιλογή **Αποθήκευση αλλαγών**.

10. Στην κλασική Azure πύλη, επιλέξτε την καθολική σύνδεση ρύθμιση των παραμέτρων επιβεβαίωσης και, στη συνέχεια, κάντε κλικ στην επιλογή **Ολοκλήρωση** , για να κλείσετε το παράθυρο διαλόγου **Ρύθμιση παραμέτρων Καθολικής σύνδεσης** .

    ![Ρύθμιση παραμέτρων Καθολικής σύνδεσης] (./media/active-directory-saas-ideascale-tutorial/IC790851.png "Ρύθμιση παραμέτρων Καθολικής σύνδεσης")
##<a name="configuring-user-provisioning"></a>Ρύθμιση παραμέτρων προμήθεια του χρήστη
  
Για να επιτρέψετε στους χρήστες Azure AD για να συνδεθείτε στο IdeaScale, πρέπει να παρασχεθεί στο IdeaScale.  
Στην περίπτωση IdeaScale, προμήθεια είναι μια μη αυτόματη εργασία.

###<a name="to-configure-user-provisioning-perform-the-following-steps"></a>Για να ρυθμίσετε την προμήθεια του χρήστη, εκτελέστε τα ακόλουθα βήματα:

1.  Συνδεθείτε στην τοποθεσία της εταιρείας σας **IdeaScale** ως διαχειριστής.

2.  Μεταβείτε στις **Ρυθμίσεις Κοινότητας**.

    ![Ρυθμίσεις Κοινότητας] (./media/active-directory-saas-ideascale-tutorial/IC790847.png "Ρυθμίσεις Κοινότητας")

3.  Μεταβείτε στις επιλογές **βασικές ρυθμίσεις \> Διαχείριση μελών**.

4.  Κάντε κλικ στην επιλογή **Προσθήκη μέλους**.

    ![Διαχείριση μελών] (./media/active-directory-saas-ideascale-tutorial/IC790852.png "Διαχείριση μελών")

5.  Στην ενότητα Προσθήκη νέο μέλος, ακολουθήστε τα παρακάτω βήματα:

    ![Προσθήκη νέου μέλους] (./media/active-directory-saas-ideascale-tutorial/IC790853.png "Προσθήκη νέου μέλους")

    1.  Στο πλαίσιο κειμένου **Διευθύνσεις ηλεκτρονικού ταχυδρομείου** , πληκτρολογήστε τη διεύθυνση ηλεκτρονικού ταχυδρομείου από έναν έγκυρο λογαριασμό AAD που θέλετε για την παροχή των.
    2.  Κάντε κλικ στην επιλογή **Αποθήκευση αλλαγών**.

    >[AZURE.NOTE] Ο κάτοχος λογαριασμός Azure Active Directory θα λάβουν ένα μήνυμα ηλεκτρονικού ταχυδρομείου με μια σύνδεση για να επιβεβαιώσετε το λογαριασμό, προκειμένου να καταστεί ενεργή.

>[AZURE.NOTE] Μπορείτε να χρησιμοποιήσετε οποιοδήποτε άλλο IdeaScale χρήστη λογαριασμού εργαλεία δημιουργίας ή APIs που παρέχεται από IdeaScale σε λογαριασμούς χρηστών AAD διάταξη.

##<a name="assigning-users"></a>Αντιστοίχιση χρηστών
  
Για να ελέγξετε τη ρύθμιση των παραμέτρων σας, πρέπει να εκχωρήσετε Azure AD χρηστών που θέλετε να επιτρέψετε χρησιμοποιώντας την εφαρμογή έχουν πρόσβαση σε αυτό αναθέτοντας σε αυτά.

###<a name="to-assign-users-to-ideascale-perform-the-following-steps"></a>Για να εκχωρήσετε στους χρήστες να IdeaScale, εκτελέστε τα ακόλουθα βήματα:

1.  Στην κλασική Azure πύλη, δημιουργήστε ένα λογαριασμό δοκιμής.

2.  Στη σελίδα **IdeaScale **ενοποίησης της εφαρμογής, κάντε κλικ στην επιλογή **εκχωρήσετε στους χρήστες**.

    ![Αντιστοίχιση χρηστών] (./media/active-directory-saas-ideascale-tutorial/IC790854.png "Αντιστοίχιση χρηστών")

3.  Επιλέξτε το χρήστη έλεγχος, κάντε κλικ στην επιλογή **Εκχώρηση**και, στη συνέχεια, κάντε κλικ στο κουμπί **Ναι** για να επιβεβαιώσετε την ανάθεση.

    ![Ναι] (./media/active-directory-saas-ideascale-tutorial/IC767830.png "Ναι")
  
Εάν θέλετε να ελέγξετε τις ρυθμίσεις καθολικής εισόδου, ανοίξτε τον πίνακα της Access. Για περισσότερες λεπτομέρειες σχετικά με τον πίνακα της Access, ανατρέξτε στο θέμα [Εισαγωγή στο πλαίσιο πρόσβασης](active-directory-saas-access-panel-introduction.md).
