#<a name="tutorial-azure-active-directory-integration-with-uservoice"></a>Πρόγραμμα εκμάθησης: Ενοποίηση καταλόγου Azure Active Directory με UserVoice
  
Είναι ο στόχος αυτού του προγράμματος εκμάθησης για να εμφανίσετε την ενοποίηση του Azure και UserVoice.  
Το σενάριο που περιγράφονται σε αυτό το πρόγραμμα εκμάθησης προϋποθέτει ότι έχετε ήδη τα ακόλουθα στοιχεία:

-   Μια έγκυρη συνδρομή Azure
-   Ένας μισθωτής UserVoice
  
Μετά την ολοκλήρωση αυτού του προγράμματος εκμάθησης, θα μπορείτε να καθολικής σύνδεσης σε μια εφαρμογή στην τοποθεσία της εταιρείας σας UserVoice (υπηρεσία παροχής που ξεκινούν από σύμβολο στην) ή με την [Εισαγωγή στο πλαίσιο πρόσβασης](active-directory-saas-access-panel-introduction.md)Azure AD χρηστών που έχουν εκχωρηθεί σε UserVoice.
  
Το σενάριο που περιγράφονται σε αυτό το πρόγραμμα εκμάθησης αποτελείται από τα ακόλουθα δομικά στοιχεία:

1.  Ενεργοποίηση την ενοποίηση εφαρμογής για UserVoice
2.  Ρύθμιση παραμέτρων Καθολικής σύνδεσης
3.  Ρύθμιση παραμέτρων προμήθεια του χρήστη
4.  Αντιστοίχιση χρηστών

![Σενάριο] (./media/active-directory-saas-uservoice-tutorial/IC777514.png "Σενάριο")

##<a name="enabling-the-application-integration-for-uservoice"></a>Ενεργοποίηση την ενοποίηση εφαρμογής για UserVoice
  
Είναι ο στόχος αυτής της ενότητας για να εφαρμόσετε διάρθρωση πώς μπορείτε να ενεργοποιήσετε την ενοποίηση εφαρμογής για το UserVoice.

###<a name="to-enable-the-application-integration-for-uservoice-perform-the-following-steps"></a>Για να ενεργοποιήσετε την ενοποίηση εφαρμογής για UserVoice, ακολουθήστε τα παρακάτω βήματα:

1.  Στην κλασική πύλη Azure, στο αριστερό παράθυρο περιήγησης, κάντε κλικ στην επιλογή **Υπηρεσία καταλόγου Active Directory**.

    ![Υπηρεσία καταλόγου Active Directory] (./media/active-directory-saas-uservoice-tutorial/IC700993.png "Υπηρεσία καταλόγου Active Directory")

2.  Από τη λίστα **καταλόγου** , επιλέξτε τον κατάλογο για την οποία θέλετε να ενεργοποιήσετε την ενοποίηση καταλόγου.

3.  Για να ανοίξετε τις εφαρμογές προβολή, στην προβολή του καταλόγου, κάντε κλικ στην επιλογή **εφαρμογές** στο μενού επάνω.

    ![Εφαρμογές] (./media/active-directory-saas-uservoice-tutorial/IC700994.png "Εφαρμογές")

4.  Κάντε κλικ στην επιλογή **Προσθήκη** στο κάτω μέρος της σελίδας.

    ![Προσθήκη εφαρμογής] (./media/active-directory-saas-uservoice-tutorial/IC749321.png "Προσθήκη εφαρμογής")

5.  Στο παράθυρο διαλόγου **Τι θέλετε να κάνετε** , κάντε κλικ στην επιλογή **Προσθήκη εφαρμογής από τη συλλογή**.

    ![Προσθήκη εφαρμογής από gallerry] (./media/active-directory-saas-uservoice-tutorial/IC749322.png "Προσθήκη εφαρμογής από gallerry")

6.  Στο **πλαίσιο αναζήτησης**, πληκτρολογήστε **UserVoice**.

    ![Συλλογή εφαρμογών] (./media/active-directory-saas-uservoice-tutorial/IC777513.png "Συλλογή εφαρμογών")

7.  Στο παράθυρο αποτελεσμάτων, επιλέξτε **UserVoice**και, στη συνέχεια, κάντε κλικ στην επιλογή **Ολοκλήρωση** , για να προσθέσετε την εφαρμογή.

    ![UserVoice] (./media/active-directory-saas-uservoice-tutorial/IC777810.png "UserVoice")

##<a name="configuring-single-sign-on"></a>Ρύθμιση παραμέτρων Καθολικής σύνδεσης
  
Στόχος αυτής της ενότητας είναι να περιγράψει τον τρόπο για να επιτρέψετε στους χρήστες να ελέγχουν την ταυτότητα UserVoice με το λογαριασμό στο Azure AD χρησιμοποιώντας Ομοσπονδία με βάση το πρωτόκολλο SAML.  
Ρύθμιση παραμέτρων Καθολικής σύνδεσης για UserVoice απαιτεί να ανακτήσετε μια τιμή αποτύπωση από ένα πιστοποιητικό.  
Εάν δεν είστε εξοικειωμένοι με αυτήν τη διαδικασία, ανατρέξτε στο θέμα [πώς μπορείτε να ανακτήσετε ένα πιστοποιητικό αποτύπωση τιμή](http://youtu.be/YKQF266SAxI).

###<a name="to-configure-single-sign-on-perform-the-following-steps"></a>Για να ρυθμίσετε τις παραμέτρους της καθολικής σύνδεσης, ακολουθήστε τα παρακάτω βήματα:

1.  Στην κλασική πύλη Azure, στη σελίδα ενοποίησης εφαρμογής **UserVoice** , κάντε κλικ στην επιλογή **Ρύθμιση παραμέτρων Καθολικής σύνδεσης** για να ανοίξετε το παράθυρο διαλόγου **Ρύθμιση παραμέτρων Καθολικής σύνδεσης** .

    ![Ρύθμιση παραμέτρων Καθολικής σύνδεσης] (./media/active-directory-saas-uservoice-tutorial/IC777515.png "Ρύθμιση παραμέτρων Καθολικής σύνδεσης")

2.  Στη σελίδα **Πώς θέλετε οι χρήστες να πραγματοποιούν είσοδο UserVoice** , επιλέξτε **Microsoft Azure AD καθολικής σύνδεσης**και, στη συνέχεια, κάντε κλικ στο κουμπί **Επόμενο**.

    ![Ρύθμιση παραμέτρων Καθολικής σύνδεσης] (./media/active-directory-saas-uservoice-tutorial/IC777516.png "Ρύθμιση παραμέτρων Καθολικής σύνδεσης")

3.  Στη σελίδα **Ρύθμιση παραμέτρων URL εφαρμογής** , στο πλαίσιο κειμένου **UserVoice εισόδου στη διεύθυνση URL** , πληκτρολογήστε τη διεύθυνση URL σας χρησιμοποιώντας το παρακάτω μοτίβο "*https://\<μισθωτή όνομα\>. UserVoice.com*", και, στη συνέχεια, κάντε κλικ στο κουμπί **Επόμενο**.

    ![Ρύθμιση παραμέτρων εφαρμογής διεύθυνσης URL] (./media/active-directory-saas-uservoice-tutorial/IC777517.png "Ρύθμιση παραμέτρων εφαρμογής διεύθυνσης URL")

4.  Στη σελίδα **Ρύθμιση παραμέτρων Καθολικής σύνδεσης στο UserVoice** , για να κάνετε λήψη του πιστοποιητικού, κάντε κλικ στην επιλογή **Λήψη πιστοποιητικού**και, στη συνέχεια, αποθηκεύστε το αρχείο πιστοποιητικού τοπικά ως **c:\\UserVoice.cer**.

    ![Ρύθμιση παραμέτρων Καθολικής σύνδεσης] (./media/active-directory-saas-uservoice-tutorial/IC777518.png "Ρύθμιση παραμέτρων Καθολικής σύνδεσης")

5.  Σε ένα παράθυρο προγράμματος περιήγησης web διαφορετικά, συνδεθείτε στην τοποθεσία της εταιρείας σας UserVoice ως διαχειριστής.

6.  Στη γραμμή εργαλείων στο επάνω μέρος, κάντε κλικ στην επιλογή ρυθμίσεις και, στη συνέχεια, επιλέξτε πύλη Web από το μενού.

    ![Ρυθμίσεις] (./media/active-directory-saas-uservoice-tutorial/IC777519.png "Ρυθμίσεις")

7.  Στην καρτέλα " **πύλη Web** ", στην ενότητα **Έλεγχος ταυτότητας χρήστη** , κάντε κλικ στην επιλογή **Επεξεργασία** για να ανοίξετε τη σελίδα του παραθύρου διαλόγου **Επεξεργασία τον έλεγχο ταυτότητας χρήστη**

    ![Πύλη Web] (./media/active-directory-saas-uservoice-tutorial/IC777520.png "Πύλη Web")

8.  Στη σελίδα του παραθύρου διαλόγου **Επεξεργασία τον έλεγχο ταυτότητας χρήστη** , εκτελέστε τα ακόλουθα βήματα:

    ![Επεξεργαστείτε τον έλεγχο ταυτότητας χρήστη] (./media/active-directory-saas-uservoice-tutorial/IC777521.png "Επεξεργαστείτε τον έλεγχο ταυτότητας χρήστη")

    1.  Κάντε κλικ στην επιλογή **Καθολικής σύνδεσης (SSO)**.
    2.  Στην κλασική πύλη Azure, στη σελίδα παράθυρο διαλόγου **Ρύθμιση παραμέτρων Καθολικής σύνδεσης στο UserVoice** , αντιγράψτε την τιμή **Διεύθυνση URL απομακρυσμένης σύνδεσης** και κατόπιν επικολλήστε το στο πλαίσιο κειμένου **SSO απομακρυσμένης εισόδου** .
    3.  Στην κλασική πύλη Azure, στη σελίδα παράθυρο διαλόγου **Ρύθμιση παραμέτρων Καθολικής σύνδεσης στο UserVoice** , αντιγράψτε την τιμή **Διεύθυνση URL απομακρυσμένης αποσυνδεθείτε** και κατόπιν επικολλήστε το στο πλαίσιο **SSO απομακρυσμένης Sign-Out πλαίσιο κειμένου**.
    4.  Αντιγράψτε την τιμή **αποτύπωση** από το πιστοποιητικό που έχει εξαχθεί και, στη συνέχεια επικολλήστε το στο πλαίσιο κειμένου **τρέχουσα αποτυπώματος SHA1 πιστοποιητικού** .  

        >[AZURE.TIP] Για περισσότερες λεπτομέρειες, ανατρέξτε στο θέμα [πώς μπορείτε να ανακτήσετε ένα πιστοποιητικό αποτύπωση τιμή](http://youtu.be/YKQF266SAxI)

    5.  Κάντε κλικ στην επιλογή **Αποθήκευση ρυθμίσεις ελέγχου ταυτότητας**.

9.  Στην κλασική Azure πύλη, επιλέξτε την καθολική σύνδεση παραμέτρων επιβεβαίωσης και, στη συνέχεια, κάντε κλικ στην επιλογή **Ολοκλήρωση** , για να κλείσετε το παράθυρο διαλόγου **Ρύθμιση παραμέτρων Καθολικής σύνδεσης** .

    ![Ρύθμιση παραμέτρων Καθολικής σύνδεσης] (./media/active-directory-saas-uservoice-tutorial/IC777522.png "Ρύθμιση παραμέτρων Καθολικής σύνδεσης")

##<a name="configuring-user-provisioning"></a>Ρύθμιση παραμέτρων προμήθεια του χρήστη
  
Για να επιτρέψετε στους χρήστες Azure AD για να συνδεθείτε στο UserVoice, πρέπει να παρασχεθεί στο UserVoice.  
Στην περίπτωση UserVoice, προμήθεια είναι μια μη αυτόματη εργασία.

###<a name="to-provision-a-user-accounts-perform-the-following-steps"></a>Η παροχή μιας λογαριασμούς χρήστη, εκτελέστε τα ακόλουθα βήματα:

1.  Συνδεθείτε στο μισθωτή σας **UserVoice** .

2.  Μεταβείτε στις **Ρυθμίσεις**.

    ![Ρυθμίσεις] (./media/active-directory-saas-uservoice-tutorial/IC777811.png "Ρυθμίσεις")

3.  Κάντε κλικ στην επιλογή **Γενικά**.

4.  Κάντε κλικ στην επιλογή **παράγοντες και δικαιώματα**.

    ![Παράγοντες και δικαιώματα] (./media/active-directory-saas-uservoice-tutorial/IC777812.png "Παράγοντες και δικαιώματα")

5.  Κάντε κλικ στην επιλογή **Προσθήκη διαχειριστές**.

    ![Προσθήκη διαχειριστών] (./media/active-directory-saas-uservoice-tutorial/IC777813.png "Προσθήκη διαχειριστών")

6.  Στο παράθυρο διαλόγου **πρόσκληση διαχειριστές** , ακολουθήστε τα παρακάτω βήματα:

    ![Πρόσκληση διαχειριστές] (./media/active-directory-saas-uservoice-tutorial/IC777814.png "Πρόσκληση διαχειριστές")

    1.  Στο texbox τα μηνύματα ηλεκτρονικού ταχυδρομείου, πληκτρολογήστε τη διεύθυνση ηλεκτρονικού ταχυδρομείου του λογαριασμού που θέλετε για την παροχή των και, στη συνέχεια, κάντε κλικ στην επιλογή **Προσθήκη**.
    2.  Κάντε κλικ στην **πρόσκληση**.

>[AZURE.NOTE] Μπορείτε να χρησιμοποιήσετε οποιοδήποτε άλλο UserVoice χρήστη λογαριασμού εργαλεία δημιουργίας ή APIs που παρέχεται από UserVoice σε λογαριασμούς χρηστών AAD διάταξη.

##<a name="assigning-users"></a>Αντιστοίχιση χρηστών
  
Για να ελέγξετε τη ρύθμιση των παραμέτρων σας, πρέπει να εκχωρήσετε Azure AD χρηστών που θέλετε να επιτρέψετε χρησιμοποιώντας την εφαρμογή έχουν πρόσβαση σε αυτό αναθέτοντας σε αυτά.

###<a name="to-assign-users-to-uservoice-perform-the-following-steps"></a>Για να εκχωρήσετε στους χρήστες να UserVoice, εκτελέστε τα ακόλουθα βήματα:

1.  Στην κλασική Azure πύλη, δημιουργήστε ένα λογαριασμό δοκιμής.

2.  Στη σελίδα **UserVoice **ενοποίησης της εφαρμογής, κάντε κλικ στην επιλογή **εκχωρήσετε στους χρήστες**.

    ![Αντιστοίχιση χρηστών] (./media/active-directory-saas-uservoice-tutorial/IC777523.png "Αντιστοίχιση χρηστών")

3.  Επιλέξτε το χρήστη έλεγχος, κάντε κλικ στην επιλογή **Εκχώρηση**και, στη συνέχεια, κάντε κλικ στο κουμπί **Ναι** για να επιβεβαιώσετε την ανάθεση.

    ![Ναι] (./media/active-directory-saas-uservoice-tutorial/IC767830.png "Ναι")
  
Εάν θέλετε να ελέγξετε τις ρυθμίσεις καθολικής εισόδου, ανοίξτε τον πίνακα της Access. Για περισσότερες λεπτομέρειες σχετικά με τον πίνακα της Access, ανατρέξτε στο θέμα [Εισαγωγή στο πλαίσιο πρόσβασης](active-directory-saas-access-panel-introduction.md).
