#<a name="tutorial-azure-active-directory-integration-with-kintone"></a>Πρόγραμμα εκμάθησης: Ενοποίηση καταλόγου Azure Active Directory με Kintone
  
Είναι ο στόχος αυτού του προγράμματος εκμάθησης για να εμφανίσετε την ενοποίηση του Azure και Kintone.  
Το σενάριο που περιγράφονται σε αυτό το πρόγραμμα εκμάθησης προϋποθέτει ότι έχετε ήδη τα ακόλουθα στοιχεία:

-   Μια έγκυρη συνδρομή Azure
-   Μια Kintone καθολικής σύνδεσης με δυνατότητα συνδρομή
  
Μετά την ολοκλήρωση αυτού του προγράμματος εκμάθησης, θα μπορείτε να καθολικής σύνδεσης σε μια εφαρμογή στην τοποθεσία της εταιρείας σας Kintone (υπηρεσία παροχής που ξεκινούν από σύμβολο στην) ή με την [Εισαγωγή στο πλαίσιο πρόσβασης](active-directory-saas-access-panel-introduction.md)Azure AD χρηστών που έχουν εκχωρηθεί σε Kintone.
  
Το σενάριο που περιγράφονται σε αυτό το πρόγραμμα εκμάθησης αποτελείται από τα εξής δομικά στοιχεία:

1.  Ενεργοποίηση την ενοποίηση εφαρμογής για Kintone
2.  Ρύθμιση παραμέτρων Καθολικής σύνδεσης
3.  Ρύθμιση παραμέτρων προμήθεια του χρήστη
4.  Αντιστοίχιση χρηστών

![Σενάριο] (./media/active-directory-saas-kintone-tutorial/IC785859.png "Σενάριο")
##<a name="enabling-the-application-integration-for-kintone"></a>Ενεργοποίηση την ενοποίηση εφαρμογής για Kintone
  
Είναι ο στόχος αυτής της ενότητας για να εφαρμόσετε διάρθρωση πώς μπορείτε να ενεργοποιήσετε την ενοποίηση εφαρμογής για το Kintone.

###<a name="to-enable-the-application-integration-for-kintone-perform-the-following-steps"></a>Για να ενεργοποιήσετε την ενοποίηση εφαρμογής για Kintone, ακολουθήστε τα παρακάτω βήματα:

1.  Στην κλασική πύλη Azure, στο αριστερό παράθυρο περιήγησης, κάντε κλικ στην επιλογή **Υπηρεσία καταλόγου Active Directory**.

    ![Υπηρεσία καταλόγου Active Directory] (./media/active-directory-saas-kintone-tutorial/IC700993.png "Υπηρεσία καταλόγου Active Directory")

2.  Από τη λίστα **καταλόγου** , επιλέξτε τον κατάλογο για την οποία θέλετε να ενεργοποιήσετε την ενοποίηση καταλόγου.

3.  Για να ανοίξετε τις εφαρμογές προβολή, στην προβολή του καταλόγου, κάντε κλικ στην επιλογή **εφαρμογές** στο μενού επάνω.

    ![Εφαρμογές] (./media/active-directory-saas-kintone-tutorial/IC700994.png "Εφαρμογές")

4.  Κάντε κλικ στην επιλογή **Προσθήκη** στο κάτω μέρος της σελίδας.

    ![Προσθήκη εφαρμογής] (./media/active-directory-saas-kintone-tutorial/IC749321.png "Προσθήκη εφαρμογής")

5.  Στο παράθυρο διαλόγου **Τι θέλετε να κάνετε** , κάντε κλικ στην επιλογή **Προσθήκη εφαρμογής από τη συλλογή**.

    ![Προσθήκη εφαρμογής από gallerry] (./media/active-directory-saas-kintone-tutorial/IC749322.png "Προσθήκη εφαρμογής από gallerry")

6.  Στο **πλαίσιο αναζήτησης**, πληκτρολογήστε **Kintone**.

    ![Συλλογή εφαρμογών] (./media/active-directory-saas-kintone-tutorial/IC785867.png "Συλλογή εφαρμογών")

7.  Στο παράθυρο αποτελεσμάτων, επιλέξτε **Kintone**και, στη συνέχεια, κάντε κλικ στην επιλογή **Ολοκλήρωση** , για να προσθέσετε την εφαρμογή.

    ![Kintone] (./media/active-directory-saas-kintone-tutorial/IC785871.png "Kintone")
##<a name="configuring-single-sign-on"></a>Ρύθμιση παραμέτρων Καθολικής σύνδεσης
  
Στόχος αυτής της ενότητας είναι να περιγράψει τον τρόπο για να επιτρέψετε στους χρήστες να ελέγχουν την ταυτότητα Kintone με το λογαριασμό στο Azure AD χρησιμοποιώντας Ομοσπονδία με βάση το πρωτόκολλο SAML.

###<a name="to-configure-single-sign-on-perform-the-following-steps"></a>Για να ρυθμίσετε τις παραμέτρους της καθολικής σύνδεσης, ακολουθήστε τα παρακάτω βήματα:

1.  Στην κλασική πύλη Azure, στη σελίδα ενοποίησης εφαρμογής **Kintone** , κάντε κλικ στην επιλογή **Ρύθμιση παραμέτρων Καθολικής σύνδεσης** για να ανοίξετε το παράθυρο διαλόγου **Ρύθμιση παραμέτρων Καθολικής σύνδεσης** .

    ![Ρύθμιση παραμέτρων Καθολικής σύνδεσης] (./media/active-directory-saas-kintone-tutorial/IC785872.png "Ρύθμιση παραμέτρων Καθολικής σύνδεσης")

2.  Στη σελίδα **Πώς θέλετε οι χρήστες να πραγματοποιούν είσοδο Kintone** , επιλέξτε **Microsoft Azure AD καθολικής σύνδεσης**και, στη συνέχεια, κάντε κλικ στο κουμπί **Επόμενο**.

    ![Ρύθμιση παραμέτρων Καθολικής σύνδεσης] (./media/active-directory-saas-kintone-tutorial/IC785873.png "Ρύθμιση παραμέτρων Καθολικής σύνδεσης")

3.  Στη σελίδα **Ρύθμιση παραμέτρων URL εφαρμογής** , στο πλαίσιο κειμένου **Kintone εισόδου στη διεύθυνση URL** , πληκτρολογήστε τη διεύθυνση URL σας χρησιμοποιώντας το παρακάτω μοτίβο "*https://company.kintone.com*" και, στη συνέχεια, κάντε κλικ στο κουμπί **Επόμενο**.

    ![Ρύθμιση παραμέτρων εφαρμογής διεύθυνσης URL] (./media/active-directory-saas-kintone-tutorial/IC785875.png "Ρύθμιση παραμέτρων εφαρμογής διεύθυνσης URL")

4.  Στη σελίδα **Ρύθμιση παραμέτρων Καθολικής σύνδεσης στο Kintone** , για να κάνετε λήψη του πιστοποιητικού, κάντε κλικ στην επιλογή **λήψη πιστοποιητικό**και, στη συνέχεια, αποθηκεύστε το αρχείο πιστοποιητικού στον υπολογιστή σας.

    ![Ρύθμιση παραμέτρων Καθολικής σύνδεσης] (./media/active-directory-saas-kintone-tutorial/IC785878.png "Ρύθμιση παραμέτρων Καθολικής σύνδεσης")

5.  Σε ένα παράθυρο προγράμματος περιήγησης web διαφορετικά, συνδεθείτε στην τοποθεσία της εταιρείας σας **Kintone** ως διαχειριστής.

6.  Κάντε κλικ στην επιλογή **Ρυθμίσεις**.

    ![Ρυθμίσεις] (./media/active-directory-saas-kintone-tutorial/IC785879.png "Ρυθμίσεις")

7.  Κάντε κλικ στην επιλογή **χρήστες και διαχείριση του συστήματος**.

    ![Χρήστες και διαχείριση του συστήματος] (./media/active-directory-saas-kintone-tutorial/IC785880.png "Χρήστες και διαχείριση του συστήματος")

8.  Στην περιοχή **σύστημα διαχείρισης \> ασφαλείας** κάντε κλικ στην επιλογή **Login**.

    ![Σύνδεση] (./media/active-directory-saas-kintone-tutorial/IC785881.png "Σύνδεση")

9.  Κάντε κλικ στην επιλογή **Έλεγχος ταυτότητας SAML Ενεργοποίηση**.

    ![Έλεγχος ταυτότητας SAML] (./media/active-directory-saas-kintone-tutorial/IC785882.png "Έλεγχος ταυτότητας SAML")

10. Στην ενότητα έλεγχος ταυτότητας SAML, ακολουθήστε τα παρακάτω βήματα:

    ![Έλεγχος ταυτότητας SAML] (./media/active-directory-saas-kintone-tutorial/IC785883.png "Έλεγχος ταυτότητας SAML")

    1.  Στην κλασική πύλη Azure, στη σελίδα παράθυρο διαλόγου **Ρύθμιση παραμέτρων Καθολικής σύνδεσης στο Kintone** , αντιγράψτε την τιμή **Διεύθυνση URL απομακρυσμένης σύνδεσης** και, στη συνέχεια, επικολλήστε τη **Διεύθυνση URL σύνδεσης** πλαισίου κειμένου.
    2.  Στην κλασική πύλη Azure, στη σελίδα παράθυρο διαλόγου **Ρύθμιση παραμέτρων Καθολικής σύνδεσης στο Kintone** , αντιγράψτε την τιμή **Διεύθυνση URL απομακρυσμένης αποσυνδεθείτε** και κατόπιν επικολλήστε το στο πλαίσιο κειμένου **Διεύθυνση URL αποσυνδεθείτε** .
    3.  Κάντε κλικ στο κουμπί **Αναζήτηση** για να αποστείλετε το πιστοποιητικό που έχετε λάβει.
    4.  Κάντε κλικ στην επιλογή **Αποθήκευση**.

11. Στην κλασική Azure πύλη, επιλέξτε την καθολική σύνδεση ρύθμιση των παραμέτρων επιβεβαίωσης και, στη συνέχεια, κάντε κλικ στην επιλογή **Ολοκλήρωση** , για να κλείσετε το παράθυρο διαλόγου **Ρύθμιση παραμέτρων Καθολικής σύνδεσης** .

    ![Ρύθμιση παραμέτρων Καθολικής σύνδεσης] (./media/active-directory-saas-kintone-tutorial/IC785884.png "Ρύθμιση παραμέτρων Καθολικής σύνδεσης")
##<a name="configuring-user-provisioning"></a>Ρύθμιση παραμέτρων προμήθεια του χρήστη
  
Για να επιτρέψετε στους χρήστες Azure AD για να συνδεθείτε στο Kintone, πρέπει να παρασχεθεί στο Kintone.  
Στην περίπτωση Kintone, προμήθεια είναι μια μη αυτόματη εργασία.

###<a name="to-provision-a-user-accounts-perform-the-following-steps"></a>Η παροχή μιας λογαριασμούς χρήστη, εκτελέστε τα ακόλουθα βήματα:

1.  Συνδεθείτε στην τοποθεσία της εταιρείας σας **Kintone** ως διαχειριστής.

2.  Κάντε κλικ στην επιλογή **Ρύθμιση**.

    ![Ρυθμίσεις] (./media/active-directory-saas-kintone-tutorial/IC785879.png "Ρυθμίσεις")

3.  Κάντε κλικ στην επιλογή **χρήστες και διαχείριση του συστήματος**.

    ![Διαχείριση του συστήματος & χρήστη] (./media/active-directory-saas-kintone-tutorial/IC785880.png "Διαχείριση του συστήματος & χρήστη")

4.  Στην περιοχή **Διαχείριση του χρήστη**, κάντε κλικ στην επιλογή **τμήματα & χρήστες**.

    ![Τμήμα & χρήστες] (./media/active-directory-saas-kintone-tutorial/IC785888.png "Τμήμα & χρήστες")

5.  Κάντε κλικ στην επιλογή **νέο χρήστη**.

    ![Νέοι χρήστες] (./media/active-directory-saas-kintone-tutorial/IC785889.png "Νέοι χρήστες")

6.  Στην ενότητα **Νέο χρήστη** , εκτελέστε τα ακόλουθα βήματα:

    ![Νέοι χρήστες] (./media/active-directory-saas-kintone-tutorial/IC785890.png "Νέοι χρήστες")

    1.  Πληκτρολογήστε ένα **Εμφανιζόμενο όνομα**, **Όνομα σύνδεσης**, **Νέο κωδικό πρόσβασης**, **Επιβεβαιώστε τον κωδικό πρόσβασης**, **Διεύθυνση ηλεκτρονικού ταχυδρομείου** και άλλες λεπτομέρειες της έναν έγκυρο λογαριασμό AAD που θέλετε για την παροχή των σε το σχετικό texboxes.
    2.  Κάντε κλικ στην επιλογή **Αποθήκευση**.

>[AZURE.NOTE] Μπορείτε να χρησιμοποιήσετε οποιοδήποτε άλλο Kintone χρήστη λογαριασμού εργαλεία δημιουργίας ή APIs που παρέχεται από Kintone σε λογαριασμούς χρηστών AAD διάταξη.

##<a name="assigning-users"></a>Αντιστοίχιση χρηστών
  
Για να ελέγξετε τη ρύθμιση των παραμέτρων σας, πρέπει να εκχωρήσετε Azure AD χρηστών που θέλετε να επιτρέψετε χρησιμοποιώντας την εφαρμογή έχουν πρόσβαση σε αυτό αναθέτοντας σε αυτά.

###<a name="to-assign-users-to-kintone-perform-the-following-steps"></a>Για να εκχωρήσετε στους χρήστες να Kintone, εκτελέστε τα ακόλουθα βήματα:

1.  Στην κλασική Azure πύλη, δημιουργήστε ένα λογαριασμό δοκιμής.

2.  Στη σελίδα **Kintone **ενοποίησης της εφαρμογής, κάντε κλικ στην επιλογή **εκχωρήσετε στους χρήστες**.

    ![Αντιστοίχιση χρηστών] (./media/active-directory-saas-kintone-tutorial/IC785891.png "Αντιστοίχιση χρηστών")

3.  Επιλέξτε το χρήστη έλεγχος, κάντε κλικ στην επιλογή **Εκχώρηση**και, στη συνέχεια, κάντε κλικ στο κουμπί **Ναι** για να επιβεβαιώσετε την ανάθεση.

    ![Ναι] (./media/active-directory-saas-kintone-tutorial/IC767830.png "Ναι")
  
Εάν θέλετε να ελέγξετε τις ρυθμίσεις καθολικής εισόδου, ανοίξτε τον πίνακα της Access. Για περισσότερες λεπτομέρειες σχετικά με τον πίνακα της Access, ανατρέξτε στο θέμα [Εισαγωγή στο πλαίσιο πρόσβασης](active-directory-saas-access-panel-introduction.md).
