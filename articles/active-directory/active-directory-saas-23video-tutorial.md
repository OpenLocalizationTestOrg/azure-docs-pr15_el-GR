# <a name="tutorial-azure-active-directory-integration-with-23-video"></a>Πρόγραμμα εκμάθησης: Ενοποίηση καταλόγου Azure Active Directory με το 23 βίντεο

Ο στόχος αυτού του προγράμματος εκμάθησης είναι να σας δείξουν πώς μπορείτε να ενοποιήσετε το 23 βίντεο με το Azure Active Directory (Azure AD).  
Ενοποίηση 23 βίντεο με Azure AD σάς παρέχει τα ακόλουθα πλεονεκτήματα: 

- Μπορείτε να ελέγξετε σε Azure AD ποιος έχει πρόσβαση σε 23 βίντεο 
- Μπορείτε να ενεργοποιήσετε τους χρήστες σας να αυτόματα να πραγματοποιήσει καθολικής 23 βίντεο (καθολικής σύνδεσης) με τους λογαριασμούς Azure AD

Εάν θέλετε να μάθετε περισσότερες λεπτομέρειες σχετικά με ΑΔΑ εφαρμογή ενοποίηση με το Azure AD, ανατρέξτε στο θέμα [Τι είναι η εφαρμογή access και καθολικής σύνδεσης με το Azure Active Directory](active-directory-appssoaccess-whatis.md).

## <a name="prerequisites"></a>Προαπαιτούμενα στοιχεία 

Για να ρυθμίσετε τις παραμέτρους Azure AD ενοποίηση με το 23 βίντεο, χρειάζεστε τα ακόλουθα στοιχεία:

- Μια συνδρομή του Azure AD
- Μια 23 βίντεο καθολικής σύνδεσης enabled συνδρομής


> [AZURE.NOTE] Για να ελέγξετε τα βήματα που περιγράφονται σε αυτό το πρόγραμμα εκμάθησης, δεν συνιστάται να χρησιμοποιείτε ένα περιβάλλον παραγωγής.


Για να ελέγξετε τα βήματα που περιγράφονται σε αυτό το πρόγραμμα εκμάθησης, θα πρέπει να ακολουθήσετε αυτές τις συστάσεις:

- Δεν πρέπει να χρησιμοποιείτε το περιβάλλον παραγωγής, εκτός εάν αυτό είναι απαραίτητο.
- Εάν δεν έχετε ένα περιβάλλον δοκιμαστική Azure AD, μπορείτε να αποκτήσετε μια μηνιαία δοκιμαστική [εδώ](https://azure.microsoft.com/pricing/free-trial/). 

 
## <a name="scenario-description"></a>Σενάριο περιγραφή
Είναι ο στόχος αυτού του προγράμματος εκμάθησης για να ελέγξετε Azure AD καθολικής σύνδεσης σε περιβάλλον δοκιμής.  
Το σενάριο που περιγράφονται σε αυτό το πρόγραμμα εκμάθησης αποτελείται από δύο κύριο μπλοκ δόμησης:

1. Προσθήκη 23 βίντεο από τη συλλογή 
2. Ρύθμιση παραμέτρων και έλεγχος Azure AD απλή σύνδεση


## <a name="adding-23-video-from-the-gallery"></a>Προσθήκη 23 βίντεο από τη συλλογή
Για να ρυθμίσετε την ενσωμάτωση 23 βίντεο στην Azure AD, πρέπει να προσθέσετε 23 βίντεο από τη συλλογή στη λίστα των διαχειριζόμενων ΑΔΑ εφαρμογών.

**Για να προσθέσετε 23 βίντεο από τη συλλογή, εκτελέστε τα ακόλουθα βήματα:**

1. Στην **πύλη του Azure κλασική**, στο αριστερό παράθυρο περιήγησης, κάντε κλικ στην επιλογή **Υπηρεσία καταλόγου Active Directory**. 

    ![Υπηρεσία καταλόγου Active Directory][1]

2. Από τη λίστα **καταλόγου** , επιλέξτε τον κατάλογο για την οποία θέλετε να ενεργοποιήσετε την ενοποίηση καταλόγου.

3. Για να ανοίξετε τις εφαρμογές προβολή, στην προβολή του καταλόγου, κάντε κλικ στην επιλογή **εφαρμογές** στο μενού επάνω.

    ![Εφαρμογές][2]

4. Κάντε κλικ στην επιλογή **Προσθήκη** στο κάτω μέρος της σελίδας.

    ![Εφαρμογές][3]

5. Στο παράθυρο διαλόγου **Τι θέλετε να κάνετε** , κάντε κλικ στην επιλογή **Προσθήκη εφαρμογής από τη συλλογή**.

    ![Εφαρμογές][4]

6. Στο πλαίσιο αναζήτησης, πληκτρολογήστε **23 βίντεο**.

    ![Εφαρμογές][5]

7. Στο παράθυρο αποτελεσμάτων, επιλέξτε **23 βίντεο**και, στη συνέχεια, κάντε κλικ στην επιλογή **Ολοκλήρωση** , για να προσθέσετε την εφαρμογή.

    ![Εφαρμογές][25]

##  <a name="configuring-and-testing-azure-ad-single-sign-on"></a>Ρύθμιση παραμέτρων και έλεγχος Azure AD απλή σύνδεση
Ο στόχος αυτής της ενότητας είναι να σας δείξουν πώς μπορείτε να ρυθμίσετε τις παραμέτρους και να ελέγξετε Azure AD καθολικής σύνδεσης με το 23 βίντεο που βασίζονται σε ένα χρήστη δοκιμής που ονομάζεται "Britta Simon".

Για καθολικής σύνδεσης για να εργαστείτε, Azure AD πρέπει να γνωρίζετε ποιος είναι ο χρήστης αντίστοιχο σε 23 βίντεο σε ένα χρήστη σε Azure AD. Με άλλα λόγια, μια σχέση σύνδεση μεταξύ ενός χρήστη Azure AD και το σχετικό χρήστη στο 23 βίντεο πρέπει να δημιουργηθεί.  
Αυτή η σχέση σύνδεση είναι εγκατεστημένος κατά την αντιστοίχιση της τιμής του **ονόματος χρήστη** στο Azure AD ως τιμή του το **όνομα χρήστη** σε 23 βίντεο.
 
Για να ρυθμίσετε τις παραμέτρους και να ελέγξετε Azure AD καθολικής σύνδεσης με το 23 βίντεο, πρέπει να ολοκληρώσετε τα παρακάτω μπλοκ δόμησης:

1. **[Ρύθμιση παραμέτρων Azure AD καθολικής σύνδεσης](#configuring-azure-ad-single-single-sign-on)** - για να ενεργοποιήσετε τους χρήστες σας για να χρησιμοποιήσετε αυτήν τη δυνατότητα.
2. **[Δημιουργία μιας Azure AD δοκιμή χρήστη](#creating-an-azure-ad-test-user)** - για να ελέγξετε Azure AD καθολικής σύνδεσης με Britta Simon.
4. **[Δημιουργία βίντεο 23 δοκιμή χρήστη](#creating-a-23-video-test-user)** - έχουν αντίστοιχο του Britta Simon σε 23 βίντεο που είναι συνδεδεμένο με το Azure AD αναπαράσταση εκείνη.
5. **[Εκχώρηση του Azure AD δοκιμή χρήστη](#assigning-the-azure-ad-test-user)** - για να ενεργοποιήσετε την Britta Simon για να χρησιμοποιήσετε Azure AD καθολικής σύνδεσης.
5. **[Δοκιμές καθολικής σύνδεσης](#testing-single-sign-on)** - για να επιβεβαιώσετε αν λειτουργεί η ρύθμιση παραμέτρων.

### <a name="configuring-azure-ad-single-sign-on"></a>Ρύθμιση παραμέτρων Azure AD καθολικής σύνδεσης

Στόχος αυτής της ενότητας είναι για να ενεργοποιήσετε Azure AD καθολικής σύνδεσης στην πύλη του Azure κλασική και για τη ρύθμιση παραμέτρων Καθολικής σύνδεσης στην εφαρμογή σας 23 βίντεο.

**Για να ρυθμίσετε τις παραμέτρους Azure AD καθολικής σύνδεσης με το 23 βίντεο, εκτελέστε τα ακόλουθα βήματα:**

1. Στην κλασική πύλη Azure, στη σελίδα ενοποίησης εφαρμογής **23 βίντεο** , κάντε κλικ στην επιλογή **Ρύθμιση παραμέτρων Καθολικής σύνδεσης** για να ανοίξετε το παράθυρο διαλόγου **Ρύθμιση παραμέτρων Καθολικής σύνδεσης** .

    ![Ρύθμιση παραμέτρων Καθολικής σύνδεσης][6] 

2. Στη σελίδα **Πώς θα θέλατε χρήστες να πραγματοποιούν είσοδο με το 23 βίντεο** , επιλέξτε **Azure AD καθολικής σύνδεσης**και, στη συνέχεια, κάντε κλικ στο κουμπί **Επόμενο**.

    ![Azure AD καθολικής σύνδεσης][7] 

3. Στη σελίδα του παραθύρου διαλόγου **Διαμόρφωση παραμέτρων ρυθμίσεων εφαρμογής** , εκτελέστε τα ακόλουθα βήματα:

    ![Azure AD καθολικής σύνδεσης][8] 
 
     μια. Στο πλαίσιο κειμένου **Διεύθυνση URL απάντηση** , πληκτρολογήστε τη διεύθυνση URL που χρησιμοποιούνται από τους χρήστες σας για σύνδεση στην τοποθεσία σας 23 βίντεο (π.χ.: *https://britta-simon.23Video.com/saml/login*).

     > [AZURE.NOTE] Ενοποίησης Active Directory με χρήση SAML 2.0 είναι διαθέσιμη για όλους τους χρήστες 23 βίντεο. Επικοινωνήστε με την υποστήριξη στο [support@23company.com](mailto:support@23company.com) Εάν χρειάζεστε το σχετικό μετα-δεδομένων.

     β. Κάντε κλικ στο κουμπί **Επόμενο**.
 
4. Στη σελίδα **Ρύθμιση παραμέτρων Καθολικής σύνδεσης 23 βίντεο** , εκτελέστε τα ακόλουθα βήματα:

    ![Azure AD καθολικής σύνδεσης][9] 

    μια. Κάντε κλικ στην επιλογή λήψη πιστοποιητικό και, στη συνέχεια, αποθηκεύστε το αρχείο στον υπολογιστή σας.

    β. Επικοινωνήστε με το 23 ομάδα υποστήριξης βίντεο μέσω [support@23company.com](mailto:support@23company.com), δώστε τους με το πιστοποιητικό που έχετε λάβει, τη **Διεύθυνση URL εκδότη**, την **Καθολική σύνδεση διεύθυνση URL της υπηρεσίας**, **Μία διεύθυνση URL Sign-Out**, και, στη συνέχεια, ζητήστε τους να ρύθμισης SSO για την εφαρμογή σας 23 βίντεο. 

    c. Κάντε κλικ στο κουμπί **Επόμενο**.


6. Στην κλασική Azure πύλη, επιλέξτε την καθολική σύνδεση παραμέτρων επιβεβαίωσης και, στη συνέχεια, κάντε κλικ στο κουμπί **Επόμενο**. 

    ![Azure AD καθολικής σύνδεσης][10]

7. Στη σελίδα **επιβεβαίωσης καθολική σύνδεση** , κάντε κλικ στην επιλογή **ολοκληρώθηκε**.  
  
    ![Azure AD καθολικής σύνδεσης][11]
