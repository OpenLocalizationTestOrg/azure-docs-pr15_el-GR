#<a name="tutorial-azure-active-directory-integration-with-freshdesk"></a>Πρόγραμμα εκμάθησης: Ενοποίηση καταλόγου Azure Active Directory με Freshdesk
  
Είναι ο στόχος αυτού του προγράμματος εκμάθησης για να εμφανίσετε την ενοποίηση του Azure και Freshdesk.  
Το σενάριο που περιγράφονται σε αυτό το πρόγραμμα εκμάθησης προϋποθέτει ότι έχετε ήδη τα ακόλουθα στοιχεία:

-   Μια έγκυρη συνδρομή Azure
-   Ένας μισθωτής Freshdesk
  
Μετά την ολοκλήρωση αυτού του προγράμματος εκμάθησης, θα μπορείτε να καθολικής σύνδεσης σε μια εφαρμογή στην τοποθεσία της εταιρείας σας Freshdesk (υπηρεσία παροχής που ξεκινούν από σύμβολο στην) ή με την [Εισαγωγή στο πλαίσιο πρόσβασης](active-directory-saas-access-panel-introduction.md)Azure AD χρηστών που έχουν εκχωρηθεί σε Freshdesk.
  
Το σενάριο που περιγράφονται σε αυτό το πρόγραμμα εκμάθησης αποτελείται από τα εξής δομικά στοιχεία:

1.  Ενεργοποίηση την ενοποίηση εφαρμογής για Freshdesk
2.  Ρύθμιση παραμέτρων Καθολικής σύνδεσης
3.  Ρύθμιση παραμέτρων προμήθεια του χρήστη
4.  Αντιστοίχιση χρηστών

![Σενάριο] (./media/active-directory-saas-freshdesk-tutorial/IC776761.png "Σενάριο")
##<a name="enabling-the-application-integration-for-freshdesk"></a>Ενεργοποίηση την ενοποίηση εφαρμογής για Freshdesk
  
Είναι ο στόχος αυτής της ενότητας για να εφαρμόσετε διάρθρωση πώς μπορείτε να ενεργοποιήσετε την ενοποίηση εφαρμογής για το Freshdesk.

###<a name="to-enable-the-application-integration-for-freshdesk-perform-the-following-steps"></a>Για να ενεργοποιήσετε την ενοποίηση εφαρμογής για Freshdesk, ακολουθήστε τα παρακάτω βήματα:

1.  Στην κλασική πύλη Azure, στο αριστερό παράθυρο περιήγησης, κάντε κλικ στην επιλογή **Υπηρεσία καταλόγου Active Directory**.

    ![Υπηρεσία καταλόγου Active Directory] (./media/active-directory-saas-freshdesk-tutorial/IC700993.png "Υπηρεσία καταλόγου Active Directory")

2.  Από τη λίστα **καταλόγου** , επιλέξτε τον κατάλογο για την οποία θέλετε να ενεργοποιήσετε την ενοποίηση καταλόγου.

3.  Για να ανοίξετε τις εφαρμογές προβολή, στην προβολή του καταλόγου, κάντε κλικ στην επιλογή **εφαρμογές** στο μενού επάνω.

    ![Εφαρμογές] (./media/active-directory-saas-freshdesk-tutorial/IC700994.png "Εφαρμογές")

4.  Κάντε κλικ στην επιλογή **Προσθήκη** στο κάτω μέρος της σελίδας.

    ![Προσθήκη εφαρμογής] (./media/active-directory-saas-freshdesk-tutorial/IC749321.png "Προσθήκη εφαρμογής")

5.  Στο παράθυρο διαλόγου **Τι θέλετε να κάνετε** , κάντε κλικ στην επιλογή **Προσθήκη εφαρμογής από τη συλλογή**.

    ![Προσθήκη εφαρμογής από gallerry] (./media/active-directory-saas-freshdesk-tutorial/IC749322.png "Προσθήκη εφαρμογής από gallerry")

6.  Στο **πλαίσιο αναζήτησης**, πληκτρολογήστε **Freshdesk**.

    ![Συλλογή εφαρμογών] (./media/active-directory-saas-freshdesk-tutorial/IC776762.png "Συλλογή εφαρμογών")

7.  Στο παράθυρο αποτελεσμάτων, επιλέξτε **Freshdesk**και, στη συνέχεια, κάντε κλικ στην επιλογή **Ολοκλήρωση** , για να προσθέσετε την εφαρμογή.

    ![Freshdesk] (./media/active-directory-saas-freshdesk-tutorial/IC776763.png "Freshdesk")
##<a name="configuring-single-sign-on"></a>Ρύθμιση παραμέτρων Καθολικής σύνδεσης
  
Στόχος αυτής της ενότητας είναι να περιγράψει τον τρόπο για να επιτρέψετε στους χρήστες να ελέγχουν την ταυτότητα Freshdesk με το λογαριασμό στο Azure AD χρησιμοποιώντας Ομοσπονδία με βάση το πρωτόκολλο SAML.  
Ρύθμιση παραμέτρων Καθολικής σύνδεσης για Freshdesk απαιτεί να ανακτήσετε μια τιμή αποτύπωση από ένα πιστοποιητικό.  
Εάν δεν είστε εξοικειωμένοι με αυτήν τη διαδικασία, ανατρέξτε στο θέμα [πώς μπορείτε να ανακτήσετε ένα πιστοποιητικό αποτύπωση τιμή](http://youtu.be/YKQF266SAxI).

###<a name="to-configure-single-sign-on-perform-the-following-steps"></a>Για να ρυθμίσετε τις παραμέτρους της καθολικής σύνδεσης, ακολουθήστε τα παρακάτω βήματα:

1.  Στην κλασική πύλη Azure, στη σελίδα ενοποίησης εφαρμογής **Freshdesk** , κάντε κλικ στην επιλογή **Ρύθμιση παραμέτρων Καθολικής σύνδεσης** για να ανοίξετε το παράθυρο διαλόγου **Ρύθμιση παραμέτρων Καθολικής σύνδεσης** .

    ![Ρύθμιση παραμέτρων Καθολικής σύνδεσης] (./media/active-directory-saas-freshdesk-tutorial/IC776764.png "Ρύθμιση παραμέτρων Καθολικής σύνδεσης")

2.  Στη σελίδα **Πώς θέλετε οι χρήστες να πραγματοποιούν είσοδο Freshdesk** , επιλέξτε **Microsoft Azure AD καθολικής σύνδεσης**και, στη συνέχεια, κάντε κλικ στο κουμπί **Επόμενο**.

    ![Ρύθμιση παραμέτρων Καθολικής σύνδεσης] (./media/active-directory-saas-freshdesk-tutorial/IC776765.png "Ρύθμιση παραμέτρων Καθολικής σύνδεσης")

3.  Στη σελίδα **Ρύθμιση παραμέτρων URL εφαρμογής** , στο πλαίσιο κειμένου **Freshdesk εισόδου στη διεύθυνση URL** , πληκτρολογήστε τη διεύθυνση URL σας χρησιμοποιώντας το παρακάτω μοτίβο "*https://\<μισθωτή όνομα\>. Freshdesk.com*", και, στη συνέχεια, κάντε κλικ στο κουμπί **Επόμενο**.

    ![Ρύθμιση παραμέτρων εφαρμογής διεύθυνσης URL] (./media/active-directory-saas-freshdesk-tutorial/IC776766.png "Ρύθμιση παραμέτρων εφαρμογής διεύθυνσης URL")

4.  Στη σελίδα **Ρύθμιση παραμέτρων Καθολικής σύνδεσης στο Freshdesk** , για να κάνετε λήψη του πιστοποιητικού, κάντε κλικ στην επιλογή **Λήψη πιστοποιητικού**και, στη συνέχεια, αποθηκεύστε το αρχείο πιστοποιητικού τοπικά ως **c:\\Freshdesk.cer**.

    ![Ρύθμιση παραμέτρων Καθολικής σύνδεσης] (./media/active-directory-saas-freshdesk-tutorial/IC776767.png "Ρύθμιση παραμέτρων Καθολικής σύνδεσης")

5.  Σε ένα παράθυρο προγράμματος περιήγησης web διαφορετικά, συνδεθείτε στην τοποθεσία της εταιρείας σας Freshdesk ως διαχειριστής.

6.  Στο μενού στο επάνω μέρος, κάντε κλικ στην επιλογή **Διαχείριση**.

    ![Διαχειριστής] (./media/active-directory-saas-freshdesk-tutorial/IC776768.png "Διαχειριστής")

7.  Στην καρτέλα **Γενικές ρυθμίσεις** , κάντε κλικ στην επιλογή **ασφάλεια**.

    ![Ασφάλεια] (./media/active-directory-saas-freshdesk-tutorial/IC776769.png "Ασφάλεια")

8.  Στην ενότητα **ασφάλεια** , ακολουθήστε τα παρακάτω βήματα:

    ![Καθολική σύνδεση] (./media/active-directory-saas-freshdesk-tutorial/IC776770.png "Καθολική σύνδεση")

    1.  Για **Μία μόνο είσοδο σε (SSO)**, επιλέξτε **στο**.
    2.  Επιλέξτε **SAML SSO**.
    3.  Στην κλασική πύλη Azure, στη σελίδα παράθυρο διαλόγου **Ρύθμιση παραμέτρων Καθολικής σύνδεσης στο Freshdesk** , αντιγράψτε την τιμή **Διεύθυνση URL απομακρυσμένης σύνδεσης** και κατόπιν επικολλήστε το στο πλαίσιο κειμένου **Διεύθυνση URL σύνδεσης SAML** .
    4.  Στην κλασική πύλη Azure, στη σελίδα παράθυρο διαλόγου **Ρύθμιση παραμέτρων Καθολικής σύνδεσης στο Freshdesk** , αντιγράψτε την τιμή **Διεύθυνση URL απομακρυσμένης αποσυνδεθείτε** και κατόπιν επικολλήστε το στο πλαίσιο κειμένου **Διεύθυνση URL αποσυνδεθείτε** .
    5.  Αντιγράψτε την τιμή **αποτύπωση** από το πιστοποιητικό που έχει εξαχθεί και, στη συνέχεια, επικολλήστε την στο πλαίσιο κειμένου **Αποτυπώματος πιστοποιητικό ασφαλείας** .  

        >[AZURE.TIP]Για περισσότερες λεπτομέρειες, ανατρέξτε στο θέμα [πώς μπορείτε να ανακτήσετε ένα πιστοποιητικό αποτύπωση τιμή](http://youtu.be/YKQF266SAxI)

    6.  Κάντε κλικ στην επιλογή **Αποθήκευση**.

9.  Στην κλασική Azure πύλη, επιλέξτε την καθολική σύνδεση ρύθμιση των παραμέτρων επιβεβαίωσης και, στη συνέχεια, κάντε κλικ στην επιλογή **Ολοκλήρωση** , για να κλείσετε το παράθυρο διαλόγου **Ρύθμιση παραμέτρων Καθολικής σύνδεσης** .

    ![Ρύθμιση παραμέτρων Καθολικής σύνδεσης] (./media/active-directory-saas-freshdesk-tutorial/IC776771.png "Ρύθμιση παραμέτρων Καθολικής σύνδεσης")
##<a name="configuring-user-provisioning"></a>Ρύθμιση παραμέτρων προμήθεια του χρήστη
  
Για να επιτρέψετε στους χρήστες Azure AD για να συνδεθείτε στο Freshdesk, πρέπει να παρασχεθεί στο Freshdesk.  
Στην περίπτωση Freshdesk, προμήθεια είναι μια μη αυτόματη εργασία.

###<a name="to-provision-a-user-accounts-perform-the-following-steps"></a>Η παροχή μιας λογαριασμούς χρήστη, εκτελέστε τα ακόλουθα βήματα:

1.  Συνδεθείτε στο μισθωτή σας **Freshdesk** .

2.  Στο μενού στο επάνω μέρος, κάντε κλικ στην επιλογή **Διαχείριση**.

    ![Διαχειριστής] (./media/active-directory-saas-freshdesk-tutorial/IC776772.png "Διαχειριστής")

3.  Στην καρτέλα **Γενικές ρυθμίσεις** , κάντε κλικ στην επιλογή **παραγόντων**.

    ![Παράγοντες] (./media/active-directory-saas-freshdesk-tutorial/IC776773.png "Παράγοντες")

4.  Κάντε κλικ στην επιλογή **Δημιουργία παράγοντα**.

    ![Δημιουργία παράγοντα] (./media/active-directory-saas-freshdesk-tutorial/IC776774.png "Δημιουργία παράγοντα")

5.  Στο παράθυρο διαλόγου πληροφορίες παράγοντα, ακολουθήστε τα παρακάτω βήματα:

    ![Πληροφορίες παράγοντα] (./media/active-directory-saas-freshdesk-tutorial/IC776775.png "Πληροφορίες παράγοντα")

    1.  Στο πλαίσιο κειμένου **Πλήρες όνομα** , πληκτρολογήστε το όνομα του λογαριασμού Azure AD που θέλετε για την παροχή των.
    2.  Στο πλαίσιο κειμένου **ηλεκτρονικού ταχυδρομείου** , πληκτρολογήστε τη διεύθυνση ηλεκτρονικού ταχυδρομείου Azure AD του Azure AD λογαριασμού που θέλετε για την παροχή των.
    3.  Στο πλαίσιο κειμένου **Τίτλος** , πληκτρολογήστε τον τίτλο του Azure AD λογαριασμού που θέλετε για την παροχή των.
    4.  Επιλέξτε **το ρόλο παράγοντες**και, στη συνέχεια, κάντε κλικ στην επιλογή **Αντιστοίχιση**.
    5.  Κάντε κλικ στην επιλογή **Αποθήκευση**.
    
        >[AZURE.NOTE] Ο κάτοχος λογαριασμός Azure AD θα λάβουν ένα μήνυμα ηλεκτρονικού ταχυδρομείου που περιλαμβάνει μια σύνδεση για να επιβεβαιώσετε το λογαριασμό πριν είναι ενεργοποιημένο.

>[AZURE.NOTE] Μπορείτε να χρησιμοποιήσετε οποιοδήποτε άλλο Freshdesk χρήστη λογαριασμού εργαλεία δημιουργίας ή APIs που παρέχεται από Freshdesk σε λογαριασμούς χρηστών AAD διάταξη.

##<a name="assigning-users"></a>Αντιστοίχιση χρηστών
  
Για να ελέγξετε τη ρύθμιση των παραμέτρων σας, πρέπει να εκχωρήσετε Azure AD χρηστών που θέλετε να επιτρέψετε χρησιμοποιώντας την εφαρμογή έχουν πρόσβαση σε αυτό αναθέτοντας σε αυτά.

###<a name="to-assign-users-to-freshdesk-perform-the-following-steps"></a>Για να εκχωρήσετε στους χρήστες να Freshdesk, εκτελέστε τα ακόλουθα βήματα:

1.  Στην κλασική Azure πύλη, δημιουργήστε ένα λογαριασμό δοκιμής.

2.  Στη σελίδα **Freshdesk **ενοποίησης της εφαρμογής, κάντε κλικ στην επιλογή **εκχωρήσετε στους χρήστες**.

    ![Αντιστοίχιση χρηστών] (./media/active-directory-saas-freshdesk-tutorial/IC776776.png "Αντιστοίχιση χρηστών")

3.  Επιλέξτε το χρήστη έλεγχος, κάντε κλικ στην επιλογή **Εκχώρηση**και, στη συνέχεια, κάντε κλικ στο κουμπί **Ναι** για να επιβεβαιώσετε την ανάθεση.

    ![Ναι] (./media/active-directory-saas-freshdesk-tutorial/IC767830.png "Ναι")
  
Εάν θέλετε να ελέγξετε τις ρυθμίσεις καθολικής εισόδου, ανοίξτε τον πίνακα της Access. Για περισσότερες λεπτομέρειες σχετικά με τον πίνακα της Access, ανατρέξτε στο θέμα [Εισαγωγή στο πλαίσιο πρόσβασης](active-directory-saas-access-panel-introduction.md).
