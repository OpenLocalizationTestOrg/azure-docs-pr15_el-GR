# <a name="azure-ad-privileged-identity-management-how-to-add-or-remove-a-user-role"></a>Azure AD Προνομιούχους διαχείρισης των ταυτοτήτων: Πώς μπορείτε να προσθέσετε ή να καταργήσετε ένα ρόλο χρήστη

Με το Azure Active Directory (AD), μια καθολικός διαχειριστής (ή διαχειριστής εταιρείας) να ενημερώσετε τους χρήστες που έχουν **οριστικά** που έχουν εκχωρηθεί σε ρόλους σε Azure AD. Αυτό γίνεται με το cmdlet του PowerShell όπως `Add-MsolRoleMember` και `Remove-MsolRoleMember`. Ή, μπορούν να χρησιμοποιήσουν την πύλη του Azure κλασική όπως περιγράφεται στο θέμα [Εκχώρηση ρόλων διαχειριστή στο Azure Active Directory](active-directory-assign-admin-roles.md).

Η εφαρμογή Azure AD Προνομιούχους διαχείρισης των ταυτοτήτων επιτρέπει στους διαχειριστές Προνομιούχους ρόλο για να κάνετε μόνιμες εκχωρήσεις ρόλων καθώς και. Επιπλέον, οι διαχειριστές Προνομιούχους ρόλο μπορούν να χρήστες **επιλέξιμο** ρόλων διαχειριστή. Διαχειριστής δικαίωμα να ενεργοποιήσετε το ρόλο όταν χρειάζονται και, στη συνέχεια, τα δικαιώματά λήξει μόλις ολοκληρώσετε την εργασία τους.

## <a name="manage-roles-with-pim-in-the-azure-portal"></a>Διαχείριση ρόλων με PIM στην πύλη του Azure

Στην εταιρεία σας, μπορείτε να εκχωρήσετε στους χρήστες για διαφορετικούς ρόλους διαχειριστή στο Azure AD, Office 365, καθώς και άλλες υπηρεσίες της Microsoft και εφαρμογές.  Μπορείτε να βρείτε περισσότερες λεπτομέρειες σχετικά με τους ρόλους διαθέσιμη σε [ρόλους σε Azure AD PIM](active-directory-privileged-identity-management-roles.md).

Για να προσθέσετε ή να καταργήσετε ένα χρήστη σε ένα ρόλο με δικαιώματα διαχείρισης των ταυτοτήτων, εμφανίστε τον πίνακα εργαλείων PIM. Στη συνέχεια, είτε κάντε κλικ στο κουμπί **χρήστες σε ρόλους διαχειριστή** ή επιλέξτε έναν συγκεκριμένο ρόλο (όπως ο Καθολικός διαχειριστής) από τον πίνακα ρόλους.

> [AZURE.NOTE] Εάν δεν έχετε ενεργοποιήσει ακόμη PIM στην πύλη του Azure, μεταβείτε [Γρήγορα αποτελέσματα με το Azure AD PIM](active-directory-privileged-identity-management-getting-started.md) για λεπτομέρειες.

Εάν θέλετε να εκχωρήσετε σε άλλο χρήστη πρόσβαση σε PIM ίδια, τους ρόλους που PIM απαιτεί να έχει ο χρήστης περιγράφονται περαιτέρω πώς [να δώσετε πρόσβαση PIM](active-directory-privileged-identity-management-how-to-give-access-to-pim.md).

## <a name="add-a-user-to-a-role"></a>Προσθήκη χρήστη σε ένα ρόλο

1. Στην [πύλη του Azure](https://portal.azure.com/), επιλέξτε το πλακίδιο **Azure AD Προνομιούχους διαχείρισης των ταυτοτήτων** στον πίνακα εργαλείων.
2. Επιλέξτε **Διαχείριση ρόλων Προνομιούχους**.
3. Στον πίνακα **ρόλο σύνοψης** , επιλέξτε το ρόλο που θέλετε να διαχειριστείτε.
4. Στο το ρόλο blade, επιλέξτε **Προσθήκη**.
5. Κάντε κλικ στην επιλογή **Επιλέξτε χρήστες** και αναζήτησης για το χρήστη του blade **Επιλέξτε τους χρήστες** .  
6. Επιλέξτε το χρήστη από τη λίστα των αποτελεσμάτων αναζήτησης και επιλέξτε **Τέλος**.
4. Κάντε κλικ στο **κουμπί OK** για να αποθηκεύσετε την επιλογή σας. Ο χρήστης που έχετε επιλέξει θα εμφανίζεται στη λίστα ως πληρούν τις απαιτήσεις για το ρόλο.

> [AZURE.NOTE]
>Νέων χρηστών σε ένα ρόλο μόνο είναι επιλέξιμα από το ρόλο από προεπιλογή. Εάν θέλετε να κάνετε μόνιμες του ρόλου, κάντε κλικ στο χρήστη στη λίστα. Τα στοιχεία του χρήστη θα εμφανίζονται σε ένα νέο blade. Επιλέξτε **δικαιώματα** στο μενού "πληροφορίες χρήστη".  
>Εάν ένας χρήστης δεν μπορεί να καταχωρήσει για τον έλεγχο ταυτότητας πολλών παραγόντων Azure (MFA) ή χρησιμοποιώντας ένα λογαριασμό Microsoft (συνήθως @outlook.com), πρέπει να κάνετε μόνιμες σε όλους τους ρόλους. Οι διαχειριστές επιλέξιμο ζητείται να εγγραφείτε για MFA κατά την ενεργοποίηση.

Τώρα που ο χρήστης είναι επιλέξιμα από ένα ρόλο, να το ενημερώσει ότι αυτά να το ενεργοποιήσετε σύμφωνα με τις οδηγίες στο θέμα [πώς μπορείτε να ενεργοποιήσετε ή να απενεργοποιήσετε ένα ρόλο](active-directory-privileged-identity-management-how-to-activate-role.md).

## <a name="remove-a-user-from-a-role"></a>Κατάργηση χρήστη από ένα ρόλο

Μπορείτε να κατάργηση χρηστών από εκχώρησης ρόλου επιλέξιμο, αλλά βεβαιωθείτε ότι υπάρχει πάντα τουλάχιστον ένας χρήστης ένα μόνιμο καθολικός διαχειριστής.

Ακολουθήστε τα παρακάτω βήματα για να καταργήσετε έναν συγκεκριμένο χρήστη από ένα ρόλο:

1. Μεταβείτε στο ρόλο στη λίστα ρόλων, επιλέγοντας ένα ρόλο στον πίνακα εργαλείων Azure AD PIM είτε κάνοντας κλικ στο κουμπί **χρήστες σε ρόλους διαχειριστή** .
2. Κάντε κλικ στο χρήστη στη λίστα χρηστών.
3. Κάντε κλικ στην επιλογή **Κατάργηση**. Ένα μήνυμα θα σας ζητήσει να επιβεβαιώσετε.
4. Κάντε κλικ στο κουμπί **Ναι** για να καταργήσετε το ρόλο από το χρήστη.

Εάν δεν είστε βέβαιοι τους χρήστες που εξακολουθείτε να χρειάζεστε τις αναθέσεις ρόλων, στη συνέχεια, μπορείτε να [ξεκινήσετε μια αναθεώρηση πρόσβασης για το ρόλο](active-directory-privileged-identity-management-how-to-start-security-review.md).
