#<a name="tutorial-azure-active-directory-integration-with-greenhouse"></a>Πρόγραμμα εκμάθησης: Ενοποίηση καταλόγου Azure Active Directory με θερμοκηπίου
  
Είναι ο στόχος αυτού του προγράμματος εκμάθησης για να εμφανίσετε την ενοποίηση του Azure και θερμοκηπίου.  
Το σενάριο που περιγράφονται σε αυτό το πρόγραμμα εκμάθησης προϋποθέτει ότι έχετε ήδη τα ακόλουθα στοιχεία:

-   Μια έγκυρη συνδρομή Azure
-   Θερμοκηπίου καθολική σύνδεση συνδρομής
  
Μετά την ολοκλήρωση αυτού του προγράμματος εκμάθησης, θα μπορείτε να καθολικής σύνδεσης σε μια εφαρμογή στην τοποθεσία της εταιρείας σας θερμοκηπίου (υπηρεσία παροχής που ξεκινούν από σύμβολο στην) ή με την [Εισαγωγή στο πλαίσιο πρόσβασης](active-directory-saas-access-panel-introduction.md)Azure AD χρηστών που έχουν εκχωρηθεί σε θερμοκηπίου.
  
Το σενάριο που περιγράφονται σε αυτό το πρόγραμμα εκμάθησης αποτελείται από τα ακόλουθα δομικά στοιχεία:

1.  Ενεργοποίηση την ενοποίηση εφαρμογής για θερμοκηπίου
2.  Ρύθμιση παραμέτρων Καθολικής σύνδεσης
3.  Ρύθμιση παραμέτρων προμήθεια του χρήστη
4.  Αντιστοίχιση χρηστών

![Σενάριο] (./media/active-directory-saas-greenhouse-tutorial/IC790783.png "Σενάριο")
##<a name="enabling-the-application-integration-for-greenhouse"></a>Ενεργοποίηση την ενοποίηση εφαρμογής για θερμοκηπίου
  
Είναι ο στόχος αυτής της ενότητας για να εφαρμόσετε διάρθρωση πώς μπορείτε να ενεργοποιήσετε την ενοποίηση εφαρμογής για θερμοκηπίου.

###<a name="to-enable-the-application-integration-for-greenhouse-perform-the-following-steps"></a>Για να ενεργοποιήσετε την ενοποίηση εφαρμογής για θερμοκηπίου, ακολουθήστε τα παρακάτω βήματα:

1.  Στην κλασική πύλη Azure, στο αριστερό παράθυρο περιήγησης, κάντε κλικ στην επιλογή **Υπηρεσία καταλόγου Active Directory**.

    ![Υπηρεσία καταλόγου Active Directory] (./media/active-directory-saas-greenhouse-tutorial/IC700993.png "Υπηρεσία καταλόγου Active Directory")

2.  Από τη λίστα **καταλόγου** , επιλέξτε τον κατάλογο για την οποία θέλετε να ενεργοποιήσετε την ενοποίηση καταλόγου.

3.  Για να ανοίξετε τις εφαρμογές προβολή, στην προβολή του καταλόγου, κάντε κλικ στην επιλογή **εφαρμογές** στο μενού επάνω.

    ![Εφαρμογές] (./media/active-directory-saas-greenhouse-tutorial/IC700994.png "Εφαρμογές")

4.  Κάντε κλικ στην επιλογή **Προσθήκη** στο κάτω μέρος της σελίδας.

    ![Προσθήκη εφαρμογής] (./media/active-directory-saas-greenhouse-tutorial/IC749321.png "Προσθήκη εφαρμογής")

5.  Στο παράθυρο διαλόγου **Τι θέλετε να κάνετε** , κάντε κλικ στην επιλογή **Προσθήκη εφαρμογής από τη συλλογή**.

    ![Προσθήκη εφαρμογής από gallerry] (./media/active-directory-saas-greenhouse-tutorial/IC749322.png "Προσθήκη εφαρμογής από gallerry")

6.  Στο **πλαίσιο αναζήτησης**, πληκτρολογήστε **θερμοκηπίου**.

    ![Συλλογή εφαρμογών] (./media/active-directory-saas-greenhouse-tutorial/IC790784.png "Συλλογή εφαρμογών")

7.  Στο παράθυρο αποτελεσμάτων, επιλέξτε **θερμοκηπίου**και, στη συνέχεια, κάντε κλικ στην επιλογή **Ολοκλήρωση** , για να προσθέσετε την εφαρμογή.

    ![Θερμοκηπίου] (./media/active-directory-saas-greenhouse-tutorial/IC790785.png "Θερμοκηπίου")
##<a name="configuring-single-sign-on"></a>Ρύθμιση παραμέτρων Καθολικής σύνδεσης
  
Στόχος αυτής της ενότητας είναι να περιγράψει τον τρόπο για να επιτρέψετε στους χρήστες να ελέγχουν την ταυτότητα θερμοκηπίου με το λογαριασμό στο Azure AD χρησιμοποιώντας Ομοσπονδία με βάση το πρωτόκολλο SAML.

###<a name="to-configure-single-sign-on-perform-the-following-steps"></a>Για να ρυθμίσετε τις παραμέτρους της καθολικής σύνδεσης, ακολουθήστε τα παρακάτω βήματα:

1.  Στην κλασική πύλη Azure, στη σελίδα ενοποίησης **θερμοκηπίου** εφαρμογής, κάντε κλικ στην επιλογή **Ρύθμιση παραμέτρων Καθολικής σύνδεσης** για να ανοίξετε το παράθυρο διαλόγου **Ρύθμιση παραμέτρων Καθολικής σύνδεσης** .

    ![Ρύθμιση παραμέτρων Καθολικής σύνδεσης] (./media/active-directory-saas-greenhouse-tutorial/IC790786.png "Ρύθμιση παραμέτρων Καθολικής σύνδεσης")

2.  Στη σελίδα **Πώς θέλετε οι χρήστες να πραγματοποιούν είσοδο θερμοκηπίου** , επιλέξτε **Microsoft Azure AD καθολικής σύνδεσης**και, στη συνέχεια, κάντε κλικ στο κουμπί **Επόμενο**.

    ![Ρύθμιση παραμέτρων Καθολικής σύνδεσης] (./media/active-directory-saas-greenhouse-tutorial/IC790787.png "Ρύθμιση παραμέτρων Καθολικής σύνδεσης")

3.  Στη σελίδα **Ρύθμιση παραμέτρων URL εφαρμογής** , στο πλαίσιο κειμένου **Εισόδου στη διεύθυνση URL** , πληκτρολογήστε τη διεύθυνση URL σας χρησιμοποιώντας το παρακάτω μοτίβο "*https://company.greenhouse.io*" και, στη συνέχεια, κάντε κλικ στο κουμπί **Επόμενο**.

    ![Ρύθμιση παραμέτρων εφαρμογής διεύθυνσης URL] (./media/active-directory-saas-greenhouse-tutorial/IC790788.png "Ρύθμιση παραμέτρων εφαρμογής διεύθυνσης URL")

4.  Στη σελίδα **Ρύθμιση παραμέτρων Καθολικής σύνδεσης στο θερμοκηπίου** , κάντε κλικ στην επιλογή **Λήψη μετα-δεδομένων**και, στη συνέχεια, αποθηκεύστε το αρχείο μετα-δεδομένων τοπικά στον υπολογιστή σας.

    ![Ρύθμιση παραμέτρων Καθολικής σύνδεσης] (./media/active-directory-saas-greenhouse-tutorial/IC790789.png "Ρύθμιση παραμέτρων Καθολικής σύνδεσης")

5.  Προώθηση αυτό το αρχείο μετα-δεδομένων για την ομάδα υποστήριξης θερμοκηπίου.

    >[AZURE.NOTE] Καθολικής σύνδεσης πρέπει να είναι ενεργοποιημένες από την ομάδα υποστήριξης θερμοκηπίου.

6.  Στην κλασική Azure πύλη, επιλέξτε την καθολική σύνδεση παραμέτρων επιβεβαίωσης και, στη συνέχεια, κάντε κλικ στην επιλογή **Ολοκλήρωση** , για να κλείσετε το παράθυρο διαλόγου **Ρύθμιση παραμέτρων Καθολικής σύνδεσης** .

    ![Ρύθμιση παραμέτρων Καθολικής σύνδεσης] (./media/active-directory-saas-greenhouse-tutorial/IC790790.png "Ρύθμιση παραμέτρων Καθολικής σύνδεσης")
##<a name="configuring-user-provisioning"></a>Ρύθμιση παραμέτρων προμήθεια του χρήστη
  
Για να επιτρέψετε στους χρήστες Azure AD για να συνδεθείτε στο θερμοκηπίου, πρέπει να παρασχεθεί σε θερμοκηπίου.  
Στην περίπτωση θερμοκηπίου, προμήθεια είναι μια μη αυτόματη εργασία.

###<a name="to-provision-a-user-accounts-perform-the-following-steps"></a>Η παροχή μιας λογαριασμούς χρήστη, εκτελέστε τα ακόλουθα βήματα:

1.  Συνδεθείτε στην τοποθεσία της εταιρείας σας **θερμοκηπίου** ως διαχειριστής.

2.  Στο μενού στο επάνω μέρος, κάντε κλικ στην επιλογή **Ρύθμιση παραμέτρων**και, στη συνέχεια, κάντε κλικ στην επιλογή **χρήστες**.

    ![Οι χρήστες] (./media/active-directory-saas-greenhouse-tutorial/IC790791.png "Οι χρήστες")

3.  Κάντε κλικ στην επιλογή **τους νέους χρήστες**.

    ![Νέο χρήστη] (./media/active-directory-saas-greenhouse-tutorial/IC790792.png "Νέο χρήστη")

4.  Στην ενότητα **Προσθήκη νέου χρήστη** , εκτελέστε τα ακόλουθα βήματα:

    ![Προσθήκη νέου χρήστη] (./media/active-directory-saas-greenhouse-tutorial/IC790793.png "Προσθήκη νέου χρήστη")

    1.  Στο πλαίσιο κειμένου **Enter μηνύματα ηλεκτρονικού ταχυδρομείου χρήστη** , πληκτρολογήστε τη διεύθυνση ηλεκτρονικού ταχυδρομείου από έναν έγκυρο λογαριασμό Azure Active Directory που θέλετε για την παροχή των.
    2.  Κάντε κλικ στην επιλογή **Αποθήκευση**.
        
        >[AZURE.NOTE] Οι κάτοχοι λογαριασμός Azure Active Directory θα λάβουν ένα μήνυμα ηλεκτρονικού ταχυδρομείου, όπως μια σύνδεση για να επιβεβαιώσετε το λογαριασμό, προκειμένου να καταστεί ενεργή.

>[AZURE.NOTE] Μπορείτε να χρησιμοποιήσετε οποιοδήποτε άλλο θερμοκηπίου χρήστη λογαριασμού εργαλεία δημιουργίας ή APIs που παρέχεται από θερμοκηπίου σε λογαριασμούς χρηστών AAD διάταξη.

##<a name="assigning-users"></a>Αντιστοίχιση χρηστών
  
Για να ελέγξετε τη ρύθμιση των παραμέτρων σας, πρέπει να εκχωρήσετε Azure AD χρηστών που θέλετε να επιτρέψετε χρησιμοποιώντας την εφαρμογή έχουν πρόσβαση σε αυτό αναθέτοντας σε αυτά.

###<a name="to-assign-users-to-greenhouse-perform-the-following-steps"></a>Για να εκχωρήσετε στους χρήστες να θερμοκηπίου, εκτελέστε τα ακόλουθα βήματα:

1.  Στην κλασική Azure πύλη, δημιουργήστε ένα λογαριασμό δοκιμής.

2.  Στη σελίδα ενοποίησης **θερμοκηπίου **εφαρμογής, κάντε κλικ στην επιλογή **εκχωρήσετε στους χρήστες**.

    ![Αντιστοίχιση χρηστών] (./media/active-directory-saas-greenhouse-tutorial/IC790794.png "Αντιστοίχιση χρηστών")

3.  Επιλέξτε το χρήστη έλεγχος, κάντε κλικ στην επιλογή **Εκχώρηση**και, στη συνέχεια, κάντε κλικ στο κουμπί **Ναι** για να επιβεβαιώσετε την ανάθεση.

    ![Ναι] (./media/active-directory-saas-greenhouse-tutorial/IC767830.png "Ναι")
  
Εάν θέλετε να ελέγξετε τις ρυθμίσεις καθολικής εισόδου, ανοίξτε τον πίνακα της Access. Για περισσότερες λεπτομέρειες σχετικά με τον πίνακα της Access, ανατρέξτε στο θέμα [Εισαγωγή στο πλαίσιο πρόσβασης](active-directory-saas-access-panel-introduction.md).
