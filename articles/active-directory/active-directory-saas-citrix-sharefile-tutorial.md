#<a name="tutorial-azure-active-directory-integration-with-citrix-sharefile"></a>Πρόγραμμα εκμάθησης: Ενοποίηση καταλόγου Azure Active Directory με Citrix ShareFile

Είναι ο στόχος αυτού του προγράμματος εκμάθησης για να εμφανίσετε την ενοποίηση του Azure και Citrix ShareFile.  
Το σενάριο που περιγράφονται σε αυτό το πρόγραμμα εκμάθησης προϋποθέτει ότι έχετε ήδη τα ακόλουθα στοιχεία:

-   Μια έγκυρη συνδρομή Azure
-   Ένας μισθωτής Citrix ShareFile

Αφού ολοκληρώσετε αυτό το πρόγραμμα εκμάθησης, θα μπορούν να καθολικής σύνδεσης σε μια εφαρμογή στην τοποθεσία της εταιρείας σας Citrix ShareFile (υπηρεσία παροχής που ξεκινούν από σύμβολο στην) ή με την [Εισαγωγή στο πλαίσιο πρόσβασης](active-directory-saas-access-panel-introduction.md)Azure AD χρηστών που έχουν εκχωρηθεί σε Citrix ShareFile.

Το σενάριο που περιγράφονται σε αυτό το πρόγραμμα εκμάθησης αποτελείται από τα ακόλουθα δομικά στοιχεία:

1.  Ενεργοποίηση την ενοποίηση εφαρμογής για Citrix ShareFile
2.  Ρύθμιση παραμέτρων Καθολικής σύνδεσης
3.  Ρύθμιση παραμέτρων προμήθεια του χρήστη
4.  Αντιστοίχιση χρηστών

![Σενάριο] (./media/active-directory-saas-citrix-sharefile-tutorial/IC773620.png "Σενάριο")
##<a name="enabling-the-application-integration-for-citrix-sharefile"></a>Ενεργοποίηση την ενοποίηση εφαρμογής για Citrix ShareFile

Στόχος αυτής της ενότητας είναι να εφαρμόσετε διάρθρωση πώς μπορείτε να ενεργοποιήσετε την ενοποίηση εφαρμογής για Citrix ShareFile.

###<a name="to-enable-the-application-integration-for-citrix-sharefile-perform-the-following-steps"></a>Για να ενεργοποιήσετε την ενοποίηση εφαρμογής για Citrix ShareFile, ακολουθήστε τα παρακάτω βήματα:

1.  Στην κλασική πύλη Azure, στο αριστερό παράθυρο περιήγησης, κάντε κλικ στην επιλογή **Υπηρεσία καταλόγου Active Directory**.

    ![Υπηρεσία καταλόγου Active Directory] (./media/active-directory-saas-citrix-sharefile-tutorial/IC700993.png "Υπηρεσία καταλόγου Active Directory")

2.  Από τη λίστα **καταλόγου** , επιλέξτε τον κατάλογο για την οποία θέλετε να ενεργοποιήσετε την ενοποίηση καταλόγου.

3.  Για να ανοίξετε τις εφαρμογές προβολή, στην προβολή του καταλόγου, κάντε κλικ στην επιλογή **εφαρμογές** στο μενού επάνω.

    ![Εφαρμογές] (./media/active-directory-saas-citrix-sharefile-tutorial/IC700994.png "Εφαρμογές")

4.  Κάντε κλικ στην επιλογή **Προσθήκη** στο κάτω μέρος της σελίδας.

    ![Προσθήκη εφαρμογής] (./media/active-directory-saas-citrix-sharefile-tutorial/IC749321.png "Προσθήκη εφαρμογής")

5.  Στο παράθυρο διαλόγου **Τι θέλετε να κάνετε** , κάντε κλικ στην επιλογή **Προσθήκη εφαρμογής από τη συλλογή**.

    ![Προσθήκη εφαρμογής από gallerry] (./media/active-directory-saas-citrix-sharefile-tutorial/IC749322.png "Προσθήκη εφαρμογής από gallerry")

6.  Στο **πλαίσιο αναζήτησης**, πληκτρολογήστε **Citrix ShareFile**.

    ![Συλλογή εφαρμογών] (./media/active-directory-saas-citrix-sharefile-tutorial/IC773621.png "Συλλογή εφαρμογών")

7.  Στο παράθυρο αποτελεσμάτων, επιλέξτε **Citrix ShareFile**και, στη συνέχεια, κάντε κλικ στην επιλογή **Ολοκλήρωση** , για να προσθέσετε την εφαρμογή.

    ![Citrix ShareFile] (./media/active-directory-saas-citrix-sharefile-tutorial/IC773622.png "Citrix ShareFile")
##<a name="configuring-single-sign-on"></a>Ρύθμιση παραμέτρων Καθολικής σύνδεσης

Στόχος αυτής της ενότητας είναι να περιγράψει τον τρόπο για να επιτρέψετε στους χρήστες να ελέγχουν την ταυτότητα Citrix ShareFile με το λογαριασμό στο Azure AD χρησιμοποιώντας Ομοσπονδία με βάση το πρωτόκολλο SAML.

###<a name="to-configure-single-sign-on-perform-the-following-steps"></a>Για να ρυθμίσετε τις παραμέτρους της καθολικής σύνδεσης, ακολουθήστε τα παρακάτω βήματα:

1.  Στην κλασική πύλη Azure, στη σελίδα ενοποίησης **Citrix ShareFile** εφαρμογής, κάντε κλικ στην επιλογή **Ρύθμιση παραμέτρων Καθολικής σύνδεσης** για να ανοίξετε το παράθυρο διαλόγου **Ρύθμιση παραμέτρων Καθολικής σύνδεσης** .

    ![Ενεργοποίηση της καθολικής σύνδεσης] (./media/active-directory-saas-citrix-sharefile-tutorial/IC773623.png "Ενεργοποίηση της καθολικής σύνδεσης")

2.  Στη σελίδα **Πώς θέλετε οι χρήστες να πραγματοποιούν είσοδο Citrix ShareFile** , επιλέξτε **Microsoft Azure AD καθολικής σύνδεσης**και, στη συνέχεια, κάντε κλικ στο κουμπί **Επόμενο**.

    ![Ρύθμιση παραμέτρων Καθολικής σύνδεσης] (./media/active-directory-saas-citrix-sharefile-tutorial/IC773624.png "Ρύθμιση παραμέτρων Καθολικής σύνδεσης")

3.  Στη σελίδα **Ρύθμιση παραμέτρων URL εφαρμογής** , στο πλαίσιο κειμένου **Citrix ShareFile εισόδου στη διεύθυνση URL** , πληκτρολογήστε τη διεύθυνση URL σας χρησιμοποιώντας το παρακάτω μοτίβο `https://<tenant-name>.shareFile.com`, και, στη συνέχεια, κάντε κλικ στο κουμπί **Επόμενο**.

    ![Ρύθμιση παραμέτρων εφαρμογής διεύθυνσης URL] (./media/active-directory-saas-citrix-sharefile-tutorial/IC773625.png "Ρύθμιση παραμέτρων εφαρμογής διεύθυνσης URL")

4.  Στη σελίδα **Ρύθμιση παραμέτρων Καθολικής σύνδεσης στο Citrix ShareFile** , για να κάνετε λήψη του πιστοποιητικού, κάντε κλικ στην επιλογή **λήψη πιστοποιητικό**και, στη συνέχεια, αποθηκεύστε το αρχείο πιστοποιητικού στον υπολογιστή σας.

    ![ConfigureSingle καθολικής σύνδεσης] (./media/active-directory-saas-citrix-sharefile-tutorial/IC773626.png "ConfigureSingle καθολικής σύνδεσης")

5.  Σε ένα παράθυρο προγράμματος περιήγησης web διαφορετικά, συνδεθείτε στην τοποθεσία της εταιρείας σας **Citrix ShareFile** ως διαχειριστής.

6.  Στη γραμμή εργαλείων στο επάνω μέρος, κάντε κλικ στην επιλογή **Διαχείριση**.

7.  Στο αριστερό παράθυρο περιήγησης, επιλέξτε **Ρύθμιση παραμέτρων Καθολικής σύνδεσης**.

    ![Διαχείριση του λογαριασμού] (./media/active-directory-saas-citrix-sharefile-tutorial/IC773627.png "Διαχείριση του λογαριασμού")

8.  Στην το **καθολικής σύνδεσης / ρύθμισης παραμέτρων SAML 2.0** σελίδα του παραθύρου διαλόγου στην περιοχή **Βασικές ρυθμίσεις**, ακολουθήστε τα παρακάτω βήματα:

    ![Καθολικής σύνδεσης] (./media/active-directory-saas-citrix-sharefile-tutorial/IC773628.png "Καθολικής σύνδεσης")

    1.  Κάντε κλικ στην επιλογή **Ενεργοποίηση SAML**.
    2.  Στην κλασική πύλη Azure, στη σελίδα παράθυρο διαλόγου **Ρύθμιση παραμέτρων Καθολικής σύνδεσης στο Citrix ShareFile** , αντιγράψτε την τιμή **Αναγνωριστικό οντότητας** και, στη συνέχεια, το επικολλήσετε στο το **σας εκδότη IDP / Αναγνωριστικό οντότητας** πλαίσιο κειμένου.
    3.  Στην κλασική πύλη Azure, στη σελίδα παράθυρο διαλόγου **Ρύθμιση παραμέτρων Καθολικής σύνδεσης στο Citrix ShareFile** , αντιγράψτε την τιμή **Διεύθυνση URL απομακρυσμένης σύνδεσης** και, στη συνέχεια, επικολλήστε τη **Διεύθυνση URL σύνδεσης** πλαισίου κειμένου.
    4.  Στην κλασική πύλη Azure, σελίδα με **τη ρύθμιση παραμέτρων Καθολικής σύνδεσης στο Citrix ShareFile** παραθύρου διαλόγου, αντιγράψτε την τιμή **Διεύθυνση URL απομακρυσμένης αποσυνδεθείτε** και κατόπιν επικολλήστε το στο πλαίσιο κειμένου **Διεύθυνση URL αποσυνδεθείτε** .
    5.  Κάντε κλικ στην επιλογή **Αλλαγή** δίπλα στο πεδίο **Πιστοποιητικό X.509** και, στη συνέχεια, αποστείλετε το πιστοποιητικό που λάβατε από την πύλη κλασική Azure AD.
        ![Βασικές ρυθμίσεις] (./media/active-directory-saas-citrix-sharefile-tutorial/IC773629.png "Βασικές ρυθμίσεις")

9.  Κάντε κλικ στην επιλογή **Αποθήκευση** στην πύλη διαχείρισης του Citrix ShareFile.

10. Στην κλασική Azure πύλη, επιλέξτε την καθολική σύνδεση παραμέτρων επιβεβαίωσης και, στη συνέχεια, κάντε κλικ στην επιλογή **Ολοκλήρωση** , για να κλείσετε το παράθυρο διαλόγου **Ρύθμιση παραμέτρων Καθολικής σύνδεσης** .

    ![Ρύθμιση παραμέτρων Καθολικής σύνδεσης] (./media/active-directory-saas-citrix-sharefile-tutorial/IC773630.png "Ρύθμιση παραμέτρων Καθολικής σύνδεσης")
##<a name="configuring-user-provisioning"></a>Ρύθμιση παραμέτρων προμήθεια του χρήστη

Για να επιτρέψετε στους χρήστες Azure AD για να συνδεθείτε στο Citrix ShareFile, πρέπει να παρασχεθεί στο Citrix ShareFile.  
Στην περίπτωση Citrix ShareFile, προμήθεια είναι μια μη αυτόματη εργασία.

###<a name="to-provision-a-user-accounts-perform-the-following-steps"></a>Η παροχή μιας λογαριασμούς χρήστη, εκτελέστε τα ακόλουθα βήματα:

1.  Συνδεθείτε στο μισθωτή σας **Citrix ShareFile** .

2.  Κάντε κλικ στην επιλογή **Διαχείριση χρηστών \> Διαχείριση Home χρήστες \> + δημιουργία υπαλλήλου**.

    ![Δημιουργία υπαλλήλου] (./media/active-directory-saas-citrix-sharefile-tutorial/IC781050.png "Δημιουργία υπαλλήλου")

3.  Πληκτρολογήστε το **μήνυμα ηλεκτρονικού ταχυδρομείου**, **όνομα** και **Επώνυμο** με μια έγκυρη Azure AD λογαριασμού που θέλετε για την παροχή των.

    ![Βασικές πληροφορίες] (./media/active-directory-saas-citrix-sharefile-tutorial/IC799951.png "Βασικές πληροφορίες")

4.  Κάντε κλικ στην επιλογή **Προσθήκη χρήστη**.

    >[AZURE.NOTE] Ο κάτοχος λογαριασμού AAD θα λάβετε ένα μήνυμα ηλεκτρονικού ταχυδρομείου και να ακολουθήσετε μια σύνδεση για να επιβεβαιώσετε το λογαριασμό, προκειμένου να καταστεί ενεργή.

>[AZURE.NOTE] Μπορείτε να χρησιμοποιήσετε οποιοδήποτε άλλο Citrix ShareFile χρήστη λογαριασμού εργαλεία δημιουργίας ή APIs που παρέχεται από Citrix ShareFile σε λογαριασμούς χρηστών AAD διάταξη.

##<a name="assigning-users"></a>Αντιστοίχιση χρηστών

Για να ελέγξετε τη ρύθμιση των παραμέτρων σας, πρέπει να εκχωρήσετε Azure AD χρηστών που θέλετε να επιτρέψετε χρησιμοποιώντας την εφαρμογή έχουν πρόσβαση σε αυτό αναθέτοντας σε αυτά.

###<a name="to-assign-users-to-citrix-sharefile-perform-the-following-steps"></a>Για να εκχωρήσετε στους χρήστες να Citrix ShareFile, εκτελέστε τα ακόλουθα βήματα:

1.  Στην κλασική Azure πύλη, δημιουργήστε ένα λογαριασμό δοκιμής.

2.  Στη σελίδα ενοποίησης **Citrix ShareFile **εφαρμογής, κάντε κλικ στην επιλογή **εκχωρήσετε στους χρήστες**.

    ![Αντιστοίχιση χρηστών] (./media/active-directory-saas-citrix-sharefile-tutorial/IC773631.png "Αντιστοίχιση χρηστών")

3.  Επιλέξτε το χρήστη έλεγχος, κάντε κλικ στην επιλογή **Εκχώρηση**και, στη συνέχεια, κάντε κλικ στο κουμπί **Ναι** για να επιβεβαιώσετε την ανάθεση.

    ![Ναι] (./media/active-directory-saas-citrix-sharefile-tutorial/IC767830.png "Ναι")

Εάν θέλετε να ελέγξετε τις ρυθμίσεις καθολικής εισόδου, ανοίξτε τον πίνακα της Access. Για περισσότερες λεπτομέρειες σχετικά με τον πίνακα της Access, ανατρέξτε στο θέμα [Εισαγωγή στο πλαίσιο πρόσβασης](active-directory-saas-access-panel-introduction.md).
