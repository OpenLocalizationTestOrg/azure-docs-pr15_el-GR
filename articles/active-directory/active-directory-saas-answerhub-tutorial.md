#<a name="tutorial-azure-active-directory-integration-with-answerhub"></a>Πρόγραμμα εκμάθησης: Ενοποίηση καταλόγου Azure Active Directory με AnswerHub

Είναι ο στόχος αυτού του προγράμματος εκμάθησης για να εμφανίσετε την ενοποίηση του Azure και [AnswerHub](http://www.dzonesoftware.com/products/answerhub-question-answer-software).  
Το σενάριο που περιγράφονται σε αυτό το πρόγραμμα εκμάθησης προϋποθέτει ότι έχετε ήδη τα ακόλουθα στοιχεία:

-   Μια έγκυρη συνδρομή Azure
-   Μια [AnswerHub](http://www.dzonesoftware.com/products/answerhub-question-answer-software) καθολικής σύνδεσης με δυνατότητα συνδρομή

Μετά την ολοκλήρωση αυτού του προγράμματος εκμάθησης, θα μπορείτε να καθολικής σύνδεσης σε μια εφαρμογή στην τοποθεσία της εταιρείας σας AnswerHub (υπηρεσία παροχής που ξεκινούν από σύμβολο στην) ή με την [Εισαγωγή στο πλαίσιο πρόσβασης](active-directory-saas-access-panel-introduction.md)Azure AD χρηστών που έχουν εκχωρηθεί σε AnswerHub.

Το σενάριο που περιγράφονται σε αυτό το πρόγραμμα εκμάθησης αποτελείται από τα ακόλουθα δομικά στοιχεία:

1.  Ενεργοποίηση την ενοποίηση εφαρμογής για AnswerHub
2.  Ρύθμιση παραμέτρων Καθολικής σύνδεσης
3.  Ρύθμιση παραμέτρων προμήθεια του χρήστη
4.  Αντιστοίχιση χρηστών

![Σενάριο] (./media/active-directory-saas-answerhub-tutorial/IC785165.png "Σενάριο")

##<a name="enabling-the-application-integration-for-answerhub"></a>Ενεργοποίηση την ενοποίηση εφαρμογής για AnswerHub

Είναι ο στόχος αυτής της ενότητας για να εφαρμόσετε διάρθρωση πώς μπορείτε να ενεργοποιήσετε την ενοποίηση εφαρμογής για το AnswerHub.

###<a name="to-enable-the-application-integration-for-answerhub-perform-the-following-steps"></a>Για να ενεργοποιήσετε την ενοποίηση εφαρμογής για AnswerHub, ακολουθήστε τα παρακάτω βήματα:

1.  Στην κλασική πύλη Azure, στο αριστερό παράθυρο περιήγησης, κάντε κλικ στην επιλογή **Υπηρεσία καταλόγου Active Directory**.

    ![Υπηρεσία καταλόγου Active Directory] (./media/active-directory-saas-answerhub-tutorial/IC700993.png "Υπηρεσία καταλόγου Active Directory")

2.  Από τη λίστα **καταλόγου** , επιλέξτε τον κατάλογο για την οποία θέλετε να ενεργοποιήσετε την ενοποίηση καταλόγου.

3.  Για να ανοίξετε τις εφαρμογές προβολή, στην προβολή του καταλόγου, κάντε κλικ στην επιλογή **εφαρμογές** στο μενού επάνω.

    ![Εφαρμογές] (./media/active-directory-saas-answerhub-tutorial/IC700994.png "Εφαρμογές")

4.  Κάντε κλικ στην επιλογή **Προσθήκη** στο κάτω μέρος της σελίδας.

    ![Προσθήκη εφαρμογής] (./media/active-directory-saas-answerhub-tutorial/IC749321.png "Προσθήκη εφαρμογής")

5.  Στο παράθυρο διαλόγου **Τι θέλετε να κάνετε** , κάντε κλικ στην επιλογή **Προσθήκη εφαρμογής από τη συλλογή**.

    ![Προσθήκη εφαρμογής από gallerry] (./media/active-directory-saas-answerhub-tutorial/IC749322.png "Προσθήκη εφαρμογής από gallerry")

6.  Στο **πλαίσιο αναζήτησης**, πληκτρολογήστε **AnswerHub**.

    ![Συλλογή εφαρμογών] (./media/active-directory-saas-answerhub-tutorial/IC785166.png "Συλλογή εφαρμογών")

7.  Στο παράθυρο αποτελεσμάτων, επιλέξτε **AnswerHub**και, στη συνέχεια, κάντε κλικ στην επιλογή **Ολοκλήρωση** , για να προσθέσετε την εφαρμογή.

    ![AnswerHub] (./media/active-directory-saas-answerhub-tutorial/IC785167.png "AnswerHub")

##<a name="configuring-single-sign-on"></a>Ρύθμιση παραμέτρων Καθολικής σύνδεσης

Στόχος αυτής της ενότητας είναι να περιγράψει τον τρόπο για να επιτρέψετε στους χρήστες να ελέγχουν την ταυτότητα AnswerHub με το λογαριασμό στο Azure AD χρησιμοποιώντας Ομοσπονδία με βάση το πρωτόκολλο SAML.  
Ως μέρος αυτής της διαδικασίας, που απαιτούνται για να δημιουργήσετε ένα αρχείο βάσης 64 κωδικοποιημένο πιστοποιητικό.  
Εάν δεν είστε εξοικειωμένοι με αυτήν τη διαδικασία, δείτε [πώς μπορείτε να μετατρέψετε ένα πιστοποιητικό δυαδικών δεδομένων σε ένα αρχείο κειμένου](http://youtu.be/PlgrzUZ-Y1o)

###<a name="to-configure-single-sign-on-perform-the-following-steps"></a>Για να ρυθμίσετε τις παραμέτρους της καθολικής σύνδεσης, ακολουθήστε τα παρακάτω βήματα:

1.  Στην κλασική πύλη Azure, στη σελίδα ενοποίησης εφαρμογής **AnswerHub** , κάντε κλικ στην επιλογή **Ρύθμιση παραμέτρων Καθολικής σύνδεσης** για να ανοίξετε το παράθυρο διαλόγου **Ρύθμιση παραμέτρων Καθολικής σύνδεσης** .

    ![Ρύθμιση παραμέτρων Καθολικής σύνδεσης] (./media/active-directory-saas-answerhub-tutorial/IC785168.png "Ρύθμιση παραμέτρων Καθολικής σύνδεσης")

2.  Στη σελίδα **Πώς θέλετε οι χρήστες να πραγματοποιούν είσοδο AnswerHub** , επιλέξτε **Microsoft Azure AD καθολικής σύνδεσης**και, στη συνέχεια, κάντε κλικ στο κουμπί **Επόμενο**.

    ![Ρύθμιση παραμέτρων Καθολικής σύνδεσης] (./media/active-directory-saas-answerhub-tutorial/IC785169.png "Ρύθμιση παραμέτρων Καθολικής σύνδεσης")

3.  Στη σελίδα **Ρύθμιση παραμέτρων URL εφαρμογής** , στο πλαίσιο κειμένου **AnswerHub εισόδου στη διεύθυνση URL** , πληκτρολογήστε τη διεύθυνση URL σας χρησιμοποιώντας το παρακάτω μοτίβο "*https://company.answerhub.com*" και, στη συνέχεια, κάντε κλικ στο κουμπί **Επόμενο**.

    ![Ρύθμιση παραμέτρων εφαρμογής διεύθυνσης URL] (./media/active-directory-saas-answerhub-tutorial/IC785170.png "Ρύθμιση παραμέτρων εφαρμογής διεύθυνσης URL")

4.  Στη σελίδα **Ρύθμιση παραμέτρων Καθολικής σύνδεσης στο AnswerHub** , για να κάνετε λήψη του πιστοποιητικού, κάντε κλικ στην επιλογή **λήψη πιστοποιητικό**και, στη συνέχεια, αποθηκεύστε το αρχείο πιστοποιητικού τοπικά στον υπολογιστή σας.

    ![Ρύθμιση παραμέτρων Καθολικής σύνδεσης] (./media/active-directory-saas-answerhub-tutorial/IC785171.png "Ρύθμιση παραμέτρων Καθολικής σύνδεσης")

5.  Σε ένα παράθυρο προγράμματος περιήγησης web διαφορετικά, συνδεθείτε στην τοποθεσία της εταιρείας σας AnswerHub ως διαχειριστής.
    >[AZURE.NOTE] Εάν χρειάζεστε βοήθεια για τη ρύθμιση των παραμέτρων AnswerHub, επικοινωνήστε με [την ομάδα υποστήριξης της AnswerHub](mailto:success@answerhub.com. ).








6.  Μετάβαση στη **Διαχείριση**.

7.  Κάντε κλικ στην καρτέλα **χρηστών και ομάδων** .

8.  Στο παράθυρο περιήγησης στην αριστερή πλευρά, στην ενότητα **Κοινωνικών ρυθμίσεις** , κάντε κλικ στην επιλογή **Διαμόρφωση SAML**.

9.  Κάντε κλικ στην καρτέλα **IDP Config** .

10. Στην καρτέλα **IDP ρύθμισης παραμέτρων** , εκτελέστε τα ακόλουθα βήματα:

    ![Ρύθμιση SAML] (./media/active-directory-saas-answerhub-tutorial/IC785172.png "Ρύθμιση SAML")

    1.  Στην κλασική πύλη Azure, στη σελίδα παράθυρο διαλόγου **Ρύθμιση παραμέτρων Καθολικής σύνδεσης στο AnswerHub** , αντιγράψτε την τιμή **Διεύθυνση URL απομακρυσμένης σύνδεσης** και κατόπιν επικολλήστε το στο πλαίσιο κειμένου **Διεύθυνση URL σύνδεσης IDP** .
    2.  Στην κλασική πύλη Azure, στη σελίδα παράθυρο διαλόγου **Ρύθμιση παραμέτρων Καθολικής σύνδεσης στο AnswerHub** , αντιγράψτε την τιμή **Διεύθυνση URL απομακρυσμένης αποσύνδεσης** και στη συνέχεια επικολλήστε το στο πλαίσιο κειμένου **Διεύθυνση URL IDP αποσύνδεσης** .
    3.  Στην κλασική πύλη Azure, στη σελίδα παράθυρο διαλόγου **Ρύθμιση παραμέτρων Καθολικής σύνδεσης στο AnswerHub** , αντιγράψτε την τιμή **Όνομα αναγνωριστικό μορφή** και κατόπιν επικολλήστε το στο πλαίσιο κειμένου **IDP όνομα αναγνωριστικό μορφή** .
    4.  Κάντε κλικ στην επιλογή **κλειδιά και πιστοποιητικά**.

11. Στην καρτέλα πλήκτρα και τα πιστοποιητικά, ακολουθήστε τα παρακάτω βήματα:

    ![Κλειδιά και πιστοποιητικά] (./media/active-directory-saas-answerhub-tutorial/IC785173.png "Κλειδιά και πιστοποιητικά")

    1.  Δημιουργία αρχείου **βάσης 64 κωδικοποιημένο** από το πιστοποιητικό που έχετε λάβει.  

        >[AZURE.TIP] Για περισσότερες λεπτομέρειες, ανατρέξτε στο θέμα [πώς μπορείτε να μετατρέψετε ένα πιστοποιητικό δυαδικών δεδομένων σε ένα αρχείο κειμένου](http://youtu.be/PlgrzUZ-Y1o)

    2.  Ανοίξτε το βάσης 64 κωδικοποιημένο πιστοποιητικό στο Σημειωματάριο, αντιγράψτε το περιεχόμενο της στο Πρόχειρο και, στη συνέχεια επικολλήστε το στο πλαίσιο κειμένου **IDP δημόσιο κλειδί (x 509 μορφή)** .
    3.  Κάντε κλικ στην επιλογή **Αποθήκευση**.

12. Στην καρτέλα **Ρύθμισης παραμέτρων IDP** , κάντε κλικ στην επιλογή **Αποθήκευση**.

13. Στην κλασική Azure πύλη, επιλέξτε την καθολική σύνδεση παραμέτρων επιβεβαίωσης και, στη συνέχεια, κάντε κλικ στην επιλογή **Ολοκλήρωση** , για να κλείσετε το παράθυρο διαλόγου **Ρύθμιση παραμέτρων Καθολικής σύνδεσης** .

    ![Ρύθμιση παραμέτρων Καθολικής σύνδεσης] (./media/active-directory-saas-answerhub-tutorial/IC785174.png "Ρύθμιση παραμέτρων Καθολικής σύνδεσης")

##<a name="configuring-user-provisioning"></a>Ρύθμιση παραμέτρων προμήθεια του χρήστη

Για να επιτρέψετε στους χρήστες Azure AD για να συνδεθείτε στο AnswerHub, πρέπει να παρασχεθεί στο AnswerHub.  
Στην περίπτωση AnswerHub, προμήθεια είναι μια μη αυτόματη εργασία.

###<a name="to-configure-user-provisioning-perform-the-following-steps"></a>Για να ρυθμίσετε τις παραμέτρους προμήθεια του χρήστη, εκτελέστε τα ακόλουθα βήματα:

1.  Συνδεθείτε στην τοποθεσία της εταιρείας σας **AnswerHub** ως διαχειριστής.

2.  Μετάβαση στη **Διαχείριση**.

3.  Κάντε κλικ στην καρτέλα **χρήστες και ομάδες** .

4.  Στο παράθυρο περιήγησης στην αριστερή πλευρά, στην ενότητα **Διαχείριση χρηστών** , κάντε κλικ στην επιλογή **Δημιουργία ή εισαγωγή τους χρήστες**.

    ![Χρήστες και ομάδες] (./media/active-directory-saas-answerhub-tutorial/IC785175.png "Χρήστες και ομάδες")

5.  Πληκτρολογήστε τη **διεύθυνση ηλεκτρονικού ταχυδρομείου**, **όνομα χρήστη** και **τον κωδικό πρόσβασης** από έναν έγκυρο λογαριασμό Azure Active Directory που θέλετε για την παροχή των σε τα σχετικά πλαίσια κειμένου και, στη συνέχεια, κάντε κλικ στην επιλογή **Αποθήκευση**.

>[AZURE.NOTE] Μπορείτε να χρησιμοποιήσετε οποιοδήποτε άλλο AnswerHub χρήστη λογαριασμού εργαλεία δημιουργίας ή APIs που παρέχεται από AnswerHub σε λογαριασμούς χρηστών AAD διάταξη.

##<a name="assigning-users"></a>Αντιστοίχιση χρηστών

Για να ελέγξετε τη ρύθμιση των παραμέτρων σας, πρέπει να εκχωρήσετε Azure AD χρηστών που θέλετε να επιτρέψετε χρησιμοποιώντας την εφαρμογή έχουν πρόσβαση σε αυτό αναθέτοντας σε αυτά.

###<a name="to-assign-users-to-answerhub-perform-the-following-steps"></a>Για να εκχωρήσετε στους χρήστες να AnswerHub, εκτελέστε τα ακόλουθα βήματα:

1.  Στην κλασική Azure πύλη, δημιουργήστε ένα λογαριασμό δοκιμής.

2.  Στη σελίδα **AnswerHub **ενοποίησης της εφαρμογής, κάντε κλικ στην επιλογή **εκχωρήσετε στους χρήστες**.

    ![Αντιστοίχιση χρηστών] (./media/active-directory-saas-answerhub-tutorial/IC785176.png "Αντιστοίχιση χρηστών")

3.  Επιλέξτε το χρήστη έλεγχος, κάντε κλικ στην επιλογή **Εκχώρηση**και, στη συνέχεια, κάντε κλικ στο κουμπί **Ναι** για να επιβεβαιώσετε την ανάθεση.

    ![Ναι] (./media/active-directory-saas-answerhub-tutorial/IC767830.png "Ναι")

Εάν θέλετε να ελέγξετε τις ρυθμίσεις καθολικής εισόδου, ανοίξτε τον πίνακα της Access. Για περισσότερες λεπτομέρειες σχετικά με τον πίνακα της Access, ανατρέξτε στο θέμα [Εισαγωγή στο πλαίσιο πρόσβασης](active-directory-saas-access-panel-introduction.md).
