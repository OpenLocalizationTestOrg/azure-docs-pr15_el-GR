#<a name="tutorial-azure-active-directory-integration-with-dropbox-for-business"></a>Πρόγραμμα εκμάθησης: Ενοποίηση καταλόγου Azure Active Directory με το Dropbox για επιχειρήσεις
  
Είναι ο στόχος αυτού του προγράμματος εκμάθησης για να εμφανίσετε την ενοποίηση του Azure και Dropbox για επιχειρήσεις.  
Το σενάριο που περιγράφονται σε αυτό το πρόγραμμα εκμάθησης προϋποθέτει ότι έχετε ήδη τα ακόλουθα στοιχεία:

-   Μια έγκυρη συνδρομή Azure
-   Δοκιμή μισθωτή στο Dropbox για επιχειρήσεις
  
Αφού ολοκληρώσετε αυτό το πρόγραμμα εκμάθησης, οι χρήστες Azure AD που έχουν εκχωρηθεί σε Dropbox για επιχειρήσεις θα μπορούν να καθολικής σύνδεσης στην εφαρμογή στο το Dropbox για επιχειρήσεις εταιρείας τοποθεσία (υπηρεσία παροχής που ξεκινούν από σύμβολο στην) ή με την [Εισαγωγή στον πίνακα της Access](active-directory-saas-access-panel-introduction.md).
  
Το σενάριο που περιγράφονται σε αυτό το πρόγραμμα εκμάθησης αποτελείται από τα εξής δομικά στοιχεία:

1.  Ενεργοποίηση την ενοποίηση εφαρμογής για το Dropbox για επιχειρήσεις
2.  Ρύθμιση παραμέτρων Καθολικής σύνδεσης
3.  Ρύθμιση παραμέτρων προμήθεια του χρήστη
4.  Αντιστοίχιση χρηστών

![Σενάριο] (./media/active-directory-saas-dropboxforbusiness-tutorial/IC769508.png "Σενάριο")



##<a name="enabling-the-application-integration-for-dropbox-for-business"></a>Ενεργοποίηση την ενοποίηση εφαρμογής για το Dropbox για επιχειρήσεις
  
Είναι ο στόχος αυτής της ενότητας για να εφαρμόσετε διάρθρωση πώς μπορείτε να ενεργοποιήσετε την ενοποίηση εφαρμογής για το Dropbox για επιχειρήσεις.

###<a name="to-enable-the-application-integration-for-dropbox-for-business-perform-the-following-steps"></a>Για να ενεργοποιήσετε την ενοποίηση εφαρμογής για το Dropbox για επιχειρήσεις, ακολουθήστε τα παρακάτω βήματα:

1.  Στην κλασική πύλη Azure, στο αριστερό παράθυρο περιήγησης, κάντε κλικ στην επιλογή **Υπηρεσία καταλόγου Active Directory**.

    ![Υπηρεσία καταλόγου Active Directory] (./media/active-directory-saas-dropboxforbusiness-tutorial/IC700993.png "Υπηρεσία καταλόγου Active Directory")

2.  Από τη λίστα **καταλόγου** , επιλέξτε τον κατάλογο για την οποία θέλετε να ενεργοποιήσετε την ενοποίηση καταλόγου.

3.  Για να ανοίξετε τις εφαρμογές προβολή, στην προβολή του καταλόγου, κάντε κλικ στην επιλογή **εφαρμογές** στο μενού επάνω.

    ![Εφαρμογές] (./media/active-directory-saas-dropboxforbusiness-tutorial/IC700994.png "Εφαρμογές")

4.  Κάντε κλικ στην επιλογή **Προσθήκη** στο κάτω μέρος της σελίδας.

    ![Προσθήκη εφαρμογής] (./media/active-directory-saas-dropboxforbusiness-tutorial/IC749321.png "Προσθήκη εφαρμογής")

5.  Στο παράθυρο διαλόγου **Τι θέλετε να κάνετε** , κάντε κλικ στην επιλογή **Προσθήκη εφαρμογής από τη συλλογή**.

    ![Προσθήκη εφαρμογής από gallerry] (./media/active-directory-saas-dropboxforbusiness-tutorial/IC749322.png "Προσθήκη εφαρμογής από gallerry")

6.  Στο **πλαίσιο αναζήτησης**, πληκτρολογήστε **Dropbox για επιχειρήσεις**.

    ![Συλλογή εφαρμογών] (./media/active-directory-saas-dropboxforbusiness-tutorial/IC701010.png "Συλλογή εφαρμογών")

7.  Στο παράθυρο αποτελεσμάτων, επιλέξτε **Dropbox για επιχειρήσεις**και, στη συνέχεια, κάντε κλικ στην επιλογή **Ολοκλήρωση** , για να προσθέσετε την εφαρμογή.

    ![Dropbox για επιχειρήσεις] (./media/active-directory-saas-dropboxforbusiness-tutorial/IC701011.png "Dropbox για επιχειρήσεις")

##<a name="configuring-single-sign-on"></a>Ρύθμιση παραμέτρων Καθολικής σύνδεσης
  
Στόχος αυτής της ενότητας είναι να περιγράψει τον τρόπο για να επιτρέψετε στους χρήστες να ελέγχουν την ταυτότητα Dropbox για επιχειρήσεις με το λογαριασμό στο Azure AD χρησιμοποιώντας Ομοσπονδία με βάση το πρωτόκολλο SAML.

Ως μέρος αυτής της διαδικασίας, που απαιτούνται για την αποστολή ενός πιστοποιητικού βάσης 64 κωδικοποιημένο το Dropbox για επιχειρήσεις μισθωτή. Εάν δεν είστε εξοικειωμένοι με αυτήν τη διαδικασία, δείτε [πώς μπορείτε να μετατρέψετε ένα πιστοποιητικό δυαδικών δεδομένων σε ένα αρχείο κειμένου](http://youtu.be/PlgrzUZ-Y1o).

###<a name="to-configure-single-sign-on-perform-the-following-steps"></a>Για να ρυθμίσετε τις παραμέτρους της καθολικής σύνδεσης, ακολουθήστε τα παρακάτω βήματα:

1.  Στην κλασική πύλη Azure, στη σελίδα ενοποίησης εφαρμογή **Dropbox για επιχειρήσεις** , κάντε κλικ στην επιλογή **Ρύθμιση παραμέτρων Καθολικής σύνδεσης** για να ανοίξετε το παράθυρο διαλόγου **Ρύθμιση παραμέτρων Καθολικής σύνδεσης** .

    ![Ρύθμιση παραμέτρων Καθολικής σύνδεσης] (./media/active-directory-saas-dropboxforbusiness-tutorial/IC749323.png "Ρύθμιση παραμέτρων Καθολικής σύνδεσης")

2.  Στη σελίδα **Πώς θέλετε οι χρήστες να πραγματοποιούν είσοδο Dropbox για επιχειρήσεις** , επιλέξτε **Microsoft Azure AD καθολικής σύνδεσης**και, στη συνέχεια, κάντε κλικ στο κουμπί **Επόμενο**.

    ![Ρύθμιση παραμέτρων Καθολικής σύνδεσης] (./media/active-directory-saas-dropboxforbusiness-tutorial/IC749327.png "Ρύθμιση παραμέτρων Καθολικής σύνδεσης")

3.  Στη σελίδα **Ρύθμιση παραμέτρων URL εφαρμογής** , εκτελέστε τα ακόλουθα βήματα:

    μια. Σύνδεση στην το Dropbox για επιχειρήσεις μισθωτή. 

    ![Ρύθμιση παραμέτρων Καθολικής σύνδεσης] (./media/active-directory-saas-dropboxforbusiness-tutorial/IC769509.png "Ρύθμιση παραμέτρων Καθολικής σύνδεσης")

    β. Στο παράθυρο περιήγησης στην αριστερή πλευρά, κάντε κλικ στην επιλογή **Κονσόλα διαχείρισης**. 

    ![Ρύθμιση παραμέτρων Καθολικής σύνδεσης] (./media/active-directory-saas-dropboxforbusiness-tutorial/IC769510.png "Ρύθμιση παραμέτρων Καθολικής σύνδεσης")

    c. Στην **Κονσόλα διαχείρισης**, κάντε κλικ στην επιλογή **Έλεγχος ταυτότητας** στο αριστερό παράθυρο περιήγησης. 

    ![Ρύθμιση παραμέτρων Καθολικής σύνδεσης] (./media/active-directory-saas-dropboxforbusiness-tutorial/IC769511.png "Ρύθμιση παραμέτρων Καθολικής σύνδεσης")

    d. Στην ενότητα **καθολικής σύνδεσης** , επιλέξτε **Ενεργοποίηση της καθολικής σύνδεσης**και, στη συνέχεια, κάντε κλικ στην επιλογή **περισσότερα** για να αναπτύξετε αυτήν την ενότητα.  

    ![Ρύθμιση παραμέτρων Καθολικής σύνδεσης] (./media/active-directory-saas-dropboxforbusiness-tutorial/IC769512.png "Ρύθμιση παραμέτρων Καθολικής σύνδεσης")

    ε. Αντιγράψτε τη διεύθυνση URL δίπλα στην επιλογή **οι χρήστες μπορούν να συνδέονται, πληκτρολογώντας τη διεύθυνση ηλεκτρονικού ταχυδρομείου ή να μεταβαίνουν απευθείας στο**. 

    ![Ρύθμιση παραμέτρων Καθολικής σύνδεσης] (./media/active-directory-saas-dropboxforbusiness-tutorial/IC769513.png "Ρύθμιση παραμέτρων Καθολικής σύνδεσης")

    f. Στην κλασική Azure πύλη, στο πλαίσιο κειμένου **DropBox για επιχειρήσεις, πραγματοποιήστε είσοδο** διεύθυνση URL, επικολλήστε τη διεύθυνση URL. 

    ![Ρύθμιση παραμέτρων Καθολικής σύνδεσης] (./media/active-directory-saas-dropboxforbusiness-tutorial/IC769514.png "Ρύθμιση παραμέτρων Καθολικής σύνδεσης")  



4. Στη σελίδα **Ρύθμιση παραμέτρων Καθολικής σύνδεσης στο Dropbox για επιχειρήσεις** , κάντε κλικ στην επιλογή **λήψη πιστοποιητικό**και, στη συνέχεια, αποθηκεύστε το αρχείο πιστοποιητικού στον υπολογιστή σας.  

    ![Ρύθμιση παραμέτρων Καθολικής σύνδεσης] (./media/active-directory-saas-dropboxforbusiness-tutorial/IC769515.png "Ρύθμιση παραμέτρων Καθολικής σύνδεσης")


5. Στη το Dropbox για επιχειρήσεις μισθωτή, στην ενότητα **καθολικής σύνδεσης** της σελίδας **ελέγχου ταυτότητας** , εκτελέστε τα ακόλουθα βήματα: 

    ![Ρύθμιση παραμέτρων Καθολικής σύνδεσης] (./media/active-directory-saas-dropboxforbusiness-tutorial/IC769516.png "Ρύθμιση παραμέτρων Καθολικής σύνδεσης")

    μια. Κάντε κλικ στην επιλογή **απαιτείται**.

    β. Στην κλασική πύλη Azure, στη σελίδα παράθυρο διαλόγου **Ρύθμιση παραμέτρων Καθολικής σύνδεσης στο Dropbox για επιχειρήσεις** , αντιγράψτε την τιμή **εισόδου διεύθυνση URL της σελίδας** και, στη συνέχεια επικολλήστε το στο πλαίσιο κειμένου **Πραγματοποιήστε είσοδο στη διεύθυνση URL** .


    c. Δημιουργία αρχείου **βάσης 64 κωδικοποιημένο** από το πιστοποιητικό που έχετε λάβει. 

    > [AZURE.TIP] Για περισσότερες λεπτομέρειες, ανατρέξτε στο θέμα [πώς μπορείτε να μετατρέψετε ένα πιστοποιητικό δυαδικών δεδομένων σε ένα αρχείο κειμένου](http://youtu.be/PlgrzUZ-Y1o).


    d. Κάντε κλικ στο κουμπί **"Επιλογή πιστοποιητικού"** και, στη συνέχεια, αναζητήστε το **βάσης 64 κωδικοποιημένο αρχείο πιστοποιητικού**.


    ε. Κάντε κλικ στο κουμπί **"Αποθήκευση αλλαγών"** για να ολοκληρώσετε τη ρύθμιση παραμέτρων σε το DropBox για επιχειρήσεις μισθωτή.


6. Στην κλασική Azure πύλη, επιλέξτε την καθολική σύνδεση ρύθμιση των παραμέτρων επιβεβαίωσης και, στη συνέχεια, κάντε κλικ στην επιλογή **Ολοκλήρωση** , για να κλείσετε το παράθυρο διαλόγου **Ρύθμιση παραμέτρων Καθολικής σύνδεσης** . 

    ![Ρύθμιση παραμέτρων Καθολικής σύνδεσης] (./media/active-directory-saas-dropboxforbusiness-tutorial/IC749329.png "Ρύθμιση παραμέτρων Καθολικής σύνδεσης")



##<a name="configuring-user-provisioning"></a>Ρύθμιση παραμέτρων προμήθεια του χρήστη
  
Στόχος αυτής της ενότητας είναι να περιγράψει τον τρόπο για να ενεργοποιήσετε την προμήθεια του χρήστη της υπηρεσίας καταλόγου Active Directory λογαριασμών χρηστών στο Dropbox για επιχειρήσεις.


### <a name="to-configure-user-provisioning-perform-the-following-steps"></a>Για να ρυθμίσετε την προμήθεια του χρήστη, εκτελέστε τα ακόλουθα βήματα:

1. Στην κλασική πύλη Azure, στη σελίδα ενοποίησης εφαρμογή **Dropbox για επιχειρήσεις** , κάντε κλικ στην επιλογή **Ρύθμιση παραμέτρων προμήθεια χρήστη** για να ανοίξετε το παράθυρο διαλόγου **Ρύθμιση παραμέτρων προμήθεια του χρήστη** .

2. Στην την ενεργοποίηση χρήστη προμήθεια στο DropBox για επιχειρήσεις σελίδας, κάντε κλικ στην επιλογή Ενεργοποίηση προμήθεια του χρήστη για να ανοίξετε είσοδος στο Dropbox για σύνδεση με το παράθυρο διαλόγου Azure AD.  

    ![Προμήθεια του χρήστη] (./media/active-directory-saas-dropboxforbusiness-tutorial/IC769517.png "Προμήθεια του χρήστη")

3. Στο παράθυρο διαλόγου **Είσοδος στο Dropbox για σύνδεση με το Azure AD** , πραγματοποιήστε είσοδο στο το Dropbox για επιχειρήσεις μισθωτή. 

    ![Προμήθεια του χρήστη] (./media/active-directory-saas-dropboxforbusiness-tutorial/IC769518.png "Προμήθεια του χρήστη")



4. Κάντε κλικ στην επιλογή **να επιτρέπεται** να εκχωρήσετε Azure AD για να αποκτήσετε πρόσβαση στο Dropbox. 

    ![Προμήθεια του χρήστη] (./media/active-directory-saas-dropboxforbusiness-tutorial/IC769519.png "Προμήθεια του χρήστη")



5. Για να ολοκληρώσετε τη ρύθμιση παραμέτρων, κάντε κλικ στο κουμπί **ολοκληρώθηκε** .  

    ![Προμήθεια του χρήστη] (./media/active-directory-saas-dropboxforbusiness-tutorial/IC769520.png "Προμήθεια του χρήστη")




##<a name="assigning-users"></a>Αντιστοίχιση χρηστών
  
Για να ελέγξετε τη ρύθμιση των παραμέτρων σας, πρέπει να εκχωρήσετε Azure AD χρηστών που θέλετε να επιτρέψετε χρησιμοποιώντας την εφαρμογή έχουν πρόσβαση σε αυτό αναθέτοντας σε αυτά.

###<a name="to-assign-users-to-dropbox-for-business-perform-the-following-steps"></a>Για να εκχωρήσετε στους χρήστες στο Dropbox για επιχειρήσεις, εκτελέστε τα ακόλουθα βήματα:

1.  Στην κλασική Azure πύλη, δημιουργήστε ένα λογαριασμό δοκιμής.

2.  Στη σελίδα ενοποίησης εφαρμογή **Dropbox για επιχειρήσεις **, κάντε κλικ στην επιλογή **εκχωρήσετε στους χρήστες**.

    ![Αντιστοίχιση χρηστών] (./media/active-directory-saas-dropboxforbusiness-tutorial/IC769521.png "Αντιστοίχιση χρηστών")

3.  Επιλέξτε το χρήστη έλεγχος, κάντε κλικ στην επιλογή **Εκχώρηση**και, στη συνέχεια, κάντε κλικ στο κουμπί **Ναι** για να επιβεβαιώσετε την ανάθεση.

    ![Ναι] (./media/active-directory-saas-dropboxforbusiness-tutorial/IC767830.png "Ναι")
  


Μπορείτε τώρα θα πρέπει να περιμένετε 10 λεπτά και να επαληθεύσετε ότι ο λογαριασμός έχει συγχρονιστεί στο Dropbox για επιχειρήσεις.

Ως πρώτο βήμα επαλήθευσης, μπορείτε να ελέγξετε την κατάσταση παροχής, κάνοντας κλικ στην επιλογή **Πίνακας εργαλείων** στη σελίδα ενοποίησης εφαρμογή **Dropbox για επιχειρήσεις** στην πύλη του Azure κλασική.

![Αντιστοίχιση χρηστών] (./media/active-directory-saas-dropboxforbusiness-tutorial/IC769522.png "Αντιστοίχιση χρηστών")


Ένας χρήστης ολοκληρώθηκε με επιτυχία προμήθεια του κύκλου υποδεικνύεται από ένα σχετικό κατάστασης.

![Αντιστοίχιση χρηστών] (./media/active-directory-saas-dropboxforbusiness-tutorial/IC769523.png "Αντιστοίχιση χρηστών")


Εάν θέλετε να ελέγξετε τις ρυθμίσεις καθολικής εισόδου, ανοίξτε τον πίνακα της Access.
Για περισσότερες λεπτομέρειες σχετικά με τον πίνακα της Access, ανατρέξτε στο θέμα [Εισαγωγή στο πλαίσιο πρόσβασης](active-directory-saas-access-panel-introduction.md).
