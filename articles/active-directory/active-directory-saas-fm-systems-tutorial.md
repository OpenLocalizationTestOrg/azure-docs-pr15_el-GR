#<a name="tutorial-azure-active-directory-integration-with-fm-systems"></a>Πρόγραμμα εκμάθησης: Ενοποίηση καταλόγου Azure Active Directory με FM: συστήματα
  
Είναι ο στόχος αυτού του προγράμματος εκμάθησης για να εμφανίσετε την ενοποίηση του Azure και FM:Systems.  
Το σενάριο που περιγράφονται σε αυτό το πρόγραμμα εκμάθησης προϋποθέτει ότι έχετε ήδη τα ακόλουθα στοιχεία:

-   Μια έγκυρη συνδρομή Azure
-   Μια FM:Systems καθολικής σύνδεσης με δυνατότητα συνδρομή
  
Μετά την ολοκλήρωση αυτού του προγράμματος εκμάθησης, θα μπορείτε να καθολικής σύνδεσης σε μια εφαρμογή στην τοποθεσία της εταιρείας σας FM:Systems (υπηρεσία παροχής που ξεκινούν από σύμβολο στην) ή με την [Εισαγωγή στο πλαίσιο πρόσβασης](active-directory-saas-access-panel-introduction.md)Azure AD χρηστών που έχουν εκχωρηθεί σε FM:Systems.
  
Το σενάριο που περιγράφονται σε αυτό το πρόγραμμα εκμάθησης αποτελείται από τα εξής δομικά στοιχεία:

1.  Ενεργοποίηση την ενοποίηση εφαρμογής για FM:Systems
2.  Ρύθμιση παραμέτρων Καθολικής σύνδεσης
3.  Ρύθμιση παραμέτρων προμήθεια του χρήστη
4.  Αντιστοίχιση χρηστών

![Σενάριο] (./media/active-directory-saas-fm-systems-tutorial/IC795899.png "Σενάριο")
##<a name="enabling-the-application-integration-for-fmsystems"></a>Ενεργοποίηση την ενοποίηση εφαρμογής για FM:Systems
  
Είναι ο στόχος αυτής της ενότητας για να εφαρμόσετε διάρθρωση πώς μπορείτε να ενεργοποιήσετε την ενοποίηση εφαρμογής για το FM:Systems.

###<a name="to-enable-the-application-integration-for-fmsystems-perform-the-following-steps"></a>Για να ενεργοποιήσετε την ενοποίηση εφαρμογής για FM:Systems, ακολουθήστε τα παρακάτω βήματα:

1.  Στην κλασική πύλη Azure, στο αριστερό παράθυρο περιήγησης, κάντε κλικ στην επιλογή **Υπηρεσία καταλόγου Active Directory**.

    ![Υπηρεσία καταλόγου Active Directory] (./media/active-directory-saas-fm-systems-tutorial/IC700993.png "Υπηρεσία καταλόγου Active Directory")

2.  Από τη λίστα **καταλόγου** , επιλέξτε τον κατάλογο για την οποία θέλετε να ενεργοποιήσετε την ενοποίηση καταλόγου.

3.  Για να ανοίξετε τις εφαρμογές προβολή, στην προβολή του καταλόγου, κάντε κλικ στην επιλογή **εφαρμογές** στο μενού επάνω.

    ![Εφαρμογές] (./media/active-directory-saas-fm-systems-tutorial/IC700994.png "Εφαρμογές")

4.  Κάντε κλικ στην επιλογή **Προσθήκη** στο κάτω μέρος της σελίδας.

    ![Προσθήκη εφαρμογής] (./media/active-directory-saas-fm-systems-tutorial/IC749321.png "Προσθήκη εφαρμογής")

5.  Στο παράθυρο διαλόγου **Τι θέλετε να κάνετε** , κάντε κλικ στην επιλογή **Προσθήκη εφαρμογής από τη συλλογή**.

    ![Προσθήκη εφαρμογής από gallerry] (./media/active-directory-saas-fm-systems-tutorial/IC749322.png "Προσθήκη εφαρμογής από gallerry")

6.  Στο **πλαίσιο αναζήτησης**, πληκτρολογήστε **FM:Systems**.

    ![Συλλογή εφαρμογών] (./media/active-directory-saas-fm-systems-tutorial/IC795900.png "Συλλογή εφαρμογών")

7.  Στο παράθυρο αποτελεσμάτων, επιλέξτε **FM:Systems**και, στη συνέχεια, κάντε κλικ στην επιλογή **Ολοκλήρωση** , για να προσθέσετε την εφαρμογή.

    ![FM: συστήματα] (./media/active-directory-saas-fm-systems-tutorial/IC800213.png "FM: συστήματα")
##<a name="configuring-single-sign-on"></a>Ρύθμιση παραμέτρων Καθολικής σύνδεσης
  
Στόχος αυτής της ενότητας είναι να περιγράψει τον τρόπο για να επιτρέψετε στους χρήστες να ελέγχουν την ταυτότητα FM:Systems με το λογαριασμό στο Azure AD χρησιμοποιώντας Ομοσπονδία με βάση το πρωτόκολλο SAML.

###<a name="to-configure-single-sign-on-perform-the-following-steps"></a>Για να ρυθμίσετε τις παραμέτρους της καθολικής σύνδεσης, ακολουθήστε τα παρακάτω βήματα:

1.  Στην κλασική πύλη Azure, στη σελίδα ενοποίησης εφαρμογής **FM:Systems** , κάντε κλικ στην επιλογή **Ρύθμιση παραμέτρων Καθολικής σύνδεσης** για να ανοίξετε το παράθυρο διαλόγου **Ρύθμιση παραμέτρων Καθολικής σύνδεσης** .

    ![Ρύθμιση παραμέτρων Καθολικής σύνδεσης] (./media/active-directory-saas-fm-systems-tutorial/IC790810.png "Ρύθμιση παραμέτρων Καθολικής σύνδεσης")

2.  Στη σελίδα **Πώς θέλετε οι χρήστες να πραγματοποιούν είσοδο FM:Systems** , επιλέξτε **Microsoft Azure AD καθολικής σύνδεσης**και, στη συνέχεια, κάντε κλικ στο κουμπί **Επόμενο**.

    ![Ρύθμιση παραμέτρων Καθολικής σύνδεσης] (./media/active-directory-saas-fm-systems-tutorial/IC795901.png "Ρύθμιση παραμέτρων Καθολικής σύνδεσης")

3.  Στη σελίδα **Ρύθμιση παραμέτρων URL εφαρμογής** , εκτελέστε τα ακόλουθα βήματα:

    ![Ρύθμιση παραμέτρων εφαρμογής διεύθυνσης URL] (./media/active-directory-saas-fm-systems-tutorial/IC795902.png "Ρύθμιση παραμέτρων εφαρμογής διεύθυνσης URL")

    1.  Στο πλαίσιο κειμένου **FM:Systems εισόδου στη διεύθυνση URL** , πληκτρολογήστε το FM:Systems **Απάντηση διεύθυνση URL** (π.χ.: *https://dpr.fmshosted.com/fminteract/ConsumerService2.aspx*).  

        >[AZURE.WARNING] Μπορείτε να λάβετε αυτήν την τιμή από την FM: συστήματα την ομάδα υποστήριξης.

    2.  Κάντε κλικ στο κουμπί **Επόμενο**

4.  Στη σελίδα **Ρύθμιση παραμέτρων Καθολικής σύνδεσης στο FM:Systems** , για να κάνετε λήψη του μετα-δεδομένων, κάντε κλικ στην επιλογή **Λήψη μετα-δεδομένων**και, στη συνέχεια, αποθηκεύστε τα μετα-δεδομένα στον υπολογιστή σας.

    ![Ρύθμιση παραμέτρων Καθολικής σύνδεσης] (./media/active-directory-saas-fm-systems-tutorial/IC795903.png "Ρύθμιση παραμέτρων Καθολικής σύνδεσης")

5.  Υποβολή το αρχείο λήψης μετα-δεδομένων για να σας FM: συστήματα την ομάδα υποστήριξης.

    >[AZURE.NOTE] FM σας: Η ομάδα υποστήριξης συστήματα έχει για να κάνετε την πραγματική ρύθμιση των παραμέτρων SSO.
Θα λάβετε μια ειδοποίηση όταν έχει ενεργοποιηθεί η SSO για τη συνδρομή σας.

6.  Στην κλασική Azure πύλη, επιλέξτε την καθολική σύνδεση ρύθμιση των παραμέτρων επιβεβαίωσης και, στη συνέχεια, κάντε κλικ στην επιλογή **Ολοκλήρωση** , για να κλείσετε το παράθυρο διαλόγου **Ρύθμιση παραμέτρων Καθολικής σύνδεσης** .

    ![Ρύθμιση παραμέτρων Καθολικής σύνδεσης] (./media/active-directory-saas-fm-systems-tutorial/IC795904.png "Ρύθμιση παραμέτρων Καθολικής σύνδεσης")
##<a name="configuring-user-provisioning"></a>Ρύθμιση παραμέτρων προμήθεια του χρήστη
  
Για να επιτρέψετε στους χρήστες Azure AD για να συνδεθείτε στο FM:Systems, πρέπει να παρασχεθεί στο FM:Systems.  
Στην περίπτωση FM:Systems, προμήθεια είναι μια μη αυτόματη εργασία.

###<a name="to-configure-user-provisioning-perform-the-following-steps"></a>Για να ρυθμίσετε την προμήθεια του χρήστη, εκτελέστε τα ακόλουθα βήματα:

1.  Σε ένα παράθυρο προγράμματος περιήγησης web, συνδεθείτε στην τοποθεσία της εταιρείας σας FM:Systems ως διαχειριστής.

2.  Μεταβείτε στις επιλογές **σύστημα διαχείρισης \> Διαχείριση ασφαλείας \> χρήστες \> λίστα χρηστών**.

    ![Διαχείριση του συστήματος] (./media/active-directory-saas-fm-systems-tutorial/IC795905.png "Διαχείριση του συστήματος")

3.  Κάντε κλικ στην επιλογή **Δημιουργία νέου χρήστη**.

    ![Δημιουργία νέου χρήστη] (./media/active-directory-saas-fm-systems-tutorial/IC795906.png "Δημιουργία νέου χρήστη")

4.  Στην ενότητα **Δημιουργία χρήστη** , εκτελέστε τα ακόλουθα βήματα:

    ![Δημιουργία χρήστη] (./media/active-directory-saas-fm-systems-tutorial/IC795907.png "Δημιουργία χρήστη")

    1.  Πληκτρολογήστε το όνομα χρήστη, τον κωδικό πρόσβασης και την επιβεβαίωση, τη διεύθυνση ηλεκτρονικού ταχυδρομείου και το Αναγνωριστικό υπαλλήλου έναν έγκυρο λογαριασμό Azure Active Directory που θέλετε για την παροχή των σε τα σχετικά πλαίσια κειμένου.
    2.  Κάντε κλικ στο κουμπί **Επόμενο**.

>[AZURE.NOTE] Μπορείτε να χρησιμοποιήσετε οποιοδήποτε άλλο FM:Systems χρήστη λογαριασμού εργαλεία δημιουργίας ή APIs που παρέχεται από FM:Systems σε λογαριασμούς χρηστών AAD διάταξη.

##<a name="assigning-users"></a>Αντιστοίχιση χρηστών
  
Για να ελέγξετε τη ρύθμιση των παραμέτρων σας, πρέπει να εκχωρήσετε Azure AD χρηστών που θέλετε να επιτρέψετε χρησιμοποιώντας την εφαρμογή έχουν πρόσβαση σε αυτό αναθέτοντας σε αυτά.

###<a name="to-assign-users-to-fmsystems-perform-the-following-steps"></a>Για να εκχωρήσετε στους χρήστες να FM:Systems, εκτελέστε τα ακόλουθα βήματα:

1.  Στην κλασική Azure πύλη, δημιουργήστε ένα λογαριασμό δοκιμής.

2.  Στη σελίδα **FM:Systems **ενοποίησης της εφαρμογής, κάντε κλικ στην επιλογή **εκχωρήσετε στους χρήστες**.

    ![Αντιστοίχιση χρηστών] (./media/active-directory-saas-fm-systems-tutorial/IC795908.png "Αντιστοίχιση χρηστών")

3.  Επιλέξτε το χρήστη έλεγχος, κάντε κλικ στην επιλογή **Εκχώρηση**και, στη συνέχεια, κάντε κλικ στο κουμπί **Ναι** για να επιβεβαιώσετε την ανάθεση.

    ![Ναι] (./media/active-directory-saas-fm-systems-tutorial/IC767830.png "Ναι")
  
Εάν θέλετε να ελέγξετε τις ρυθμίσεις καθολικής εισόδου, ανοίξτε τον πίνακα της Access. Για περισσότερες λεπτομέρειες σχετικά με τον πίνακα της Access, ανατρέξτε στο θέμα [Εισαγωγή στο πλαίσιο πρόσβασης](active-directory-saas-access-panel-introduction.md).
