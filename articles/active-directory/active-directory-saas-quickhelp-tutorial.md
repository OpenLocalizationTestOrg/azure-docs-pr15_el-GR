# <a name="tutorial-azure-active-directory-integration-with-quickhelp"></a>Πρόγραμμα εκμάθησης: Ενοποίηση καταλόγου Azure Active Directory με QuickHelp

Ο στόχος αυτού του προγράμματος εκμάθησης είναι να σας δείξουν πώς μπορείτε να ενοποιήσετε το QuickHelp με το Azure Active Directory (Azure AD).  
Ενοποίηση QuickHelp με Azure AD σάς παρέχει τα ακόλουθα πλεονεκτήματα: 

- Μπορείτε να ελέγξετε σε Azure AD ποιος έχει πρόσβαση σε QuickHelp 
- Μπορείτε να ενεργοποιήσετε τους χρήστες σας να αυτόματα να συνδεθεί στην QuickHelp (καθολικής σύνδεσης) με τους λογαριασμούς Azure AD
- Μπορείτε να διαχειριστείτε τους λογαριασμούς σας σε μια κεντρική θέση - κλασική πύλη του Azure

Εάν θέλετε να μάθετε περισσότερες λεπτομέρειες σχετικά με ΑΔΑ εφαρμογή ενοποίηση με το Azure AD, ανατρέξτε στο θέμα [Τι είναι η εφαρμογή access και καθολικής σύνδεσης με το Azure Active Directory](active-directory-appssoaccess-whatis.md).

## <a name="prerequisites"></a>Προαπαιτούμενα στοιχεία 

Για να ρυθμίσετε τις παραμέτρους ενοποίησης Azure AD με QuickHelp, χρειάζεστε τα ακόλουθα στοιχεία:

- Μια συνδρομή του Azure AD
- Μια QuickHelp καθολικής σύνδεσης με δυνατότητα συνδρομής


> [AZURE.NOTE] Για να ελέγξετε τα βήματα που περιγράφονται σε αυτό το πρόγραμμα εκμάθησης, δεν συνιστάται να χρησιμοποιείτε ένα περιβάλλον παραγωγής.


Για να ελέγξετε τα βήματα που περιγράφονται σε αυτό το πρόγραμμα εκμάθησης, θα πρέπει να ακολουθήσετε αυτές τις συστάσεις:

- Δεν πρέπει να χρησιμοποιείτε το περιβάλλον παραγωγής, εκτός εάν αυτό είναι απαραίτητο.
- Εάν δεν έχετε ένα περιβάλλον δοκιμαστική Azure AD, μπορείτε να αποκτήσετε μια μηνιαία δοκιμαστική [εδώ](https://azure.microsoft.com/pricing/free-trial/). 

 
## <a name="scenario-description"></a>Σενάριο περιγραφή
Είναι ο στόχος αυτού του προγράμματος εκμάθησης για να ελέγξετε Azure AD καθολικής σύνδεσης σε περιβάλλον δοκιμής.  
Το σενάριο που περιγράφονται σε αυτό το πρόγραμμα εκμάθησης αποτελείται από δύο κύριο μπλοκ δόμησης:

1. Προσθήκη QuickHelp από τη συλλογή 
2. Ρύθμιση παραμέτρων και έλεγχος Azure AD μονό καθολικής σύνδεσης


## <a name="adding-quickhelp-from-the-gallery"></a>Προσθήκη QuickHelp από τη συλλογή
Για να ρυθμίσετε την ενσωμάτωση των QuickHelp στην Azure AD, πρέπει να προσθέσετε QuickHelp από τη συλλογή στη λίστα των διαχειριζόμενων ΑΔΑ εφαρμογών.

**Για να προσθέσετε QuickHelp από τη συλλογή, ακολουθήστε τα παρακάτω βήματα:**

1. Στην **πύλη του Azure κλασική**, στο αριστερό παράθυρο περιήγησης, κάντε κλικ στην επιλογή **Υπηρεσία καταλόγου Active Directory**. 

    ![Υπηρεσία καταλόγου Active Directory][1]

2. Από τη λίστα **καταλόγου** , επιλέξτε τον κατάλογο για την οποία θέλετε να ενεργοποιήσετε την ενοποίηση καταλόγου.

3. Για να ανοίξετε τις εφαρμογές προβολή, στην προβολή του καταλόγου, κάντε κλικ στην επιλογή **εφαρμογές** στο μενού επάνω.

    ![Εφαρμογές][2]

4. Κάντε κλικ στην επιλογή **Προσθήκη** στο κάτω μέρος της σελίδας.

    ![Εφαρμογές][3]

5. Στο παράθυρο διαλόγου **Τι θέλετε να κάνετε** , κάντε κλικ στην επιλογή **Προσθήκη εφαρμογής από τη συλλογή**.

    ![Εφαρμογές][4]

6. Στο πλαίσιο αναζήτησης, πληκτρολογήστε **QuickHelp**.

    ![Εφαρμογές][5]

7. Στο παράθυρο αποτελεσμάτων, επιλέξτε **QuickHelp**και, στη συνέχεια, κάντε κλικ στην επιλογή **Ολοκλήρωση** , για να προσθέσετε την εφαρμογή.

    ![Εφαρμογές][500]


##  <a name="configuring-and-testing-azure-ad-single-sign-on"></a>Ρύθμιση παραμέτρων και έλεγχος Azure AD μονό καθολικής σύνδεσης
Ο στόχος αυτής της ενότητας είναι να σας δείξουν πώς μπορείτε να ρυθμίσετε τις παραμέτρους και να ελέγξετε Azure AD καθολικής σύνδεσης με QuickHelp που βασίζεται σε ένα χρήστη δοκιμής που ονομάζεται "Britta Simon".


Για να ρυθμίσετε τις παραμέτρους και να ελέγξετε Azure AD καθολικής σύνδεσης με QuickHelp, πρέπει να ολοκληρώσετε τα παρακάτω μπλοκ δόμησης:

1. **[Ρύθμιση παραμέτρων Azure AD καθολικής σύνδεσης](#configuring-azure-ad-single-single-sign-on)** - για να ενεργοποιήσετε τους χρήστες σας για να χρησιμοποιήσετε αυτήν τη δυνατότητα.
2. **[Δημιουργία μιας Azure AD δοκιμή χρήστη](#creating-an-azure-ad-test-user)** - για να ελέγξετε Azure AD καθολικής σύνδεσης με Britta Simon.
4. **[Δημιουργία μιας QuickHelp δοκιμή χρήστη](#creating-a-quickhelp-test-user)** - έχουν αντίστοιχο του Britta Simon στο QuickHelp που είναι συνδεδεμένο με το Azure AD αναπαριστάται με εκείνη.
5. **[Εκχώρηση του Azure AD δοκιμή χρήστη](#assigning-the-azure-ad-test-user)** - για να ενεργοποιήσετε την Britta Simon για να χρησιμοποιήσετε Azure AD καθολικής σύνδεσης.
5. **[Δοκιμές καθολικής σύνδεσης](#testing-single-sign-on)** - για να επιβεβαιώσετε αν λειτουργεί η ρύθμιση παραμέτρων.

### <a name="configuring-azure-ad-single-sign-on"></a>Ρύθμιση παραμέτρων Azure AD καθολικής σύνδεσης

Στόχος αυτής της ενότητας είναι για να ενεργοποιήσετε Azure AD καθολικής σύνδεσης στην πύλη του Azure κλασική και για τη ρύθμιση παραμέτρων Καθολικής σύνδεσης στην εφαρμογή σας QuickHelp.


**Για να ρυθμίσετε τις παραμέτρους Azure AD καθολικής σύνδεσης με QuickHelp, ακολουθήστε τα παρακάτω βήματα:**

1. Στην κλασική πύλη Azure, στη σελίδα ενοποίησης εφαρμογής **QuickHelp** , κάντε κλικ στην επιλογή **Ρύθμιση παραμέτρων Καθολικής σύνδεσης** για να ανοίξετε το παράθυρο διαλόγου **Ρύθμιση παραμέτρων Καθολικής σύνδεσης** .

    ![Ρύθμιση παραμέτρων Καθολικής σύνδεσης][6] 

2. Στη σελίδα **Πώς θέλετε οι χρήστες να πραγματοποιούν είσοδο QuickHelp** , επιλέξτε **Azure AD καθολικής σύνδεσης**και, στη συνέχεια, κάντε κλικ στο κουμπί **Επόμενο**.

    ![Azure AD καθολικής σύνδεσης][7] 

3. Στη σελίδα του παραθύρου διαλόγου **Διαμόρφωση παραμέτρων ρυθμίσεων εφαρμογής** , εκτελέστε τα ακόλουθα βήματα:

    ![Ρύθμιση παραμέτρων εφαρμογής][8] 
 
     μια. Στο πλαίσιο κειμένου **Εισόδου στη διεύθυνση URL** , πληκτρολογήστε τη διεύθυνση URL που χρησιμοποιούνται από τους χρήστες σας να καθολικής σύνδεσης στην τοποθεσία σας QuickHelp (π.χ.:* https://quickhelp.com/bsiazure/*).

     > [AZURE.NOTE] Εάν δεν γνωρίζετε την τιμή της εισόδου στη διεύθυνση URL, επικοινωνήστε με την ομάδα υποστήριξης QuickHelp.

     β. Κάντε κλικ στο κουμπί **Επόμενο**.

 
4. Στη σελίδα **Ρύθμιση παραμέτρων Καθολικής σύνδεσης στο QuickHelp** , εκτελέστε τα εξής βήματα: κάντε κλικ στην επιλογή **Λήψη μετα-δεδομένων**και, στη συνέχεια, αποθηκεύστε το αρχείο μετα-δεδομένων τοπικά στον υπολογιστή σας.

    ![Τι είναι το Azure AD Connect][9] 



1. Σύνδεση στην τοποθεσία της εταιρείας σας QuickHelp ως διαχειριστής.

2. Στο μενού στο επάνω μέρος, κάντε κλικ στην επιλογή **Διαχείριση**.

    ![Ρύθμιση παραμέτρων Καθολικής σύνδεσης][21]


1. Στο μενού **QuickHelp διαχείρισης** , κάντε κλικ στην επιλογή **Ρυθμίσεις**.

    ![Ρύθμιση παραμέτρων Καθολικής σύνδεσης][22]

1. Κάντε κλικ στην επιλογή **Ρυθμίσεις ελέγχου ταυτότητας**.

1. Στη σελίδα **Ρυθμίσεις ελέγχου ταυτότητας** , ακολουθήστε τα παρακάτω βήματα

    ![Ρύθμιση παραμέτρων Καθολικής σύνδεσης][23]

    μια. Ως **Τύπο SSO**, επιλέξτε **WSFederation**.

    β. Για να αποστείλετε το αρχείο σας έχουν ληφθεί Azure μετα-δεδομένων, κάντε κλικ στο κουμπί **Αναζήτηση**, μεταβείτε στο αρχείο, τέλος, στη συνέχεια, κάντε κλικ στην επιλογή **Αποστολή μετα-δεδομένων**.

    c. Στο πλαίσιο κειμένου **ηλεκτρονικού ταχυδρομείου** , πληκτρολογήστε **http://schemas.xmlsoap.org/ws/2005/05/identity/claims/emailaddress**.

    d. Στο πλαίσιο κειμένου **όνομα** , **πληκτρολογήστε http://schemas.xmlsoap.org/ws/2005/05/identity/claims/givenname**.

    ε. Στο πλαίσιο κειμένου **Όνομα επώνυμο** , **Τύπος http://schemas.xmlsoap.org/ws/2005/05/identity/claims/surname**

    f. Στη **Γραμμή ενεργειών**, κάντε κλικ στην επιλογή **Αποθήκευση**.







6. Στην κλασική Azure πύλη, επιλέξτε την καθολική σύνδεση παραμέτρων επιβεβαίωσης και, στη συνέχεια, κάντε κλικ στο κουμπί **Επόμενο**.

### <a name="creating-a-quickhelp-test-user"></a>Δημιουργία ενός χρήστη δοκιμής QuickHelp

Στόχος αυτής της ενότητας είναι να δημιουργήσετε ένα χρήστη που ονομάζεται Britta Simon στο QuickHelp.
Για καθολικής σύνδεσης για να εργαστείτε, Azure AD πρέπει να γνωρίζετε ποιος είναι ο χρήστης αντίστοιχο στο QuickHelp σε έναν χρήστη στο Azure AD. Με άλλα λόγια, μια σχέση σύνδεση μεταξύ ενός χρήστη Azure AD και το σχετικό χρήστη στο QuickHelp πρέπει να καθοριστούν.

QuickHelp υποστηρίζει προμήθεια μόνο στο χρόνο. Αυτό σημαίνει ότι, εάν χρειάζεται, ένα λογαριασμό χρήστη δημιουργείται αυτόματα στο QuickHelp και ο λογαριασμός είναι συνδεδεμένο με το λογαριασμό Azure AD.

Δεν υπάρχει κανένα στοιχείο ενέργειας για εσάς σε αυτήν την ενότητα.

    ![Εκχώρηση χρήστη][201] 

2. Στη λίστα εφαρμογών, επιλέξτε **QuickHelp**.

    ![Εκχώρηση χρήστη][202] 

1. Στο μενού στο επάνω μέρος, κάντε κλικ στην επιλογή **χρήστες**.
