#<a name="tutorial-azure-active-directory-integration-with-bamboo-hr"></a>Πρόγραμμα εκμάθησης: Ενοποίηση καταλόγου Azure Active Directory με μπαμπού HR

Είναι ο στόχος αυτού του προγράμματος εκμάθησης για να εμφανίσετε την ενοποίηση του Azure και BambooHR.  
Το σενάριο που περιγράφονται σε αυτό το πρόγραμμα εκμάθησης προϋποθέτει ότι έχετε ήδη τα ακόλουθα στοιχεία:

-   Μια έγκυρη συνδρομή Azure
-   Μια BambooHR καθολικής σύνδεσης με δυνατότητα συνδρομή

Μετά την ολοκλήρωση αυτού του προγράμματος εκμάθησης, θα μπορείτε να καθολικής σύνδεσης σε μια εφαρμογή στην τοποθεσία της εταιρείας σας BambooHR (υπηρεσία παροχής που ξεκινούν από σύμβολο στην) ή με την [Εισαγωγή στο πλαίσιο πρόσβασης](active-directory-saas-access-panel-introduction.md)Azure AD χρηστών που έχουν εκχωρηθεί σε BambooHR.

Το σενάριο που περιγράφονται σε αυτό το πρόγραμμα εκμάθησης αποτελείται από τα ακόλουθα δομικά στοιχεία:

1.  Ενεργοποίηση την ενοποίηση εφαρμογής για BambooHR
2.  Ρύθμιση παραμέτρων Καθολικής σύνδεσης
3.  Ρύθμιση παραμέτρων προμήθεια του χρήστη
4.  Αντιστοίχιση χρηστών

![Σενάριο] (./media/active-directory-saas-bamboo-hr-tutorial/IC796685.png "Σενάριο")
##<a name="enabling-the-application-integration-for-bamboohr"></a>Ενεργοποίηση την ενοποίηση εφαρμογής για BambooHR

Είναι ο στόχος αυτής της ενότητας για να εφαρμόσετε διάρθρωση πώς μπορείτε να ενεργοποιήσετε την ενοποίηση εφαρμογής για το BambooHR.

###<a name="to-enable-the-application-integration-for-bamboohr-perform-the-following-steps"></a>Για να ενεργοποιήσετε την ενοποίηση εφαρμογής για BambooHR, ακολουθήστε τα παρακάτω βήματα:

1.  Στην κλασική πύλη Azure, στο αριστερό παράθυρο περιήγησης, κάντε κλικ στην επιλογή **Υπηρεσία καταλόγου Active Directory**.

    ![Υπηρεσία καταλόγου Active Directory] (./media/active-directory-saas-bamboo-hr-tutorial/IC700993.png "Υπηρεσία καταλόγου Active Directory")

2.  Από τη λίστα **καταλόγου** , επιλέξτε τον κατάλογο για την οποία θέλετε να ενεργοποιήσετε την ενοποίηση καταλόγου.

3.  Για να ανοίξετε τις εφαρμογές προβολή, στην προβολή του καταλόγου, κάντε κλικ στην επιλογή **εφαρμογές** στο μενού επάνω.

    ![Εφαρμογές] (./media/active-directory-saas-bamboo-hr-tutorial/IC700994.png "Εφαρμογές")

4.  Κάντε κλικ στην επιλογή **Προσθήκη** στο κάτω μέρος της σελίδας.

    ![Προσθήκη εφαρμογής] (./media/active-directory-saas-bamboo-hr-tutorial/IC749321.png "Προσθήκη εφαρμογής")

5.  Στο παράθυρο διαλόγου **Τι θέλετε να κάνετε** , κάντε κλικ στην επιλογή **Προσθήκη εφαρμογής από τη συλλογή**.

    ![Προσθήκη εφαρμογής από gallerry] (./media/active-directory-saas-bamboo-hr-tutorial/IC749322.png "Προσθήκη εφαρμογής από gallerry")

6.  Στο **πλαίσιο αναζήτησης**, πληκτρολογήστε **BambooHR**.

    ![Συλλογή εφαρμογών] (./media/active-directory-saas-bamboo-hr-tutorial/IC796686.png "Συλλογή εφαρμογών")

7.  Στο παράθυρο αποτελεσμάτων, επιλέξτε **BambooHR**και, στη συνέχεια, κάντε κλικ στην επιλογή **Ολοκλήρωση** , για να προσθέσετε την εφαρμογή.

    ![BambooHR] (./media/active-directory-saas-bamboo-hr-tutorial/IC796687.png "BambooHR")
##<a name="configuring-single-sign-on"></a>Ρύθμιση παραμέτρων Καθολικής σύνδεσης

Στόχος αυτής της ενότητας είναι να περιγράψει τον τρόπο για να επιτρέψετε στους χρήστες να ελέγχουν την ταυτότητα BambooHR με το λογαριασμό στο Azure AD χρησιμοποιώντας Ομοσπονδία με βάση το πρωτόκολλο SAML.  
Ως μέρος αυτής της διαδικασίας, που απαιτούνται για να δημιουργήσετε ένα αρχείο βάσης 64 κωδικοποιημένο πιστοποιητικό.  
Εάν δεν είστε εξοικειωμένοι με αυτήν τη διαδικασία, δείτε [πώς μπορείτε να μετατρέψετε ένα πιστοποιητικό δυαδικών δεδομένων σε ένα αρχείο κειμένου](http://youtu.be/PlgrzUZ-Y1o).

###<a name="to-configure-single-sign-on-perform-the-following-steps"></a>Για να ρυθμίσετε τις παραμέτρους της καθολικής σύνδεσης, ακολουθήστε τα παρακάτω βήματα:

1.  Στην κλασική πύλη Azure, στη σελίδα ενοποίησης εφαρμογής **BambooHR** , κάντε κλικ στην επιλογή **Ρύθμιση παραμέτρων Καθολικής σύνδεσης** για να ανοίξετε το παράθυρο διαλόγου **Ρύθμιση παραμέτρων Καθολικής σύνδεσης** .

    ![Σενάριο] (./media/active-directory-saas-bamboo-hr-tutorial/IC796685.png "Σενάριο")

2.  Στη σελίδα **Πώς θέλετε οι χρήστες να πραγματοποιούν είσοδο BambooHR** , επιλέξτε **Microsoft Azure AD καθολικής σύνδεσης**και, στη συνέχεια, κάντε κλικ στο κουμπί **Επόμενο**.

    ![Ρύθμιση παραμέτρων Καθολικής σύνδεσης] (./media/active-directory-saas-bamboo-hr-tutorial/IC796688.png "Ρύθμιση παραμέτρων Καθολικής σύνδεσης")

3.  Στη σελίδα **Ρύθμιση παραμέτρων URL εφαρμογής** , στο πλαίσιο κειμένου **BambooHR εισόδου στη διεύθυνση URL** , πληκτρολογήστε τη διεύθυνση URL που χρησιμοποιούνται από τους χρήστες σας για να συνδεθείτε εφαρμογή σας BambooHR (π.χ.: https://company.bamboohr.com), και, στη συνέχεια, κάντε κλικ στο κουμπί **Επόμενο**.

    ![Ρύθμιση παραμέτρων εφαρμογής διεύθυνσης URL] (./media/active-directory-saas-bamboo-hr-tutorial/IC796689.png "Ρύθμιση παραμέτρων εφαρμογής διεύθυνσης URL")

4.  Στη σελίδα **Ρύθμιση παραμέτρων Καθολικής σύνδεσης στο BambooHR** , κάντε κλικ στην επιλογή **λήψη πιστοποιητικό**και, στη συνέχεια, αποθηκεύστε το αρχείο πιστοποιητικού στον υπολογιστή σας.

    ![Ρύθμιση παραμέτρων Καθολικής σύνδεσης] (./media/active-directory-saas-bamboo-hr-tutorial/IC796690.png "Ρύθμιση παραμέτρων Καθολικής σύνδεσης")

5.  Σε ένα παράθυρο προγράμματος περιήγησης web διαφορετικά, συνδεθείτε στην τοποθεσία της εταιρείας σας BambooHR ως διαχειριστής.

6.  Στην κεντρική σελίδα, ακολουθήστε τα παρακάτω βήματα:

    ![Καθολικής σύνδεσης] (./media/active-directory-saas-bamboo-hr-tutorial/IC796691.png "Καθολικής σύνδεσης")

    1.  Κάντε κλικ στην επιλογή **εφαρμογές**.
    2.  Στο μενού εφαρμογές στα αριστερά, κάντε κλικ στην επιλογή **Καθολικής σύνδεσης**.
    3.  Κάντε κλικ στην επιλογή **SAML καθολικής σύνδεσης**.

7.  Στην ενότητα **SAML καθολικής σύνδεσης** , ακολουθήστε τα παρακάτω βήματα:

    ![SAML καθολικής σύνδεσης] (./media/active-directory-saas-bamboo-hr-tutorial/IC796692.png "SAML καθολικής σύνδεσης")

    1.  Στην κλασική πύλη Azure, στη σελίδα παράθυρο διαλόγου **Ρύθμιση παραμέτρων Καθολικής σύνδεσης στο BambooHR** , αντιγράψτε την τιμή **Μία μόνο είσοδο στη διεύθυνση URL της υπηρεσίας** και στη συνέχεια επικολλήστε το στο πλαίσιο κειμένου **Διεύθυνση URL σύνδεσης SSO** .
    2.  Δημιουργία αρχείου **βάσης 64 κωδικοποιημένο** από το πιστοποιητικό που έχετε λάβει.  

        >[AZURE.TIP] Για περισσότερες λεπτομέρειες, ανατρέξτε στο θέμα [πώς μπορείτε να μετατρέψετε ένα πιστοποιητικό δυαδικών δεδομένων σε ένα αρχείο κειμένου](http://youtu.be/PlgrzUZ-Y1o)

    3.  Ανοίξτε το βάσης 64 κωδικοποιημένο πιστοποιητικό στο Σημειωματάριο, αντιγράψτε το περιεχόμενο της στο Πρόχειρο και στη συνέχεια επικολλήστε το στο πλαίσιο κειμένου **Πιστοποιητικό X.509**
    4.  Κάντε κλικ στην επιλογή **Αποθήκευση**.

8.  Στην κλασική Azure πύλη, επιλέξτε την καθολική σύνδεση παραμέτρων επιβεβαίωσης και, στη συνέχεια, κάντε κλικ στην επιλογή **Ολοκλήρωση** , για να κλείσετε το παράθυρο διαλόγου **Ρύθμιση παραμέτρων Καθολικής σύνδεσης** .

    ![Ρύθμιση παραμέτρων Καθολικής σύνδεσης] (./media/active-directory-saas-bamboo-hr-tutorial/IC796693.png "Ρύθμιση παραμέτρων Καθολικής σύνδεσης")
##<a name="configuring-user-provisioning"></a>Ρύθμιση παραμέτρων προμήθεια του χρήστη

Για να επιτρέψετε στους χρήστες Azure AD για να συνδεθείτε στο BambooHR, πρέπει να παρασχεθεί στο BambooHR.  
Στην περίπτωση BambooHR, προμήθεια είναι μια μη αυτόματη εργασία.

###<a name="to-provision-a-user-accounts-perform-the-following-steps"></a>Η παροχή μιας λογαριασμούς χρήστη, εκτελέστε τα ακόλουθα βήματα:

1.  Συνδεθείτε στην τοποθεσία σας **BambooHR** ως διαχειριστής.

2.  Στη γραμμή εργαλείων στο επάνω μέρος, κάντε κλικ στην επιλογή **Ρυθμίσεις**.

    ![Ρύθμιση] (./media/active-directory-saas-bamboo-hr-tutorial/IC796694.png "Ρύθμιση")

3.  Κάντε κλικ στην επιλογή " **Επισκόπηση**".

4.  Στο αριστερό παράθυρο περιήγησης, μεταβείτε στις επιλογές **ασφαλείας \> χρήστες**.

5.  Πληκτρολογήστε τη διεύθυνση όνομα, τον κωδικό πρόσβασης και ηλεκτρονικού ταχυδρομείου χρήστη από έναν έγκυρο λογαριασμό AAD που θέλετε για την παροχή των σε τα σχετικά πλαίσια κειμένου.

6.  Κάντε κλικ στην επιλογή **Αποθήκευση**.

>[AZURE.NOTE] Μπορείτε να χρησιμοποιήσετε οποιοδήποτε άλλο BambooHR χρήστη λογαριασμού εργαλεία δημιουργίας ή APIs που παρέχεται από BambooHR σε λογαριασμούς χρηστών AAD διάταξη.

##<a name="assigning-users"></a>Αντιστοίχιση χρηστών

Για να ελέγξετε τη ρύθμιση των παραμέτρων σας, πρέπει να εκχωρήσετε Azure AD χρηστών που θέλετε να επιτρέψετε χρησιμοποιώντας την εφαρμογή έχουν πρόσβαση σε αυτό αναθέτοντας σε αυτά.

###<a name="to-assign-users-to-bamboohr-perform-the-following-steps"></a>Για να εκχωρήσετε στους χρήστες να BambooHR, εκτελέστε τα ακόλουθα βήματα:

1.  Στην κλασική Azure πύλη, δημιουργήστε ένα λογαριασμό δοκιμής.

2.  Στη σελίδα **BambooHR **ενοποίησης της εφαρμογής, κάντε κλικ στην επιλογή **εκχωρήσετε στους χρήστες**.

    ![Αντιστοίχιση χρηστών] (./media/active-directory-saas-bamboo-hr-tutorial/IC796695.png "Αντιστοίχιση χρηστών")

3.  Επιλέξτε το χρήστη έλεγχος, κάντε κλικ στην επιλογή **Εκχώρηση**και, στη συνέχεια, κάντε κλικ στο κουμπί **Ναι** για να επιβεβαιώσετε την ανάθεση.

    ![Ναι] (./media/active-directory-saas-bamboo-hr-tutorial/IC767830.png "Ναι")

Εάν θέλετε να ελέγξετε τις ρυθμίσεις καθολικής εισόδου, ανοίξτε τον πίνακα της Access. Για περισσότερες λεπτομέρειες σχετικά με τον πίνακα της Access, ανατρέξτε στο θέμα [Εισαγωγή στο πλαίσιο πρόσβασης](active-directory-saas-access-panel-introduction.md).
