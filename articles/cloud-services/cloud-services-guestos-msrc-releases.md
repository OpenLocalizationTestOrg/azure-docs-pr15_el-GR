# <a name="azure-guest-os"></a>Λειτουργικό σύστημα Azure επισκέπτη 

Οι παρακάτω πίνακες δείχνουν τις ενημερώσεις του Microsoft Security απόκριση Center (MSRC) εφαρμόζεται σε το λειτουργικό σύστημα επισκέπτη Azure. Αναζήτηση σε αυτό το άρθρο για να προσδιορίσετε εάν μια συγκεκριμένη ενημερωμένη έκδοση ισχύει για το λειτουργικό σύστημα επισκέπτη που χρησιμοποιείτε. Ενημερώσεις διαθέτουν πάντα προς τα εμπρός για συγκεκριμένη [οικογένεια] [ family-explain] αυτές έχουν εισαχθεί στο.
